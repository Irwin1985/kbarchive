Q80920: FltSim: EFIS Flight Path for Martha’s Vineyard Does Not Show

Article: Q80920
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

In Microsoft Flight Simulator for the Macintosh version 4.0, when the EFIS
Flight Path Display for Martha's Vineyard is turned on and locked on the ILS
signal with NAV 1 set to 108.7 under NAVCOM, the rectangles will flash and then
turn off. The Omni-Bearing Indicator (OMI) will be correct, but the rectangles
do not appear.

Additional query words: 4.00
