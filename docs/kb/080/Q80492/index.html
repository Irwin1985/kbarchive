Q80492: "Path/File Access Error" with MS-DOS Editor

Article: Q80492
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using MS-DOS Editor and you try to save a read-only file, the
following error message will be displayed:

  Path/File access error

CAUSE
=====

You cannot save changes to a read-only file using the current filename. To save
changes, save the file under another filename.
