Q171751: FS95: No Air Traffic Control in Flight Simulator for Windows 95

Article: Q171751
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbsound kbusage
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for Windows 95, version 1.0 
- Microsoft Expansion Pack for Microsoft Flight Simulator for Windows 95, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Communication between Control Tower and pilot is not available in Flight
Simulator for Windows 95 (FS95). The Communications dialog box, accessible from
the Aircraft menu gives you the option to "Listen to Air Traffic Control", but
this does not activate the Control Tower sounds. The to "Listen to Air Traffic
Control" option increases the message-scrolling strip across the bottom of the
viewer window.

MORE INFORMATION
================

Voice Air Traffic Control is not available in FS95 during normal game play.
Voice Air Traffic Control is only available during Adventures. The Pilot's voice
is available during Lessons; the Pilot's voice will be that of the Instructor
during the Lesson. Sounds during Challenges will be the same as during normal
flight operation.

Additional query words: 1.00 6.00 flightsim fltsim flt sim fs6 fs95 ATC tower sound voices
