SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book MCSE Readiness Review Exam 70-028:
Administering Microsoft SQL Server 7.0, ISBN 0-7356-0672-2.

The following topics are covered:

- Page 199-200: Change "active" To "inactive" In Answer A

MORE INFORMATION
================

Page 199-200: Change "active" To "inactive" In Answer A
-------------------------------------------------------

On page 199-200, in Answer A, change the word "active" to "inactive". The
TRUNCATE_ONLY clause truncates the log by removing the inactive part of it
without making a backup copy of the log.

Change:
"A. Removes the active part of the log without making a backup copy"

To:
"A. Removes the inactive part of the log without making a backup copy"


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
