Q269806: MSB in Concert/Lands on Mars: How to View the About Dialog Box

Article: Q269806
Product(s): Microsoft Home Kids Products
Version(s): 1.0
Operating System(s): 
Keyword(s): kbui mskids kbimu
Last Modified: 14-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Scholastic's The Magic School Bus in Concert 
- Scholastic's The Magic School Bus Lands on Mars 
- Scholastic's Magic School Bus Series: Volcanoes, version 1.0 
- Scholastic's The Magic School Bus Discovers Flight, version 1.0 
- Scholastic's The Magic School Bus Whales and Dolphins, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to view the About dialog box in Microsoft
Scholastic's The Magic School Bus in Concert or Microsoft Scholastic's The Magic
School Bus Lands on Mars.

MORE INFORMATION
================

To view the About dialog box, press ALT+SHIFT+A.

The About dialog box contains the following information:

- The About tab contains the Product ID (PID) number and the Version number for
  the program.

  NOTE: If you request help from Microsoft Product Support Services, you must
  provide the PID number and the version number for the program.

- The Credits tab contains a list of the credits for the licensed photographs
  and video clips that are included in the program.

- The EULA tab contains the complete text of the End User License Agreement
  (EULA) for the program.
