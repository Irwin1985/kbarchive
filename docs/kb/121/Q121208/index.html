Q121208: SAMPLE: SNDPLAY Sample Plays a Sound File

Article: Q121208
Product(s): Microsoft Windows Software Development Kit
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): _IK kbfile kbmm kbsample kb16bitonly kbOSWin310
Last Modified: 04-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Software Development Kit (SDK) 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

SNDPLAY.EXE is a self-extracting file that contains all files necessary to build
the SNDPLAY program that demonstrates a simple application that plays a sound
file specified on the command line. It displays an error message if no sound
file was specified in the command line or the specified sound could not be
played, and immediately exits whether the sound is played or not.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  Sndplay.exe
  (http://download.microsoft.com/download/platformsdk/sample63/3.1/W31/EN-US/SNDPLAY.EXE)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
