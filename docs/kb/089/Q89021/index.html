Q89021: Memory Configurations Needed on HP LaserJets for Printing

Article: Q89021
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

This article discusses the amount of RAM that different Hewlett-Packard (HP)
LaserJet printers need for the following operations:

- Printing full-page graphics

- Printing with the HP PostScript cartridge

- Printing the HP LaserJet hardware configuration page

MORE INFORMATION
================

Printing Full-Page Graphics
---------------------------

The minimum additional memory required to support each type of printer is listed
below:

                   Mem Req'd     Mem Req'd    Default
  Model            for 300 dpi   for 150 dpi  Memory
  ------------------------------------------------------------------

  HP LaserJet II   1 MB          N/A          640K
  HP LaserJet IID  2 MB          1 MB         640K  (in duplex mode)

Printing with the HP PostScript Cartridge
-----------------------------------------

When an HP PostScript cartridge is used with an HP LaserJet printer, additional
memory is required for duplex printing and printing letter or legal size pages.
The minimum additional memory required to support each type of printer is listed
below:

  Model                  Letter   Legal     Duplex    Default Memory
  ------------------------------------------------------------------

  HP LaserJet III        1   MB   2   MB    N/A       1 MB
  HP LaserJet IIP        1.5 MB   2.5 MB    N/A       512K
  HP LaserJet IID        2.5 MB   2.5 MB    4.5 MB    512K
  HP LaserJet IIID       2   MB   2   MB    4   MB    1 MB

Printing the HP LaserJet Hardware Configuration Page
----------------------------------------------------

The HP LaserJet printer can report details about its hardware configuration. To
print this information:

1. Turn the printer's ONLINE button off.

2. Press the PRESS FONTS TEST option until the message "Test page queued" is
  displayed.

You can now print the printer configuration information, including the amount of
memory installed.
