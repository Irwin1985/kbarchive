Q89016: Apple LaserWriter Requires ROM Revision 2.0 or Later

Article: Q89016
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The original Apple LaserWriter and LaserWriter Plus printers require a ROM
revision of 2.0 to work correctly with Microsoft Windows. To determine the ROM
revision, see the status page that is printed when the printer is first turned
on.

NOTE: This information does not apply to Apple LaserWriter NT and NTX printers.
The Apple LaserWriter NT and NTX printers only have ROM versions of 1.0; there
are no ROM revisions for these printers.
