Q316964: INFO: How to Obtain the Latest Visual FoxPro for Windows 7.0 SP

Article: Q316964
Product(s): Microsoft FoxPro
Version(s): 7.0 SP1
Operating System(s): 
Keyword(s): 
Last Modified: 24-JAN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 7.0 SP1 
-------------------------------------------------------------------------------

SUMMARY
=======

Service packs are the means by which Microsoft distributes bug fixes. Service
packs keep a product current. Service packs include updates and may include
system administration tools, drivers, and additional components, which are
bundled for easy downloading.

Service packs are cumulative; each new service pack contains all the fixes that
exist in previous service packs, as well as any new fixes. You do not need to
install a previous service pack before you install the latest one.

MORE INFORMATION
================

Visual FoxPro 7.0 Service Pack 1 (SP1)
--------------------------------------

Release Date: January 17, 2002

How to Obtain:

The following file is available for download from the Microsoft Download Center:

  DownloadDownload VFP70SP1.msp now
  (http://download.microsoft.com/download/visualfoxpro7/SP/1.0/W98NT42KMeXP/EN-US/VFP70SP1.msp)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
