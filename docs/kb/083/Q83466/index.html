Q83466: "H" Section of Hardware Compatibility List Is Missing

Article: Q83466
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 26-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The "H" section of the hardware compatibility list for printers is missing from
the first 10,000 boxes of the Microsoft Windows operating system version 3.1.

MORE INFORMATION
================

This section includes the following printers:

Hermes H 606
Hermes H 606 PostScript (13 fonts)

HP DeskJet

HP DeskJet 500C (Needs driver from Hewlett Packard)
HP DeskJet Plus
HP DraftMaster I (Plotter)
HP DraftMaster II (Plotter)
HP DraftPro (Plotter)
HP DraftPro DXL (Plotter)
