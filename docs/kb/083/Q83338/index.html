Q83338: Downloading TrueType Fonts & LaserJets Before the Series IIs

Article: Q83338
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

Downloading TrueType fonts as PCL fonts to Hewlett-Packard (HP) printers
developed prior to the LaserJet Series II printers is not supported in Microsoft
Windows version 3.1. Such printers include the original HP LaserJet, the HP
LaserJet Plus, and the HP LaserJet 500+. On these printers, True Type fonts are
printed as raster graphics.

MORE INFORMATION
================

The HP LaserJet Series II printers and subsequent HP printers, including the HP
LaserJet III family, can either download TrueType fonts or print them as raster
graphics. This ability is controlled by the Print TrueType As Graphics check box
in the printer driver's Options dialog box. Downloading TrueType fonts is the
default selection. This results in the quickest printing times because less data
must be sent to the printer.

The HP products included here are manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding these products'
performance or reliability.
