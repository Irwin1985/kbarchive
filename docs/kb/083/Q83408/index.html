Q83408: Windows 3.0a Printer Drivers: File Sizes & Supported Printers

Article: Q83408
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 25-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 3.0a 
-------------------------------------------------------------------------------


SUMMARY
=======

This article contains the printer drivers, version numbers, expanded and
compressed file sizes, file dates, printer names (as they appear in the
Installed Printers box), and a list of the printers supported with each driver
supplied with Microsoft Windows version 3.0a.
