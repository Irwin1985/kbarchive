Q230060: Doc Err: The Main Directory is C:\Inetpub\Nntproot by Default

Article: Q230060
Product(s): Internet Information Server
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): 
Last Modified: 04-AUG-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Internet Information Server 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The "How Microsoft NNTP Service Works" topic in the online product documentation
states the following:

  The main directory is C:\Inetpub\Nntproot by default, which can be changed in
  the Home Directory property sheet of Microsoft NNTP Service.

However, the main directory is actually C:\InetPub\nntpfile\root by default.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

To go directly to this topic, click the Search tab in the product documentation
to search on "How Microsoft NNTP Service Works" (including the quotation marks),
or open the topic directly by using the following URL:

  http://localhost/iisHelp/iis/misc/default.asp
