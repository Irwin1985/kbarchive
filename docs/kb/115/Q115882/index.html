SUMMARY
=======

This article contains a listing of files included on the 3.5-inch 1.44- megabyte
(MB) Microsoft MS-DOS 6.2 Upgrade disks. This list includes the disk number,
compressed filename, compressed size, expanded filename, and uncompressed size.
