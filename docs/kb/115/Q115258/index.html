Q115258: NewGen ImagerPlus 12 Printer Emulations

Article: Q115258
Product(s): Microsoft Windows Printing Issues
Version(s): WINDOWS:3.1,6.0,6.0a
Operating System(s): 
Keyword(s): 
Last Modified: 07-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word for Windows, versions 6.0, 6.0a 
- Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The NewGen ImagerPlus 12 laser printer supports PostScript Level 1, HPPCL 4, and
HPPCL 5 printer emulations. It also supports the HPGL graphics language.

MORE INFORMATION
================

The NewGen ImagerPlus 12 is a 1200 x 1200 dots-per-inch (dpi) laser printer
manufactured by NewGen Systems Corporation. It uses the Weitek RISC processor
operating at 25 megahertz (MHz) and comes with 40 megabytes (MB) RAM in its
standard form. The maximum image size is 12 x 19.5 inches. Its built-in gamma
correction controllers have capabilities for adjusting image contrast.

The table below provides examples of printers that work with the aforementioned
emulation modes. For example, if you set your printer to HPPCL 4 emulation, you
can then use the printer drivers for the Hewlett- Packard Series II printer.

Emulation mode       Printer example
------------------------------------

HPPCL 4              Hewlett-Packard LaserJet Series II
HPPCL 5              Hewlett-Packard LaserJet III
PostScript Level 1   LaserWriter II NTX
