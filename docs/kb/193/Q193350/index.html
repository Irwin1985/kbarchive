Q193350: Virtual Globe Err Msg: Unable to Locate Favorites Folder…

Article: Q193350
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kberrmsg kbui kbimu
Last Modified: 12-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click the Favorites menu while you are running Encarta Virtual Globe
99, you may receive the following error message:

  Unable to locate Favorites Folder. Creating new one.

NOTE: If you are running Encarta Virtual Globe 99 on a Microsoft Windows NT-based
computer, you do not receive this error message.

CAUSE
=====

This behavior can occur if you delete the Encarta Virtual Globe Favorites folder
while you are running Encarta Virtual Globe.

RESOLUTION
==========

To resolve this issue, click OK. A new Favorites folder is created.

MORE INFORMATION
================

By default, Encarta Virtual Globe 99 stores any Favorites you add in the
following folder:

  Program Files\Microsoft Reference\Encarta Virtual Globe 99\Favorites
