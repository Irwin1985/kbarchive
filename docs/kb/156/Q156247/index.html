Q156247: Hellbender: How to Play a One Player Network Game

Article: Q156247
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbhowto
Last Modified: 08-NOV-2001

kbusage kbnetwork

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Hellbender for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article gives instructions on how to play one player network games
in Microsoft Hellbender.

MORE INFORMATION
================

Use the following steps to play a one player network game:

1. Start Hellbender.

2. Click Multiplayer.

3. Click OK.

4. Click Create Game.

5. Decrease the Number of Players to 1 and press ENTER.

6. Click Next.

7. Click Play Game.
