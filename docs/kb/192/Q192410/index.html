Q192410: Encarta Virtual Globe 1998 Does Not Start

Article: Q192410
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kb3rdparty kbenv kbsound kbimukbfaq
Last Modified: 24-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 1998 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Encarta Virtual Globe, 1998 edition, the program may
not start.

CAUSE
=====

This behavior can occur if the Compaq PSV220A Fax/Voice modem is installed in
your computer.

RESOLUTION
==========

To resolve this issue, follow these steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double click Multimedia.

3. Click the Advanced tab.

  NOTE: If you are using Microsoft Windows 98, click the Devices tab.

4. Click the PLUS SIGN (+) next to Audio Devices to expand the branch.

5. Click Audio For Modem Wave Properties, and then click Properties.

6. Click "Do not use audio features on this device," and then click OK.

7. Click Audio For Voice Modem Wave Device, and then click Properties.

8. Click "Do not use audio features on this device," and then click OK.

9. Click OK.

10. Close Control Panel, and then restart the computer.

NOTE: Disabling audio for the voice modem may cause voice mail and other related
features to not function properly.


MORE INFORMATION
================

Encarta Virtual Globe also may not start if a wave audio file is playing when
you attempt to start the program.

Encarta Virtual Globe plays wave audio files while its introductory screen is
loading. If Media Player and other files needed to play wave audio files are
being used by another program, Encarta Virtual Globe may not start.

To resolve this problem, quit any other programs that may be playing wave audio
files before you attempt to start Encarta Virtual Globe.
