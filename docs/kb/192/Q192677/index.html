Q192677: XWEB: Name, Company, Business Phone, & E-mail Fields are Empty

Article: Q192677
Product(s): Microsoft Exchange
Version(s): WINDOWS:5.5
Operating System(s): 
Keyword(s): 
Last Modified: 22-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
- Microsoft Outlook HTML Form Converter 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========


You may receive the following warning message when trying to save a custom HTML
contact from within the Outlook Web Access Client:

  The name, company, business phone, and e-mail fields are empty. You will have
  to use the next/previous button to open this item.

This behavior will occur only if the following conditions are met:

- The custom HTML contact form does not contain a General Page.

- and -

- Only one of the following fields were completed before trying to save the
  contact:

  First Name, Company or Business Phone

WORKAROUND
==========

Simply complete more than one of the required fields before saving the contact.
