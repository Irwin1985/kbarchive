Q192884: INFO: SpeechKit by Chant Inc. Support Information

Article: Q192884
Product(s): Microsoft Visual Basic for Windows
Version(s): WINDOWS:6.0; winnt:6.0

The information in this article applies to:

- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0

- Microsoft Visual C++, 32-bit Enterprise Edition, version 6.0

SUMMARY
=======

SpeechKit by Chant Inc. is component software that allows you to create speech
recognition and speech synthesis software for your desktop and Web- enabled
programs.

This article shows you where to download an evaluation copy of the software
component and where to obtain support information.

MORE INFORMATION
================

You can download an evaluation copy of the SpeechKit from the following
location:

  http://microsoft.com/technet/resource/downloads/thirdparty/speechkit/
  speechkit.htm

In order to use the evaluation copy, you must obtain a serial number from Chant
Inc.

Support for the SpeechKit by Chant Inc. is also provided by Chant Inc. For
support information, please contact the company directly at:

  Chant, Inc.
  4712 Admiralty Way
  Suite 550
  Marina del Rey, CA 90292-6905

  Phone: (310) 410-9895
  FAX : (310) 410-4896

  Email: support@chant.net
  Web Site: http://www.chant.net
