Q192423: EVG99: Back Button Does Not Return You to Previous Screen

Article: Q192423
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbtool kbui kbimu
Last Modified: 19-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Virtual Flights feature of Encarta Virtual Globe 99, you cannot
use the Back button to return to content you viewed before you started a Virtual
Flight.

RESOLUTION
==========

To resolve this issue, follow these steps:

1. On the Virtual Flights page, click Go To Map.

2. Insert the Encarta Virtual Globe 99 CD-ROM into your CD-ROM drive when you
  are prompted to do so, and then click OK. Encarta Virtual Globe 99 displays
  the default view of the globe.

3. Click Back.

4. When you are prompted to insert the Installation and Resources CD-ROM into
  your CD-ROM drive, click Cancel.

5. Click Back. Encarta Virtual Globe 99 displays the Virtual Flights Home page.

6. Click Back to return to the content you viewed before you started a Virtual
  Flight.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
99, version 1.0.
