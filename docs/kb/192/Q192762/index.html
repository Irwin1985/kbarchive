Q192762: Dbf2rs.exe Converts a VFP Cursor to ADO 2.x Recordset

Article: Q192762
Product(s): Microsoft FoxPro
Version(s): WINDOWS:2.1 SP2,2.6,6.0
Operating System(s): 
Keyword(s): kbfile kbDatabase kbGrpDSFox kbGrpDSMDAC kbDSupport kbADO210sp2 kbMDAC210SP2 kbMDAC260
Last Modified: 11-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 6.0 
- Microsoft Data Access Components versions 2.1 SP2, 2.6 
-------------------------------------------------------------------------------

SUMMARY
=======

Dbf2rs.exe is a self-extracting executable that contains a Visual FoxPro program
named Dbf2rs.prg, which shows how to convert a Visual FoxPro cursor to an
ActiveX Data Objects (ADO) recordset. The executable file also contains the
necessary ADO include files. Running the program in Visual FoxPro creates an ADO
recordset from a Visual FoxPro cursor.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

  DBF2RS.exe
  (http://download.microsoft.com/download/vfox60/sample/1/WIN98/EN-US/DBF2RS.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

NOTE: This sample requires ADO version 2.0.

REFERENCES
==========

For additional information on ADO 2.x, please see the Microsoft ActiveX Data
Objects Web site, which is located at the following URL:

  http://www.microsoft.com/data/ado/
