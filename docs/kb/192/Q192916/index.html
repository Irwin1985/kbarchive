Q192916: Outwars: Cannot Hear Sound in Game

Article: Q192916
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0,95
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbsound kbimu msgame outwars kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outwars, version 1.0 
- the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Outwars, you may not hear any sound in the game.

CAUSE
=====

This behavior can occur if you are using one of the following IBM Aptiva
computers with an installed IBM MWave sound card:

- IBM Aptiva 2134
- IBM Aptiva 2144
- IBM Aptiva 2168
- IBM Aptiva 2176

RESOLUTION
==========

To resolve this behavior, download and install the latest drivers for the IBM
MWave sound card. To do this, visit the following IBM Web site:

  http://www.pc.ibm.com/techinfo/b6d6.html

NOTE: If you are using an IBM Aptiva 2144 or 2168 computer, you need to download
and install an additional driver update. This driver update is available from
the following IBM Web site:

  http://www.pc.ibm.com/techinfo/b8b2.html
