Q192133: Cannot Sign Up for MSN Using a Credit Card Expiring After 2000

Article: Q192133
Product(s): The Microsoft Network
Version(s): WINDOWS:1.3
Operating System(s): 
Keyword(s): kbmsn
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 1.3 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to sign up for an MSN, The Microsoft Network, account, you may
receive one of the following error messages:

- 

  This is not a valid year.

- 

  Signup cannot be completed at this time. Please try again later.

CAUSE
=====

This behavior can occur if your credit card expires after the year 2000 or
during the month that you are signing up for the MSN account.

RESOLUTION
==========

To resolve this issue, upgrade to MSN version 2.5.

Additional query words: msnet msnetwork microsoft-net m.s.n.
