Q192342: HOWTO: Locate Files Cited by the Coverage Profiler Help Topic

Article: Q192342
Product(s): Microsoft FoxPro
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 11-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Visual FoxPro for Windows version 6.0 Help file topic for the Coverage
Profiler, refers to various project, source, class, and header files that can be
used to customize the Coverage Profiler feature. The files you can use to
customize the Coverage Profiler can be found in a file named Xsource.zip.

MORE INFORMATION
================

The Xsource.zip file is located in the ..\VFP98\Tools\XSource folder.

1. Unzip the files in the Xsource.zip file, using the UNZIP -d parameter to
  preserve subdirectory names. You may select the folder where the Xsource.zip
  file resides.

2. A subdirectory named \vfpsource is created.

3. Several sub-folders are created, including one named "\Coverage", which
  contains the following files:
