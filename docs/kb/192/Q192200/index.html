Q192200: XCLN: Schedule+ 1.0 Supports dates to 2019

Article: Q192200
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbYear2000
Last Modified: 17-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Schedule+ 1.0 supports dates through 2019. Any dates entered in the
year fields later than 2019 will be converted to 19XX. When selecting dates, the
scroll bar will only advance to 2019 for the year.

MORE INFORMATION
================

This is by design. Schedule+ was designed with a 100-year date window beginning
with 1920 and ending with 2019.
