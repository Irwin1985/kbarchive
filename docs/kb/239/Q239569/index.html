Q239569: XWEB: Requirements for Outlook Web Access

Article: Q239569
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 27-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the requirements for Outlook Web Access (OWA). The following
topics are discussed:

- Server requirements
- Client requirements
- Additional resources

MORE INFORMATION
================

Server Requirements
-------------------

To install the OWA components on a server, the following hardware and software is
required:

- Single or dual 60-megahertz (MHz) Pentium microprocessor (Pentium 133
  recommended)
- 64 megabytes (MB) of RAM
- High-speed network connection to Microsoft Exchange Server
- Microsoft Windows NT Server 4.0 with Service Pack 3 or later (the latest
  service pack for Windows NT Server 4.0 is recommended)
- Microsoft Internet Information Server (IIS) 3.0 or later
- Active Server Pages (ASP)
  You can install the ASP components from the latest Windows NT Server 4.0
  service pack CD-ROM or download the latest Windows NT 4.0 service pack and
  the ASP components from http://www.microsoft.com/windows/servicepacks/. The
  ASP components are also included with IIS 4.0.

NOTE: Outlook Web Access is included with Exchange Server 5.5. Microsoft
recommends that you apply the latest service pack for Exchange Server 5.5,
because it includes enhanced Outlook Web Access components. For additional
information about how to obtain the latest service pack for Exchange Server 5.5,
click the article number below to view the article in the Microsoft Knowledge
Base:

  Q191014 XGEN: How to Obtain the Latest Exchange Server 5.5 Service Pack

Client Requirements
-------------------

To connect to an OWA server, you must have a compatible Internet browser such as
the following:

- Microsoft Internet Explorer 3.02 or later
- A third-party Internet browser that supports frames

Additional Resources
--------------------

For detailed deployment instructions and troubleshooting suggestions for
implementing OWA, please see the "Planning and Deploying Outlook Web Access"
document that is available at the following address on the World Wide Web:

  http://support.microsoft.com/support/exchange/content/whitepapers/owaguide.doc

For information about troubleshooting known issues with the OWA components,
please see the OWA troubleshooter that is available at the following address on
the World Wide Web:

  http://support.microsoft.com/support/exchange/content/whitepapers/owa_tshoot.asp
