Q146337: Schd+ for Win 3.x Limited to 6 Open Archive and Project Files

Article: Q146337
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:3.x,7.0
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 7.0, used with:
   - the operating system: Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Schedule+ for Windows 3.x is limited to opening 6 Archive or Project
files. Attempting to open more than 6 of these files will result in the
following error:

  The file could not be opened. Your computer is out of memory.

STATUS
======

Microsoft has confirmed this to be a problem in version 7.0 of Microsoft
Schedule+. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.

Additional query words: 7.0
