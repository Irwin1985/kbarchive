SUMMARY
=======

The following article lists the directory of the 3.5-inch Microsoft Scenes
Impressionist Collection Disk 2 (Disk Assy 213-051-050). The Impressionist
Collection comes on two 3.5-inch disks. The program can be ordered on 5.25- inch
disks by using the 1.2 MB Disk Offer coupon that comes in the package.
