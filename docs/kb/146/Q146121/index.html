Q146121: Emulation Modes for Raven Printers

Article: Q146121
Product(s): Microsoft Windows Printing Issues
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

kbprint kbref

SUMMARY
=======

This article lists emulation modes for Raven printers as suggested by Raven.

This information is provided to suggest possible emulations only; printer
emulations are not supported by Microsoft. For additional information about
Raven printers, please contact Raven.

MORE INFORMATION
================

Raven is a subsidiary of Panasonic, so the Panasonic emulations should be used
first.
