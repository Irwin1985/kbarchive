Q257805: Encarta World English Dictionary: QuickDefine Dialog Box Closes

Article: Q257805
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta World English Dictionary, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you right-click a word in any Microsoft Office program, and then click
Define, the QuickDefine window opens briefly, and then disappears.

CAUSE
=====

This behavior can occur if the Focus feature in the Microsoft IntelliPoint
software or the "Activation follows mouse" feature in the Tweak UI utility is
turned on.

RESOLUTION
==========

To resolve this issue, turn off the Focus feature in the IntelliPoint software
or the "Activation follows mouse" feature in the Tweak UI utility:

Turn Off the Focus Feature in the IntelliPoint Software
-------------------------------------------------------

To turn off the Focus feature in the IntelliPoint software:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Mouse.

3. Click the StepSavers tab.

4. Under Focus, click to clear the "Select a window just by pointing at the
  titlebar" check box.

5. Click OK, and then close Control Panel.

Turn Off the Activation Follows Mouse Feature in the Tweak UI Utility
---------------------------------------------------------------------

To turn off the "Activation follows mouse" feature in the Tweak UI utility:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Tweak UI.

3. On the Mouse tab, click to clear the "Activation follows mouse (X-Mouse)"
  check box.

4. Click OK, and then close Control Panel.
