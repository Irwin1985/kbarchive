Q143143: Error Printing From Macintosh to LPT Port of Windows NT Server

Article: Q143143
Product(s): Microsoft Windows NT
Version(s): winnt:3.51
Operating System(s): 
Keyword(s): kbprint kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you send a print job from a Macintosh computer to a print device connected
to the LPT port of the Windows NT Server computer, the print job does one of the
following:

  - Stays in the "spooling" status

  -or-

  - Prints after a long delay with the following postscript error
    message:

       ERROR: timeout
       OFFENDING COMMAND: timeout


RESOLUTION
==========

To resolve this problem, install the fix mentioned below.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For
information on obtaining the Service Pack, query on the following word in the
Microsoft Knowledge Base (without the spaces):

  S E R V P A C K


Additional query words: prodnt dlc sfm
