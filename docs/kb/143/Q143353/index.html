Q143353: Old Shortcuts May Not Work with MSN 1.2

Article: Q143353
Product(s): The Microsoft Network
Version(s): WINDOWS:1.2,2.5
Operating System(s): 
Keyword(s): 
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.2, 2.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install version 1.2 or 2.5 of MSN, The Microsoft Network, your
shortcuts to MSN services may not work.

CAUSE
=====

Shortcuts created during the MSN Beta do not work with MSN 1.2 or 2.5 because of
changes in the way MSN servers read shortcut addresses.

RESOLUTION
==========

Create new shortcuts for the MSN services you want to use.
