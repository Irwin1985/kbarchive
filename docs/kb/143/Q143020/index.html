Q143020: PPT7: White Shadows Overlay Text with HP LaserJet 4

Article: Q143020
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbole kbprint kbtool kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you print text shadowed in white to a Hewlett-Packard (HP) LaserJet 4 printer
in HPPCL mode, the shadow is printed on top of the text.

RESOLUTION
==========

To resolve this problem, use any of the following methods.

Method 1: Use the HP III si and Print TrueType As Graphics Option
-----------------------------------------------------------------

Install an HP III si printer driver and select the Print TrueType As Graphics
check box.

1. On the Windows Start menu, point to Settings, and then click Printers.

2. Double-click the Add Printer icon and follow the instructions in the Add
  Printer Wizard to install the HP LaserJet III si printer driver.

3. Locate the HP III si printer driver in the Printers list and click it with
  the right mouse button.

4. Click Properties, and then click the Font tab.

5. Click to select the Print TrueType as Graphics check box, and then click OK.

6. Print your slide using this printer driver.

Method 2: Use a PostScript Printer Driver
-----------------------------------------

If your printer supports PostScript, use the HP LaserJet 4/4M PostScript printer
driver. To install the printer driver, follow these steps:

1. On the Windows Start menu, point to Settings, and then click Printers.

2. Double-click the Add Printer icon and follow the instructions in the Add
  Printer Wizard to install the HP 4/4M PostScript printer driver.

3. Print from your program with this driver.
