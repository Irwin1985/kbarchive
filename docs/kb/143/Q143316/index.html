Q143316: Unconf: Haunted House GPF Using Some Ensoniq Sound Cards

Article: Q143316
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

1.00

The information in this article applies to:

- Microsoft Haunted House for Windows, version 1.0

SUMMARY
=======

Certain Ensoniq Soundscape sound drivers may cause general protection faults
(GPF) when running The Ultimate Haunted House.

The following error message is an example of this:

  HHOUSE caused a General Protection Fault in module AAPIPKB3.DRV...

MORE INFORMATION
================

If you have an Ensoniq sound card and are experiencing these types of error
messages, you need to contact Ensoniq Corp. for their updated driver, version
1.30.03, for correcting the issue in Windows 3.x.

You can contact Ensoniq Corp. at the following number to request information on
obtaining the new version 1.30.03 driver for Windows 3.x, or any other
drivers/information:

Voice: (610) 647-3930
Fax: (610) 647-8908

Ensoniq also has the following Web page on the World Wide Web:

  http://www.ensoniq.com

NOTE: Customers using the Ensoniq sound cards bundled on their computers may have
to check with the original equipment manufacturer (OEM) for updated drivers.

Additional query words: 1.00 tapipkb3.drv newwin.exe newvoy.exe AAPIPKB4.DRV gp general protection fault uhh
