Q143364: 3D Movie Maker: Tools Available on CD-ROM for Troubleshooting

Article: Q143364
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 3D Movie Maker for Windows, version 1.0 
- Microsoft Nickelodeon 3D Movie Maker for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft 3D Movie Maker includes a troubleshooting tool, Msinfo32, on the
compact disc.

MORE INFORMATION
================

Msinfo32
--------

Use Msinfo32.exe (located in the D:\Pss directory) to troubleshoot sound and
video playback problems and other problems involving media playback and CD-ROM
data transfer (for more information regarding the operation of Msinfo32, please
see the Msinfo32 online Help).

In addition to Msinfo32, there are several files you can use to test your machine
for playback capability:

Filename       Directory   Size        Length (sec)
--------       ---------   ----        ------------

ADPCM11.wav    D:\Pss      36.7 KB     6.804
   Audio Format: Microsoft ADPCM, 11,025 MHz, 4 bit Mono

ADPCM22.wav    D:\Pss      74.6 KB     6.839
   Audio Format: Microsoft ADPCM, 22,050 MHz, 4 bit Mono

04logn3.avi    D:\3dmovie   1,383 KB   7.000
   Audio Format: PCM, 22,050 Hz, 8 bit Mono
   Video Format: 640x480, 8 Bits, 70 Frames, 10.000 Frames/Sec,
                      197 KB/Sec, MS-CRAM

NOTE: These files are on the compact disc and are used in 3D Movie Maker. You
might also choose to test other files with different formats.

One primary step in troubleshooting is to determine if the type of media (sound,
video, etc.) the program uses can be played using a standard program, such as
Media Player.
