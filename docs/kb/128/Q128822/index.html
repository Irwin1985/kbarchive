Q128822: Fax/Modem Settings Ignored by Microsoft Fax

Article: Q128822
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you change modem-specific settings in the Modem Properties dialog box in
Microsoft Fax, the fax transport does not use the new settings. For example, the
fax transport does not use a new Speaker Volume or Answer After <n> Rings
setting.

CAUSE
=====

This behavior occurs only with certain fax/modems. A list of these fax/modems is
not currently available.


RESOLUTION
==========

To make sure the new settings are used, restart the application that will use
them. For example, restart Microsoft Exchange after you change modem-specific
settings in Microsoft Fax.

MORE INFORMATION
================

To access fax/modem settings in Microsoft Fax, follow these steps:

1. Double-click the Microsoft Fax Status icon on the taskbar.

  NOTE: If this icon is not present, first double-click the Inbox icon on the
  desktop, then click the Fax profile if prompted.

2. On the Options menu, click Modem Properties.
