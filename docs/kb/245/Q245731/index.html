Q245731: DirectX 7.0a: List of Files Displayed in DirectX Diagnostic Tool

Article: Q245731
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbref kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains the list of Microsoft DirectX 7.0a files that are
displayed in the DirectX Diagnostic Tool on a Microsoft Windows 98-based
computer if you successfully installed DirectX 7.0a.

You can use this tool to verify that these files are installed on your computer
after you download and install DirectX 7.0a from the following Microsoft DirectX
Web site:

  http://www.microsoft.com/downloads/release.asp?ReleaseID=16819

MORE INFORMATION
