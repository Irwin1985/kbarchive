Q245603: How to Create a Windows NT Printer Share

Article: Q245603
Product(s): Microsoft Windows NT
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbprint
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Server, Enterprise Edition version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to share a printer connected to your computer with
other users on your network.

MORE INFORMATION
================

After you install the printer you are going to share, you can create a printer
share by following these steps:

1. Click Start, point to Settings, and then click Printers.

2. Right-click the printer that you are going to share, and then click Sharing.

3. Click Shared, type the name of the new printer share in the Share Name box,
  and then click OK.
