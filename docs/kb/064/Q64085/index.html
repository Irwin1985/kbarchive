Q64085: Changing Portrait and Landscape Orientation in DayBook

Article: Q64085
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0
Operating System(s): 
Keyword(s): 
Last Modified: 01-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you change the paper orientation in DayBook, the change isn't recognized
the first time you print. Any changes made are only recognized the second time
you print.

CAUSE
=====

This is due to a problem with the communication between DayBook and the Control
Panel.

STATUS
======

Microsoft has confirmed this to be a problem in Daybook version 1.0. This
problem does not appear in later versions.
