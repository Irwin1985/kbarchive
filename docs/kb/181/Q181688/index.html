Q181688: Developing for MS Agent: Examples Folder Cannot Be Read

Article: Q181688
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 19-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Developing for Microsoft Agent ISBN 1-57231-720-5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to view the Examples folder or any of its subfolders, you may
receive an error message similar to the following:

  "D:\Examples\c is not accessible. The disk structure is corrupt and
  non-readable."

  - or -

  "D:\Examples\c is not accessible. This folder was moved or removed."

CAUSE
=====

The cause of this problem is not known at this time.


RESOLUTION
==========

In the United States, you can call 1-800-MS-PRESS to obtain a replacement CD-ROM
that will resolve this problem.

Outside the United States, please return your book and disc to the point of
purchase, or contact your local Microsoft Press distributor for further
information. A list of Microsoft Press distributors worldwide is available at
the following World Wide Web address:

http://mspress.microsoft.com/worldwide/

STATUS
======

Microsoft Press has confirmed this to be a problem in "Developing for Microsoft
Agent."
