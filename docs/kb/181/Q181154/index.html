Q181154: DRW98: Microsoft Draw 98 Available for Download

Article: Q181154
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:98
Operating System(s): 
Keyword(s): kbfile kbdta
Last Modified: 15-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Draw 98 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

The following file is available for download from the Microsoft Download Center.
With Microsoft Draw 98, you can create 3-D text effects, diagrams, and so on.
Draw 98 is an OLE server and can be used with any OLE-compliant program.

The following file is available for download from the Microsoft Download Center:

  Draw98.exe
  (http://download.microsoft.com/download/office97std/Update/1/WIN98/EN-US/draw98.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Additional query words: publisher mso97.dll escher office art drw drawing
