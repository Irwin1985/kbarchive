Q73744: Tandy DOS 5.0 Available from Radio Shack

Article: Q73744
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 23-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Tandy technical support [(817) 878-6875] reports that Tandy has released Tandy
DOS version 5.0, which has part number 25-4176 and is available from any Radio
Shack store.

MORE INFORMATION
================

Four models of Tandy computers are on the Microsoft MS-DOS 5 Operating System
Hardware Compatibility List:

  Tandy 4000 LX
  Tandy 4020 LX
  Tandy 4025 LIX
  Tandy 5000 MC

If a computer is not on the list, it may or may not run the Microsoft MS-DOS 5
Upgrade; that is, Microsoft has not tested or confirmed the compatibility.
