Q73290: MS-DOS 5 Working Disk Contents (5.25-Inch Disk Set)

Article: Q73290
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 22-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are directory listings of the working disks created by using Setup
with the /F parameter (5.25-inch disks). (The first listing was obtained using
the DIR command with the /AH parameter to view the hidden system files.)
