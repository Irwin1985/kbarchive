Q73650: Appnote Reference: Corp Support (LAN, SQL, Mail, OS/2, PM)

Article: Q73650
Product(s): Microsoft Product Support Information
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

The following is a list of the Application Notes currently available in the
Corporate Support group:

                 CORPORATE SUPPORT APPLICATION NOTES
                 ===================================

Mail for AppleTalk Networks
---------------------------

MM0741  Configuring the Remote Server Connection
       No disk included
       Applies to Microsoft Mail for AppleTalk Networks, version 3.1
       Software Library: Not available
       MSDL: MM0741.EXE

  This document describes how to configure the Microsoft Mail Remote
  Server Connection to automatically connect and exchange information
  between two servers. The Remote Server Connection requires
  AppleTalk(R) Remote Access to establish communications through
  modems and phone lines.

MM0795  General Troubleshooting Flowcharts
       No disk
       Applies to Microsoft Mail for AppleTalk Networks, version 3.0, 3.1
       Software Library: Not available
       MSDL: MM0795.SEA

  This Application Note contains two flowcharts that will assist you
  in troubleshooting the Microsoft Mail server and the Microsoft Mail
  workstation. These charts have been designed and reviewed by
  Microsoft engineers who work with Microsoft Mail for AppleTalk
  Networks.

Mail for PC Networks
--------------------

WA0641  Database Maintenance Utilities
       1 Disk included
       Applies to Microsoft Mail for PC Networks, versions 2.1x, 3.0, 3.2
       UNAVAILABLE in Software Library, MSDL
       Last Update: 2/95

  Describes operational details for utilities used to repair and
  maintain the database for Microsoft Mail for PC Networks.

WA0725  Directory Synchronization (Dir-Sync)
       1 Disk included
       Applies to Microsoft Mail for PC Networks, versions 3.0, 3.2
       Software Library: S14075
       Last Update: 10/94
       MSDL: WA0725.EXE
      
  This application note is designed for the mail administrator and
  outlines the procedures directory synchronization (Dir-Sync) performs
  during normal operation. The network administrator can use these
  procedures to examine Dir-Sync files, to reset these files, or to
  perform a manual Dir-Sync process to fully synchronize the postoffice
  address lists (POLs). This Application Note is a supplement to the
  Microsoft Mail "Administrator's Guide."

WA0728  Troubleshooting the Mail Connection Gateway
       No disk
       Applies to Microsoft Mail Connection Gateway version 1.0b
       Software Library: (pending)
       MSDL: yes

  This application note is designed to help you troubleshoot and
  correct common problems encountered with the Microsoft Mail
  Connection gateway, version 1.0b.

WA0940  Recommendations for Better Performance
       No Disk
       Applies to Microsoft Mail for PC Networks, versions 3.0 and 3.2
       Software Library: WA0940, S14753
       MSDL: yes

  This Application Note provides several recommendations that should
  improve the performance of large implementations of Microsoft Mail for
  PC Networks by decreasing delivery time and by making the standard
  deviation of delivery times predictable and reasonably close to the
  median.


Mail Gateways
-------------

GW0645  Configuring the SMTP Gateway and Domain Name Service (DNS)
       No Disk
       Applies to Mail Gateway to SMTP version 3.0
       Software Library: GW0645, Sxxxxx (pending)
       MSDL: Yes

  Describes how to configure Microsoft Mail so you can exchange mail
  with other users outside your local domain. For example: exchanging
  mail between two users working at different companies.
