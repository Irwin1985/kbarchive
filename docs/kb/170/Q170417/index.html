Q170417: Closing ZAK Setup Window Causes Setup to Stop

Article: Q170417
Product(s): Microsoft Windows NT
Version(s): 1.0
Operating System(s): 
Keyword(s): kbsetup kbui
Last Modified: 21-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you click the Close button in the upper-right corner of the parent Setup
window for Zero Administration Kit (ZAK) server, Setup quits and must be run
again.

RESOLUTION
==========

If you close the parent Setup window by clicking the Close button, you must run
Setup again.

STATUS
======

This behavior is by design.
