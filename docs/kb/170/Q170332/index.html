Q170332: Windows NT Workstation Banner on Start Menu Is Truncated

Article: Q170332
Product(s): Microsoft Windows NT
Version(s): 1.0
Operating System(s): 
Keyword(s): kbui
Last Modified: 21-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click the Start button on a Windows NT-based computer configured as a
Zero Administration Kit (ZAK) Appstation, the vertical "Windows NT Workstation"
banner on the Start menu may be truncated.

CAUSE
=====

Because of the small number of items present on the Start menu, there is not
enough space for the entire "Windows NT Workstation" title to be displayed.

RESOLUTION
==========

To disable the "Windows NT Workstation" banner, use the following steps.

NOTE: Depending on the configuration of your ZAK Appstation, you may be required
to log on with administrative privledges to perform these steps.

1. Click the Start button, point to Settings, and then click Taskbar.

2. Click the "Show small icons in Start menu" check box to select it.

3. Click OK.
