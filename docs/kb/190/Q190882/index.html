Q190882: SAMPLE: FSOSAMP.EXE Application Uses the FileSystem Object

Article: Q190882
Product(s): Microsoft Visual Basic for Windows
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): kbfile kbsample kbVBp600 kbGrpDSVB kbFSO
Last Modified: 11-JAN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

FsoSamp.exe is a self-extracting executable file containing a Microsoft Visual
Basic project that demonstrates the use of the FileSystem Object (scrrun.dll)
that ships with the versions of Visual Basic listed above.

This "Explorer" style sample application demonstrates accessing the properties of
Drives, Folders, and Files, as well as creating and editing text files on the
machine in which it is running.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

  Fsosamp.exe
  (http://download.microsoft.com/download/vb60ent/Sample3/1/W9XNT4/EN-US/Fsosamp.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
