Q120413: Microsoft Mail for PC Networks Version 3.2a Update

Article: Q120413
Product(s): Microsoft Mail For PC Networks
Version(s): 3.2a
Operating System(s): 
Keyword(s): kbfile kbgraphxlinkcritical
Last Modified: 05-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for PC Networks, version 3.2a 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Mail for PC Networks Update version 3.2a is available on OnLine,
the Microsoft TechNet CD, and the Internet.

Microsoft Mail for PC Networks version 3.2a Update is available at no charge for
customers who are currently running version 3.2 of Microsoft Mail for PC
Networks. Customers currently using versions 2.1 or 3.0 of Mail must purchase
the 3.2a Postoffice Update for a minimal update fee, to receive 3.2
documentation and software.

The files listed below are available to download from the Microsoft Download
Center. For information on how to download these files, please see the
"Instructions for Downloading" section below.

FILES AVAILABLE FROM ONLINE SERVICES
------------------------------------

The Microsoft Mail for PC Networks 3.2a Update consists of three self- extracting
files:

Item ID     Filename          Description 
S14882      MAIL32AS.EXE      Server updates for 3.2a
S14883      MAIL32AC.EXE      Client updates for 3.2a
S14884      MAIL32AM.EXE      Macintosh client updates for 3.2a

The following files are available for download from the Microsoft Download
Center:

  DownloadDownload Mail32ac.exe now
  (http://download.microsoft.com/download/pcmail/Update/7/DOS/EN-US/Mail32ac.exe)

  DownloadDownload Mail32as.exe now
  (http://download.microsoft.com/download/pcmail/Update/8/DOS/EN-US/Mail32as.exe)

  DownloadDownload Mail32am.exe now
  (http://download.microsoft.com/download/msmail/Update/3.2/W9XNT4/EN-US/Mail32am.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.


MORE INFORMATION
================

IMPORTANT: MAIL32AS.EXE, MAIL32AC.EXE, and MAIL32AM.EXE contain READ_1ST.TXT,
which includes the update policy, pertinent update information, and a list of
all the files that will be modified. This file also contains instructions on how
to create the setup disks. Please read this file first.

For Windows clients, you MUST run the Setup program to properly update each
workstation to version 3.2a. Because of the number of files modified and version
number consistency issues, it is not sufficient to copy individual .DLL and
executable files in lieu of running SETUP.EXE. Running the Setup program is the
only way to ensure each and every file is the latest version.

NOTE: This update has been placed on online services for your convenience. If you
plan to download this update, please note that it is approximately 5 MB in size.
Since the download time will be lengthy, the connect charges could be costly.

An extra utility, such as MMFClean, is not shipped with this Update. MMFClean is
available as MMFUTIL.EXE, a self-extracting file, on the Microsoft Software
Library (MSL).
