SUMMARY
=======

Below is the table of contents of the October 1993 issue of the "FoxPro
Advisor." This publication is published by Data Based Solutions, Inc. 4010
Morena Blvd., Suite 200, San Diego, CA 92117 (619) 483-6400. For phone orders
only, call: 800-336-6060

MORE INFORMATION
================

Expert FoxPro. Creating an Invoice Screen. One-to-many screen; Screen Builder;
ON KEY LABEL; PUSH KEY; POP KEY; _TALLY. By Yair Alan Griver. Page 8.

FoxPro Fundamentals. Using Procedures and User-Defined Functions. DO WITH;
FUNCTION; PARAMETERS; PROCEDURE; = / STORE. By Miriam Liskin. Page 10.

Developer View. Pushing deLimits. Ad hoc selection criteria; Character Strings;
Delimiters; Runtime user input; Screen Builder. By Alan Schwartz. Page 18.

Functional FoxPro. Financial Functions Fox Forgot. Future Value; Present Value;
Time Value of Money; FV(); PV(). By Malcolm C. Rubel. Page 22.

Program Control. Using TIMEOUT to Protect Data. BROWSE; INLIST(); READ;
#READCLAUSES; READKEY(); TIMEOUT; VALID. By Drew Speedie. Page 24.

Add-On Product. Dual Field Incremental Searching with Jkey. Shareware;
Sequential; Match; Desired Record; INDEX; SEEK; LOCATE; Find. By Peter Diotte.
Page 28.

Special Report. Microsoft + FoxPro: How's It Going? Picture of Fox Developers in
front of Microsoft building. By John L. Hawkins. Page 30.

Report Writer. Multiple One-to-Many Relations in Report Writer. Multi- column
reports; SELECT SQL; two detail bands; all children first; IIF().two child
files. UNION. By Tamar E. Granor. Page 32.

DDE. Using DDE Warm and Hot Links Between FoxPro & Excel. DDEAdvise;
DDERequest; DDEPoke; DDETerminate; READ SHOW clause; SHOW GETS OFF. By Menachem
Bazian. Page 38.

User Groups. Page 52.

Q&A. FoxPro Solutions. BROWSE NOEDIT Append; Project Manager EXCLUDE; SET
BELL ON; _WRAP; CHR(7). By Steve Freides and Tamar E. Granor. Page 58.
