Q160928: Components Installed by The Microsoft Network version 2.0

Article: Q160928
Product(s): The Microsoft Network
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-JAN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the components installed by MSN, The Microsoft Network.

================

You do not have the option to uninstall MSN, but you can uninstall individual
components using these steps:

1. In Control Panel, double-click Add/Remove Programs.

2. On the Install/Uninstall tab, click the component you want to remove, and
  then click Add/Remove.
