Q234895: Microsoft Office 2000 Professional At a Glance Comments and Corr

Article: Q234895
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 09-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Office 2000 Professional At a Glance ISBN 1-57231-937-2 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Office 2000 Professional At a
Glance, ISBN 1-57231-937-2.

The following topics are covered:

- Page viii: Change The Titles Of Two Sections

- Page 7: New Tab In New Office Documents Dialog Box

- Pages 18-19: New Tab In New Office Documents Dialog Box

- Page 74: Incorrect Button Icon

- Pages 156 and 158: New Tab In New Presentations Dialog Box

- Page 198: New Tab Added To The New Dialog Box In Access

- Page 240: Change Title

- Page 248: Change Title

- Page 273: Step 1 In First Procedure Is Incorrect

MORE INFORMATION
================

Page viii: Change The Titles Of Two Sections
--------------------------------------------

Two of the subtitles in section 13 are not consistent with the others.
Change the reference to page 240 to: "Creating and Sending an E-Mail Message."

Change the reference to page 248 to: "Attaching a File to an E-Mail Message."


Page 7: New Tab In New Office Documents Dialog Box
--------------------------------------------------

The image at the top of the page may not match your screen because the New Office
Documents dialog box has changed since the printing of this book.


Pages 18-19: New Tab In New Office Documents Dialog Box
-------------------------------------------------------

The image at the top of the page may not match your screen because the New Office
Documents dialog box has changed since the printing of this book.


Page 74: Incorrect Button Icon
------------------------------

On page 74, step 2 contains examples of four formatting buttons. The icon on the
Align Left button is incorrect. The icon displayed is the Align Right icon.


Pages 156 and 158: New Tab In New Presentations Dialog Box
----------------------------------------------------------

The image may not match your screen because the New Presentations dialog box has
changed since the printing of this book.


Page 198: New Tab Added To The New Dialog Box In Access
-------------------------------------------------------

The image may not match your screen because the New dialog box has changed since
the printing of this book.


Page 240: Change Title
----------------------

Change the title on this page to "Creating and Sending an E-Mail Message" to
match the new Table of Contents.


Page 248: Change Title
----------------------

Change the title on this page to "Attaching a File to an E-Mail Message" to match
the new Table of Contents. Change the procedure title to "Attach a File to an
E-Mail Message".


Page 273: Step 1 In First Procedure Is Incorrect
------------------------------------------------

Change Step 1 in the first procedure to read:
"Click the View menu, point to Go To, and then click News."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.

Additional query words: EUBOOK OFF2000 EUAAG 1-57231-937-2
