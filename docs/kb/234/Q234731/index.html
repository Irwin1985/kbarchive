Q234731: MS Pocket Guide to MS Internet Explorer 5 Comments and Correctio

Article: Q234731
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 09-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Pocket Guide to Microsoft Internet Explorer 5 ISBN 1-57231-985-2 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Pocket Guide to Microsoft
Internet Explorer 5, ISBN 1-57231-985-2.

The following topics are covered:

- Page 22: Delete Sentence That Is No Longer Correct

- Page 43: Fix Two Incorrect Entries

- Page 75: Screen Has Changed

- Page 92: Last Sentence Of Last Entry Is Incorrect

- Page 99: Second Procedure Incorrect

- Page 100: Procedure Is Incorrect

- Page 114: Menu Name Incorrect

- Page 128: Info In Toolbars Entry Is Incorrect

- Page 133: Procedure Incorrect

- Page 136: Rewrite Incorrect Information

- Page 146: Button Name Incorrect

- Page 157: Command Omitted From Toolbars Submenu

- Page 159: Several Submenu Errors

- Page 163: Delete Incorrect Commands From Menu

- Page 164: Menu Command Name Incorrect

MORE INFORMATION
================

Page 22: Delete Sentence That Is No Longer Correct
--------------------------------------------------

The last sentence in the Opening the Address Book subentry under the Address Book
entry is incorrect. Please delete the entire sentence beginning: "From within
Internet Explorer..."


Page 43: Fix Two Incorrect Entries
----------------------------------

Both of the entries on this page contain errors.

In the second sentence of the Connection Wizard entry, change:
"...choose Programs, Internet Explorer, and Connection Wizard."

To: "... choose Programs, Accessories, Communications, and Internet Connection
Wizard."

Rewrite the entire second paragraph of the Cookies entry as follows:

If you want to be warned before accepting cookies, choose the Tools menu's
Internet Options command and click the Security tab. Select the Internet zone
and click Custom Level. Scroll down the list to the Cookies category, and under
the Allow Cookies That Are Stored On Your Computer heading, click the Prompt
option button. Then click OK twice.


Page 75: Screen Has Changed
---------------------------

The screen on this page has changed. Please replace it with the most recent
version of the IE5 screen.


Page 92: Last Sentence Of Last Entry Is Incorrect
-------------------------------------------------

The last sentence of the My Computer entry is incorrect. Please rewrite it to
say:
"To use Internet Explorer in this way, start Internet Explorer and then choose
the View menu's Explorer Bar command and the submenu's Folders command."


Page 99: Second Procedure Incorrect
-----------------------------------

Step 3 in the second procedure is incorrect. Please change:
"Click the New Message toolbar button."

To: "Click the New Post toolbar button."


Page 100: Procedure Is Incorrect
--------------------------------

The first sentences in both steps 3 and 5 in the procedure are incorrect.

Change step 3 to read: "Click the New Post toolbar button."

Change step 5 to read: "Click the Attach toolbar button."


Page 114: Menu Name Incorrect
-----------------------------

In the Setting Up Security Zones subentry of the Security Zones entry, change the
first sentence to read:
"...choose the Tools menu's Internet Options..."


Page 128: Info In Toolbars Entry Is Incorrect
---------------------------------------------

In the Outlook Express's toolbar subentry of the Toolbars entry, rewrite the
sentence to say the following:

"Customizing the toolbar in Outlook Express works in exactly the same way; just
right-click the toolbar, choose the Customize command, and then make your
changes."


Page 133: Procedure Incorrect
-----------------------------

Step 2 of the Wallet procedure should say:
"Choose the Tools menu's Internet..."


Page 136: Rewrite Incorrect Information
---------------------------------------

The Saving World Wide Web Documents and Images subentry is incorrect.

Change the 2nd sentence to say:
"...save the web page you see on your screen."

Rewrite the rest of the subentry as follows:
"When Internet Explorer displays the Save Web Page dialog box, use the Save As
type drop-down list box to describe which part of the web page you want to save.
To save the entire web page and to create a folder that contains all of the
individual files that make up the web page, select Web Page, Complete.

To save an image, you can also often right-click the image and then choose the
shortcut menu's Save Picture As command."


Page 146: Button Name Incorrect
-------------------------------

The 4th sentence of the 2nd subentry is incorrect. Change it to read:
"...click the Reply toolbar button..."


Page 157: Command Omitted From Toolbars Submenu
-----------------------------------------------

The Radio command is missing from the submenu. Between the Links and Customize
commands, add the following:

"Radio

Adds or removew the Radio toolbar. You can use the Radio toolbar to listen to
Internet radio."


Page 159: Several Submenu Errors
--------------------------------

1. Between the Windows Update and Internet Options rows of the Mail and News
submenu, add the following row:
"Show Related Links
Opens the Search bar and displays a list of links related to the web page you're
currently viewing."

2. Change the Internet Option command to say "Internet Options"

3. On the Help Menu, change the Tour command to say "Web Tutorial."

4. Delete the Repair row on the Help Menu.


Page 163: Delete Incorrect Commands From Menu
---------------------------------------------

Change: "Mark Connection As Read"
To: "Mark Conversation As Read"

Delete the entries for Flag Message, Find People, Find Text, and Find Message.


Page 164: Menu Command Name Incorrect
-------------------------------------

Near the bottom of the View menu entries, change: "Next Unread Thread"
To: "Next Unread Conversation"


Additional query words: EUBOOK EUPG IE50
