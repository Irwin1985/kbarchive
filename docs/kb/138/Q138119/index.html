Q138119: Windows 95 Readme.txt File in Drivers\Other\Creative\Sb16-awe

Article: Q138119
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The following information is a copy of the information in the Readme.txt file in
the Drivers\Other\Creative\Sb16-awe folder on the Windows 95 CD- ROM. Setup does
not copy this file to your hard disk.

MORE INFORMATION
================

Creative Technologies Sound Blaster 16 & AWE 32
-----------------------------------------------

This sound driver is intended for use with Creative Technologies Sound Blaster 16
& AWE 32 sound cards.

This driver is enhanced over the version built into Windows 95, as follows:

* Some systems, using the current SB 16/AWE32 drivers, encounter CRC errors while
downloading data and playing sound files. This new driver eliminates these CRC
errors.

See INSTALL.TXT for instructions about how to install this driver into Windows
95.
