Q48445: Emulation Modes for Brother Dot-Matrix Printers

Article: Q48445
Product(s): Microsoft Windows Printing Issues
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

The following are the emulation modes for the dot-matrix printers from Brother
International, Inc.:

  Brother Model           Emulation Mode
  -------------           --------------

  M1009                   Mode I  = Epson FX series
  M1109                   Mode II = IBM Proprinter
  M1409
  M1509
  M1709
  M1809
  M1909

  M1109AP                 Apple ImageWriter

  M1209                   Epson FX Series
                          IBM Proprinter

  M1724L                  Epson LQ1000
                          IBM Proprinter XL
                          Diablo/Brother HR

  M1824L                  IBM Proprinter XL
  M1924L                  Epson LQ Series
                          Diablo 630
                          Brother HR

  M2518                   IBM Proprinter
                          Epson EX/FX

  M4018                   IBM Proprinter
                          Epson FX Series

If you have a question about switching emulation modes, refer to your printer
documentation or contact Brother International, Inc.

To obtain phone numbers for Brother, query on the following words in the
Microsoft Knowledge Base:

  "brother" (without the quotation marks) and "phonebook" (without the
  quotation marks)
