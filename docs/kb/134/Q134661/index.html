Q134661: PDFs Included with Systems Management Server 1.1

Article: Q134661
Product(s): Microsoft Systems Management Server
Version(s): winnt:1.0,1.1
Operating System(s): 
Keyword(s): kbnetwork
Last Modified: 09-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of the package definition files (PDFs) included with
Systems Management Server version 1.1:

File           Description
====================================================================
ACS200.PDF     Access 2.00
ACS2PAK.PDF    Access Service Pack 2.00
AUDIT.PDF      Audit 1.00
DEINSTAL.PDF   Deinstall Microsoft Systems Management Server Client
              Components
DOS50.PDF      Microsoft DOS 5.0

DOS62.PDF      Microsoft DOS 6.2

EXC50C.PDF     Excel 5.0c
MSAPPS.PDF     MSAPPS 2.0 Shared Applications (OLE servers)
OFF42A.PDF     Microsoft Office 4.2
OFF42C.PDF     Microsoft Office 4.2c
OFP43C.PDF     Microsoft Office Professional 4.3c
OFP43_.PDF     Microsoft Office Professional 4.3
PATCHRDR.PDF   SMS 1.1 Patch Utility for Redirector
PPT40A.PDF     PowerPoint 4.0a
PPT40C.PDF     PowerPoint 4.0c
PRJ40_.PDF     Microsoft Project 4.0
UPGRADE.PDF    Upgrade Microsoft Systems Management Components
WAS351.PDF     Windows NT Server 3.51

WFW310.PDF     Windows For Workgroups 3.10
