Q256648: Asheron’s Call: Game Appears Too Dark

Article: Q256648
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbdisplay kbui kbimu msgame kbgraphxlinkcritical
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call, version 1.0 
- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Asheron's Call, the game may appear too dark.

CAUSE
=====

This behavior can occur if the Brightness setting in the game is set too low.

RESOLUTION
==========

To resolve this issue, increase the Brightness setting in the game:

1. On the Asheron's Call game screen, click Options ([GRAPHIC:the green triangle
  with two check boxes]) in the lower right corner of the screen.

2. Click the Sound/Graphics tab.

3. Move the Brightness slider to the right.

4. Click Apply.

Additional query words: 1.00 msgame igz msngz ashcall ac dim kbgraphx asherons
