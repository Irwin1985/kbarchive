Q152654: XADM: Welcome Message to New Users Not Available

Article: Q152654
Product(s): Microsoft Exchange
Version(s): winnt:4.0,5.0,5.5
Operating System(s): 
Keyword(s): kbenv
Last Modified: 22-MAY-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Exchange Server does not have an option for sending new users a
welcome message.

MORE INFORMATION
================

In version 3.x of Microsoft Mail for PC Networks, it was possible to send a
welcome message to new users via the WELCOME.GLB file. In Microsoft Exchange, it
is by design that no user input source for welcome messages is available.
