Q152997: Using the Micro Solutions Backpack CD-ROM Drive w/ Windows NT

Article: Q152997
Product(s): Microsoft Windows NT
Version(s): 3.50 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 3.5, 3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

This article has information on using the Micro Solutions Backpack CD-ROM drive
with Windows NT. Windows NT does not include drivers for this device and the
device is not on the Hardware Compatibility List (HCL).

MORE INFORMATION
================

This information is available from Micro Solutions, document number 3002.

Micro Solutions currently has Windows NT support available in beta form for
Backpack CD-ROM drives. Only Backpack models 163550/163700 (double speed) and
164550/164700 (quad speed) are supported. This driver will allow installation of
Windows NT and Backpack CD-ROM drive usage on Windows NT workstations. This
driver is not intended for use under Windows 95.

The Backpack "16-bit Built-in Sound Board" is not supported under Windows NT.
There are no current plans to provide Windows NT support for the Backpack Sound
Board.

It is important to note that Backpack CD-ROM drive models 160550 (single speed)
and 162550 (double speed) contain a different control interface and are not
supported by Micro Solutions's Windows NT drivers.

How to Obtain the Software
--------------------------

The beta drivers can be downloaded in a complete driver package from Micro
Solutions's Web site at: http://www.micro-solutions.com or from its Technical
Support BBS at 815.756.9100.

Technical Support
-----------------

Support Technicians will not be available on the phone for this beta driver for
the Windows NT Backpack CD-ROM drive. If you experience problems, please fax or
e-mail (ntcdbeta@micro-solutions.com, subject line: EPP problem) Micro Solutions
the brand and model of the computer and the type of EPP port (chipset if
possible).

Please include:
Your name
E-mail address
Phone Number (Voice and Fax)
Brand Name and exact model of computer
Type of parallel port (EPP chipset manufacturer if possible)
