Q152858: XGEN: How to Obtain Exchange Server 4.0 U.S. Service Pack 2

Article: Q152858
Product(s): Microsoft Exchange
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbsetup kbgraphxlinkcriticalkbfixlist
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

WARNING: Due to U.S. State Dept. export regulations, all SP2 client
software on these online services contains EXPORTABLE (40-bit) security
only. U.S. & Canadian customers should NOT install these, or else they will
no longer be able to read existing 64-bit encrypted mail. A North American-
only version of SP2 is available to US and Canadian customers (for shipping
and handling) by calling Microsoft at 1-800-360-7561.

SUMMARY
=======

This article lists the files available for Microsoft Exchange Service Pack 2. It
describes where they can be found and how to install them. For additional
information, click the article number below to view the article in the Microsoft
Knowledge Base:

  Q153655 README.TXT: Microsoft Exchange 4.0 U.S Service Pack 2

MORE INFORMATION
================

The Microsoft Exchange Server U.S. Service Pack 2 is a cumulative build of all
fixes to Microsoft Exchange Server version 4.0.

AVAILABLE FORMATS FOR MICROSOFT EXCHANGE SERVER 4.0 SERVICE PACK 2
------------------------------------------------------------------

Service Pack 2 is available as single self-extracting archive files for Intel
(x86), MIPS, PPC, and ALPHA-based computers.

ORDERING MICROSOFT EXCHANGE SERVER 4.0 SERVICE PACK 2
-----------------------------------------------------

If you do not have access to the electronic services listed below, you can order
Service Pack 2 (CD-ROM) from the Microsoft at 1-800-360-7561.
