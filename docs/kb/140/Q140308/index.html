Q140308: Cinemania 95: Error "Cannot Find One of the Required Fonts…"

Article: Q140308
Product(s): Microsoft Home Multimedia Titles
Version(s): 1995 edition
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Cinemania for Windows 1995 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to run Cinemania 1995, you receive the following error
message:

  Cinemania cannot find one or more of the required fonts (Arial, MSReference1,
  MSReference2). To reinstall the required fonts, please run setup again.

This occurs when the following fonts are not installed:

- Arial

- MSReference1

- MSReference2

RESOLUTION
==========

Remove and then reinstall Cinemania using the steps below:

1. Run the Cinemania Setup program.

2. Click Remove All.

3. Run Setup again to reinstall Cinemania.

MORE INFORMATION
================

During Setup, Cinemania 1995 installs the following fonts to your system:

  Arial             (TrueType)
  MS Reference 1    (TrueType)
  MS Reference 2    (TrueType)

When you remove Cinemania 95 before reinstalling, Setup is forced to check the
system files. Any missing system files on the Cinemania disc, including fonts
needed to run the program correctly, are reinstalled.
