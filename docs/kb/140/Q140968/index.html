Q140968: How to Search SNA Server Articles by KB Subcategories

Article: Q140968
Product(s): Microsoft SNA Server
Version(s): 2.0,2.1,2.11,2.11 SP1,3.0,4.0
Operating System(s): 
Keyword(s): 
Last Modified: 06-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft SNA Server, versions 2.0, 2.1, 2.11, 2.11 SP1, 3.0, 4.0, on platform(s):
   - the operating system: Microsoft Windows NT 
-------------------------------------------------------------------------------

SUMMARY
=======

The following KBSubcategory keywords are used to help make queries of the
Microsoft SNA Server for Windows NT Knowledge Base quicker and more complete.
The keywords have an "sna" prefix so that a full-text query will not find the
keyword in articles not directly related to the category.
