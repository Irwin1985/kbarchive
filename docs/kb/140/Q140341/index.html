Product(s): Microsoft Visual Basic for Windows

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Standard Edition, 32-bit, for Windows, version 4.0 
- Microsoft Visual Basic Professional Edition, 16-bit, for Windows, version 4.0 
- Microsoft Visual Basic Professional Edition, 32-bit, for Windows, version 4.0

SUMMARY
=======

The following table lists fixed problems in Visual Basic for Windows, version
4.0. This is not a comprehensive list and is current as of Nov. 26, 1995. For
more information about each issue, query in the Microsoft Knowledge Base on the
article identifier (ID).

A similar article lists the unfixed bugs. To get both these lists, query on the
following word in the Microsoft Knowledge Base:

  kblist
