Q68540: Installing Windows Printer Description Files from the SDL

Article: Q68540
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 03-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

The Supplemental Driver Library (SDL) contains Windows Printer Description (WPD)
files that require the Windows 3.0 PostScript Driver (PSCRIPT.DRV) for
installation.

MORE INFORMATION
================

Use the following steps to install any of the WPD files listed at the bottom of
this article:

Installation Procedure For 5.25 inch SDL disk set
-------------------------------------------------

1. Install the PSCRIPT.DRV printer driver from your SDL Disk 4.

2. Select PostScript Printer from the Installed Printers list box.

3. Choose Configure.

4. Choose Setup. (The PostScript Printer dialog box appears.)

5. Choose Add Printer.

6. Type the name of the drive that contains Disk 4 of the Supplemental Driver
  Library. The Control Panel displays a list of additional PostScript printers
  that can be installed.

7. Select the printer from the Add Printer list box.

8. Select Add. The Control Panel installs the printer.

9. Select Done. The Printer pull-down box of the PostScript Printer dialog box
  displays the newly installed printer(s).

10. Choose OK and proceed with normal printer setup in the Control Panel.

The following Supplemental Driver Library (SDL) Windows Printer Description (WPD)
files are located on the 5.25-inch, 1.2MB Disk 4, and 3.5-inch 720 K Disk 7:

  Description                             Filename
  -------------                           ----------

  HP LaserJet IID (PostScript cartridge)  HPIID522.WPD
  HP LaserJet IIP (PostScript cartridge)  HPIIP522.WPD
  HP LaserJet III (PostScript cartridge)  HPIII522.WPD
  IBM 4019 version 52.1 (17 Fonts)        IBM17521.WPD
  IBM 4019 version 52.1 (39 Fonts)        IBM39521.WPD
  NEC Silentwriter2 90 version 52.2       N2090522.WPD
  NEC Silentwriter2 290 version 52.0      N2290520.WPD
  Okidata OL840/PS version 51.8           OL840518.WPD
  QMS PS 820 Turbo                        Q820_517.WPD
  QMS PS 2200 version 51.0                Q2200510.WPD
  TI MicroLaser PS17 version 52.1         TIM17521.WPD
  TI MicroLaser PS35 version 52.1         TIM35521.WPD
  Unisys AP9415 version 47.0              U9415470.WPD
