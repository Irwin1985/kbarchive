Q195279: Hitchhiker’s Guide to VB & SQL Server, 5th Ed. Corrections

Article: Q195279
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 07-JAN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Hitchhiker's Guide to Visual Basic and SQL Server, 5th Edition ISBN 1-57231-567-9 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book "Hitchhiker's Guide to Visual Basic and SQL
Server, Fifth Edition," ISBN 1-57231-567-9.

The following topics are covered:

- rdoGridProject.ocx Missing From CD-ROM

MORE INFORMATION
================

RdoGridProject.ocx Missing From CD-ROM
--------------------------------------

When you attempt to open the \RDO Examples\rdoGrid\rdoGridProject.vbp project
from the CD-ROM, you may receive the following error message:

  Unable to set the version compatible component:
  E:\RDO Examples\rdoGrid\rdoGridProject.ocx

To avoid this error message, please follow these steps:

1. In the \RDO Examples\rdoGrid\ folder, make a copy of rdoGridControl.ocx.

2. Rename the copy to rdoGridProject.ocx.

3. Open the project again.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
