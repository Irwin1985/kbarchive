Q175422: Monster Truck Madness: Garbled or Missing Fonts

Article: Q175422
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbdisplay kbenv
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Monster Truck Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you launch Monster Truck Madness for Windows, version 1.0, the program's
fonts may appear missing or garbled.

RESOLUTION
==========

Make sure your display resolution is set higher than 640 x 480. To do this, use
the following steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Display.

3. Click the Settings tab.

4. Make sure Desktop Area is set higher than 640 x 480 pixels. For example, 800
  x 600 pixels.

5. Click OK.

6. Click Yes if you are asked to restart your computer.

If your resolution is already set higher than 640 x 480, do the following:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Fonts.

3. Locate the font Monotype Sorts. Verify the filename is:

  Mtsorts_.ttf

4. Click and hold the mouse on the font name and drag the font to the desktop or
  Recycle Bin.

5. Try to run Monster Truck Madness again.

Additional query words: 1.00 heiroglyphic text mono type mono-type
