Q242572: Games: Compatibility Issues with Some CD-ROM Drives

Article: Q242572
Product(s): Microsoft Home Games
Version(s): 2.0
Operating System(s): 
Keyword(s): kb3rdparty kbenv kbhw kbimu msgame kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires II: The Age of Kings, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a list of CD-ROM drives that may not be fully compatible
with the Microsoft games listed at the beginning of this article.

NOTE: Some additional CD-ROM drives may not be compatible with the encryption
technology used by the games listed at the beginning of this article.

MORE INFORMATION
================

You may encounter problems running one of the games listed at the beginning of
this article if one of the following CD-ROM drives is installed in your
computer.

Acer 44X
--------

You may be unable to successfully install the game from this CD-ROM drive.

There is no known resolution.

Acer CD-624a
------------

This CD-ROM drive may be unable to read the CD-ROM for the game when you use a
real-mode CD-ROM driver.

To resolve this issue, use a protected-mode CD-ROM driver.

Memorex 2216
------------

This CD-ROM drive may be unable to read the CD-ROM for the game.

There is no known resolution.

Memorex 2224
------------

This CD-ROM drive may take up to 3 minutes to recognize the game CD-ROM and start
the game.

There is no known resolution.

Plextor
-------

This CD-ROM drive may slow down to 2X speed when you insert the CD-ROM for the
game.

You may be able to resolve this issue by installing an updated version of the
Plextor Manager 2000 software. To do this, contact Plextor to inquire about how
to obtain and install a version of Plextor Manager 2000 that includes a "No Read
Slowdown" option.

Sanyo CRD-256P
--------------

This CD-ROM drive may be unable to read the CD-ROM for the game if the computer
has not been restarted since the last time you ran the game.

To work around this issue, eject and then reinsert the game CD-ROM before you
attempt to start the game. If the issue continues to occur, restart the computer
before you attempt to start the game.

Sony CDU77E 2X
--------------

This CD-ROM drive may be unable to read the CD-ROM for the game if it is
configured as the slave drive on an IDE channel.

To resolve this issue, configure the CD-ROM drive as the master drive on an IDE
channel.

Yamaha 2216E and CDR6416S
-------------------------

These CD-ROM drives may be unable to read the CD-ROM for the game.

There is no known resolution.

Yamaha 4416SX
-------------

This CD-ROM drive may be unable to read the CD-ROM for the game.

To resolve this issue, contact Yamaha to inquire about how to obtain and install
a firmware upgrade for the CD-ROM drive. The third-party products discussed in
this article are manufactured by vendors independent of Microsoft; we make no
warranty, implied or otherwise, regarding these products' performance or
reliability.
