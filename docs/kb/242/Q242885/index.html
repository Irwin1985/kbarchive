Q242885: AoE II: Game Screen Distorted When Screen Size Is 1280 x 1024

Article: Q242885
Product(s): Microsoft Home Games
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 12-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires II: The Age of Kings, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you change the Screen Size setting to 1280 x 1024 in Age of Empires II: The
Age of Kings, the game screen may be distorted.

CAUSE
=====

This behavior can occur if a Sierra Screaming 3D video adapter is installed in
your computer.

RESOLUTION
==========

To resolve this issue, press ALT+F4 to quit Age of Empires II, and then restart
the program. On the startup screen, click Options, and then select a lower
Screen Size setting.

================
