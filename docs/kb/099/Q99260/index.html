Q99260: PC Forms: Err Msg: Invalid Message

Article: Q99260
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:1.0,3.0,3.2
Operating System(s): 
Keyword(s): 
Last Modified: 17-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Electronic Forms Designer, version 1.0, used with:
   - Microsoft Mail for PC Networks, versions 3.0, 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to delete a message from a shared folder that was created using
version 1.0 of Microsoft Electronic Forms Designer, the following error message
may be displayed:

  Invalid Message

CAUSE
=====

This error can occur when you use version 3.0 of Microsoft Mail for Windows.
This is a known problem with custom message types and shared folders in the
Windows client.

RESOLUTION
==========

Upgrade to version 3.2 of Microsoft Mail for Windows.
