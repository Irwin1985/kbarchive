Q163135: WD97: Word Hangs When Printing Open Document from Open Dialog

Article: Q163135
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbprint kbualink97 word97
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

Word 97 stops responding (hangs) if you attempt to print a document from the
Open dialog box (File menu) while the document is already open in Word.

This problem occurs if you select the Print command from the shortcut menu or the
Commands and Settings menu in the Open dialog box.

WORKAROUND
==========

To work around this problem, use either of the following methods:

- Close the document and then print it from the Open dialog box.

  -or-

- Close the Open dialog box, switch to the open document (if not the active
  document), and then click Print on the File menu.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

This problem has been corrected in Word 2000.

MORE INFORMATION
================

Turning off background printing or print spooling does not resolve this
problem.
