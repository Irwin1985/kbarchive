Q163111: Remote Access Service (RAS) Error Code List

Article: Q163111
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): 2000,4.0,95
Operating System(s): 
Keyword(s): kberrmsg win95 win98
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows 95 
- Microsoft Windows 98 
- Microsoft Windows 2000 Professional 
- Microsoft Windows 2000 Server 
- Microsoft Windows 2000 Advanced Server 
- Microsoft Windows 2000 Datacenter Server 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the error codes that Remote Access Service (RAS) reports in
Windows NT 4.0 and Windows 95/98.
