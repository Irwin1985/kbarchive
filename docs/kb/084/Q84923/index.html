Q84923: Vector Fonts Don’t Print When Printing TrueType As Graphics

Article: Q84923
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

In Microsoft Windows version 3.1, vector fonts (Windows ships with Modern,
Roman, and Script vector fonts) do not print on Hewlett-Packard LaserJet Series
printers if the Print TrueType As Graphics option is selected in the printer
driver Options dialog box.

MORE INFORMATION
================

Printing Vector Fonts
---------------------

To print vector fonts to an HP LaserJet Series II (HPPCL.DRV) or Series III
(HPPCL5A.DRV) printer:

1. Choose the Printers icon in the Windows Control Panel.

2. Select the printer in the Installed Printers box.

3. Choose the Setup button.

4. Choose the Options button.

5. Clear the Print TrueType As Graphics check box.

6. Choose the OK button.

7. Choose the OK button in the Setup dialog box.

8. Choose the Close button in the Printers window.

9. Close the Control Panel.

STATUS
======

Microsoft has confirmed this to be a problem in Windows version 3.1. We are
researching this problem and will post new information here as it becomes
available.

REFERENCES
==========

"Microsoft Windows Resource Kit" for operating system version 3.1, pages 315,
317
