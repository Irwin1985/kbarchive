Q82093: HP DeskJet 500C Memory and Hard Disk Requirements

Article: Q82093
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 07-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SUMMARY
=======

When you are printing with Microsoft Windows version 3.0, make sure that your
system meets the memory and disk requirements needed for the Hewlett- Packard
(HP) DeskJet 500C printer driver version 2.0.

MORE INFORMATION
================

The following is a summary of the necessary requirements to print to the HP
DeskJet 500C printer driver version 2.0.

1. To print in the Black Only Print Mode, you need the following:

   - Standard Black Blending Method: 1 MB (640K of conventional memory plus
     384K of extended memory)

  -or-

   - Grayscale Pattern or Grayscale Scatter Black Blending Method: 2 MB (640K
     of conventional memory plus 1408K of extended memory)

2. To print in the Color/Composite Black Print Mode, you need 2 MB (640K of
  conventional memory plus 1408K of extended memory)

3. To install the printer driver and the three printer fonts that come with the
  printer (Courier, CG Times, Univers), you need 5 MB of free hard disk space.

Three files that are required to print with the HP 500C driver are the
following:

  Filename        File Size     Date
  --------        ---------     ----
  DJ500C.DRV      192196        10/11/91
  DJ500C.HLP       36730        10/11/91
  DJCOLOR1.HPM     61132        10/11/91

For more information, contact the Hewlett-Packard Personal Peripherals Assistance
Line.

The HP DeskJet 500C is manufactured by a vendor independent of Microsoft; we make
no warranty, implied or otherwise, regarding this product's performance or
reliability.
