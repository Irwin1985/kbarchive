Q198306: FS98/2000: Flight Simulator Starts Minimized

Article: Q198306
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbenv kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
- Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Flight Simulator 98 or Microsoft Flight Simulator 2000
from the Start menu, Flight Simulator may start as a minimized program
(appearing as a button on the taskbar).

In addition, Flight Simulator may stop responding (hang) when you quit the game.

CAUSE
=====

This behavior can occur if you quit Flight Simulator when it is running
minimized.

RESOLUTION
==========

To resolve this issue:

1. Right-click Start, and then click Open.

2. Double-click the Programs folder.

3. Double-click the Microsoft Games folder.

4. Double-click the Flight Simulator 98 or Flight Simulator 2000 folder.

5. Right-click Flight Simulator 98 or Flight Simulator 2000, and then click
  Properties.

6. Click the Shortcut tab.

7. In the Run box, click "Normal window," and then click OK.

8. Close all open windows.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

This behavior can also occur if you attempt to shut down the computer while
Flight Simulator is running minimized. In this case, your computer may hang or
display a black screen after you see the "Please wait while your computer shuts
down" or "Windows is shutting down" screen.
