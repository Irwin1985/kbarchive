Q198926: PRB: Visual C++ Not Tested for Windows NT Checked Build

Article: Q198926
Product(s): Microsoft C Compiler
Version(s): 2.0,2000,4.0,5.0,6.0
Operating System(s): 
Keyword(s): kbide kbOSWinNT kbOSWin2000 kbVC200 kbVC400 kbVC500 kbVC600
Last Modified: 02-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual C++, 32-bit Editions, versions 2.0, 4.0, 5.0, 6.0, on platform(s):
   - the operating system: Microsoft Windows NT 
- Microsoft Visual C++.NET (2002) 
- Microsoft Windows 2000 Professional 
- Microsoft Windows 2000 Server 
- Microsoft Windows 2000 Advanced Server 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may get an access violation during installation or during start up of
Developer Studio.

CAUSE
=====

The 32-bit versions of Visual C++ are not designed or tested to work under
Windows NT Checked Build Operating System versions.

STATUS
======

This behavior is by design.
