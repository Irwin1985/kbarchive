Q308546: Err Msg: Link to the Room You Have Clicked on Is Out of Date

Article: Q308546
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zone.com 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click a link to a game room or lobby on the Microsoft Zone.com Web
site, you may receive the following error message:

  The link to the room that you have clicked on is out of date. Please refresh
  the current Web page in your Web browser, and then try to reconnect.

CAUSE
=====

This behavior can occur if the game room or lobby links on the Zone.com Web site
have recently changed, and your browser is still pointing to the old links.

RESOLUTION
==========

To resolve this issue, wait for the links to be updated in your Web browser, or
try clicking a different game room or lobby for the particular game that you
want to play.

To ensure that outdated Zone pages are not stored on your computer, use the
following for your Web browser.

Microsoft Internet Explorer 4.x and Later
-----------------------------------------

1. On the View menu (Internet Explorer 4.<x>) or Tools menu (Internet
  Explorer 5.0), click Internet Options.

2. Under Temporary Internet Files, click Delete Files, and then click Yes or OK.

3. Click Settings.

4. Under "Check for newer versions of stored pages," click "Every time you start
  Internet Explorer."

5. Click OK twice.

6. Quit and then restart Internet Explorer.

7. Connect to the Zone.
