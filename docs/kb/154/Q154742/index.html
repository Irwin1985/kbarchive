Q154742: Codewiz.exe the Xbase Code Import Wizard for VFP

Article: Q154742
Product(s): Microsoft FoxPro
Version(s): WINDOWS:3.0,3.0b,5.0
Operating System(s): 
Keyword(s): kbfile kbDatabase kbvfp300 kbvfp300b kbvfp500
Last Modified: 24-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Codewiz.exe is a file that allows you to convert Xbase code written in Clipper,
FoxPro, FoxBASE, or dBASE into a Visual FoxPro form. It converts the following
Xbase commands into their equivalent objects in Visual FoxPro:

     @ ... SAY
     @ ... GET
     USE

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  Codewiz.exe
  (http://download.microsoft.com/download/fox26win/Utility/2/WIN98/EN-US/Codewiz.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Codewiz.exe contains the following files:

- Readme.txt

- Nwizard.dbf

- Nwizard.fpt

- Wzxbvfp3.app

- Wzxbvfp5.app
