Q253789: PRB: Screen Saver Activated While Playing Back MPEG Movie File

Article: Q253789
Product(s): Microsoft Windows NT
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbOSWin2000 kbDSupport
Last Modified: 24-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 2000 Advanced Server 
- Microsoft Windows 2000 Server 
- Microsoft Windows 2000 Professional 
-------------------------------------------------------------------------------

SYMPTOMS
========

While you watch a Moving Picture Expert Group (MPEG) movie on a Windows
2000-based computer, the Screen Saver may activate and you may no longer be able
to see or hear the movie.

CAUSE
=====

Windows 2000 does not prevent the Screen Saver from activating while you watch
an MPEG movie.

RESOLUTION
==========

The following are ways to easily workaround this problem:

- Disable the Screen Saver before you start an MPEG movie.

- Set the Screen Saver's activation time to a time that is longer than the
  length of the movie.
