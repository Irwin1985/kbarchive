Q253750: Asheron’s Call Error Msg: Correct Version Is Not Installed

Article: Q253750
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbtool igz kbimu msgame
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call, version 1.0 
- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Asheron's Call, you may receive the
following error message:

  You do not have the correct version of Asheron's Call installed.

CAUSE
=====

This behavior can occur if one or more files in the Asheron's Call folder are
damaged or missing.

RESOLUTION
==========

To resolve this issue, remove and then reinstall Asheron's Call:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. Click Microsoft Asheron's Call, and then click Add/Remove.

4. Follow the instructions on the screen to remove Asheron's Call.

5. Click Start, point to Find, and then click "Files or Folders".

6. In the Named box, type the following line (including the quotation marks):

  "Asheron's Call"

7. In the "Look in" box, click My Computer, and then click Find Now.

8. On the Edit menu, click Select All.

9. On the File menu, click Delete.

10. When you are prompted to confirm the multiple file deletion, click Yes.

11. Close the Find: Files Named "Asheron's Call" window.

12. Insert the Asheron's Call CD-ROM into the CD-ROM drive.

13. Follow the instructions on the screen to reinstall Asheron's Call.

14. Start Asheron's Call and automatically update the necessary files. To do
  this, visit the following Zone.com Web site:

Asheron's Call:

  http://zone.msn.com/asheronscall/minilaunch2.asp

Asheron's Call - Dark Majesty:

  http://zone.msn.com/asheronscallxp/minilaunch2.asp
