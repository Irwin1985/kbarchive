Q149081: How to Properly Reinstall MSN Version 1.2

Article: Q149081
Product(s): The Microsoft Network
Version(s): WINDOWS:1.0,1.05,1.2
Operating System(s): 
Keyword(s): 
Last Modified: 12-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05, 1.2 
-------------------------------------------------------------------------------

SUMMARY
=======

There are some situations that may require you to reinstall MSN, The Microsoft
Network. This article describes how to reinstall MSN version 1.2 properly. You
can also use the steps in this article to install MSN version 1.2 on a computer
that did not previously have MSN version 1.2 installed.

MORE INFORMATION
================

Use the following steps to reinstall MSN version 1.2:

1. In Control Panel, double-click Add/Remove Programs. On the Windows Setup tab,
  click the The Microsoft Network check box to clear it, and then click OK.

2. If Microsoft Plus! for Windows 95 is installed, follow these steps:
  a. In Control Panel, double-click Add/Remove Programs.

  b. On the Install/Uninstall tab, click Microsoft Plus! For Windows 95, and
     then click Add/Remove.

  c. Click Add/Remove.

  d. Click the Internet Jumpstart Kit check box to clear it, and then click
     Continue.

  e. When you are prompted to confirm the removal, click Yes.

     NOTE: If you have downloaded and installed any Microsoft Internet Explorer
     updates, you do not have to remove the update. You need to remove only the
     Internet Jumpstart Kit.

3. Reinstall MSN. To do so, follow the instructions in step 1. Click the The
  Microsoft Network check box to select it (instead of clearing it).

4. If Microsoft Plus! is installed, reinstall the Internet Jumpstart Kit. To do
  so, follow the instructions in step 2. In step D, click the Internet
  Jumpstart Kit check box to select it (instead of clearing it).

  When you reinstall the Internet Jumpstart Kit, Internet Explorer version 1.0
  is installed. If you have downloaded and installed an update to Internet
  Explorer, reinstall the update now.

  NOTE: If you reinstall the Internet Jumpstart Kit out of the order outlined in
  these steps, you may be prompted to update to MSN version 1.2 when you
  connect to MSN even if you have already done so. If this occurs, see the
  following article in the Microsoft Knowledge Base for instructions:

  Q141370 Installing Microsoft Plus! After MSN 1.2 Overwrites Msnver.txt

5. Double-click the MSN icon on the desktop to connect to MSN. Once the
  connection is established, you are prompted to upgrade to MSN version 1.2.
  Allow MSN to update your computer to version 1.2.

  NOTE: You cannot use MSN until you upgrade to version 1.2.
