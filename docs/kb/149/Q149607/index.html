Q149607: Must Restart After Installing Microsoft Phone on Presario

Article: Q149607
Product(s): Microsoft Mail For PC Networks
Version(s): 1.0
Operating System(s): 
Keyword(s): win95 msphone1
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Phone, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install the Microsoft Phone software on a Compaq Presario model 520,
720, or 920 computer, the following error message may be displayed if you start
Microsoft Phone before you restart your computer:

  Phone is unable to access the voice commands. Try closing Phone,
  restart your computer, and try again.
  Win32 error (module SPEECH).
  The program has selected a non-existent speech engine.

If you restart your computer and then start Microsoft Phone again, the error
message is not displayed, but the Answering Machine is turned off.

CAUSE
=====

Compaq Presario model 520, 720, and 920 computers include a voice phone that is
supported by the Microsoft Phone software. However, after you install the
Microsoft Phone software on these computers, you must restart the computer for
the Microsoft Phone software to function properly.

RESOLUTION
==========

If you receive the error message stated above the first time you start Microsoft
Phone on a Compaq Presario model 520, 720, or 920 computer, restart your
computer, and then start Microsoft Phone again. To turn on the Answering Machine
after you start Microsoft Phone, follow these steps:

1. In Microsoft Phone, click Phone Properties on the Options menu.

2. Click the Answer tab.

3. Click the Turn On Answering Machine check box to select it.
