Q149827: Explorapedia Nature: How to Run Setup

Article: Q149827
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0,1.2
Operating System(s): 
Keyword(s): 
Last Modified: 30-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Explorapedia series: World of Nature for Windows, versions 1.0, 1.2 
-------------------------------------------------------------------------------

SUMMARY
=======

The bundle of Explorapedia World of Nature (Nature) and the Microsoft EasyBall
does not contain instructions about how to install World of Nature. This article
provides the instructions to install World of Nature.

MORE INFORMATION
================

Installing World of Nature
--------------------------

To install Explorapedia World of Nature, use the instructions below for your
version of Windows:

Windows 95/98
-------------

1. Insert the World of Nature CD-ROM into the CD-ROM drive, click Start, and
  then click Run.

2. In the Open box, type the following line, and then click OK

  "<d>:\setup.exe" (without the quotation marks)

  where <d> is the letter of your CD-ROM drive.

3. Follow the instructions on the screen to finish the program installation.

Windows 3.x
-----------

1. On the File menu in Program Manager, click Run.

2. On the Command Line, type the following line, and then click OK

  "<d>:\setup.exe" (without the quotation marks)

  where <d> is the letter of your CD-ROM drive.

3. Follow the instructions on the screen to finish the program installation

Additional Installation Information
-----------------------------------

World of Nature version 1.2:

Your Explorapedia disk includes two versions of Explorapedia:

- A 16-bit version for Windows 3.x

- A 32-bit version for Windows 95/98 and Windows NT 3.51 and later.

Setup automatically installs the appropriate Explorapedia software for your
operating system. The version of the software for Windows 95/98 and Windows NT
is most appropriate for those operating systems. If you upgrade your Windows 3.x
operating system to Windows 95/98 or Windows NT, you should run Explorapedia
Setup again to install the 32-bit version of the software.

For more information about Explorapedia World of Nature, version 1.2, please see
the following article in the Microsoft Knowledge Base:

  Q149825 Explorapedia Nature: Contents of Version 1.2 Readme.wri

---------------------------
