Q274540: Windows ME (Millennium Edition) Step by Step Comments and Correc

Article: Q274540
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Windows Me Step by Step ISBN 0-7356-0990-X 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Windows Me (Millennium Edition) Step by
Step, ISBN 0-7356-0990-x.

The following topics are covered:

- Index: Entry Page-range Errors

- Page 222: Typographical Error

MORE INFORMATION
================

Index: Entry Page-range Errors
------------------------------

Several of the entries in the index have the incorrect pages referenced. The
correct page numbers are listed below.

Page 241:
Add/Remove Programs feature, 239-40

Page 244:
desktop/arranging icons, 235-36

Page 246:
folders/My Documents folder, 5, 6, 235-36

Page 247:
icons/arranging on desktop, 235-36

Page 250:
mouse/changing configuration, 234
mouse/checking settings for SBS exercises, 233-234

Page 257:
windows/arranging icons, 235-36
windows/changing sizes, 235
windows/changing views, 235
Windows Me/adding components after installation, 239-40
Windows Me/installation overview, 237-38
Windows Me/installing on blank hard disk, 238
Windows Me/installing over earlier version, 238-39
Windows Me/setup options, 238


Page 222:  Typographical Error
------------------------------

In the second paragraph, 5th Line on Page 222 in the section "Installing Other
Hardware", change:

"and then run the hardware;"

To:

"and then run the software;"


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.

Additional query words: EUBOOK WinMe CATAPULT
