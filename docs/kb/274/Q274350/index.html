Q274350: Crimson Skies: Known Video Issues in Crimson Skies

Article: Q274350
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kb3rdparty kbgraphic kbhw kbimu kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Crimson Skies 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes some of the known video related issues in Microsoft
Crimson Skies.

MORE INFORMATION
================

The following list contains known video related issues in Crimson Skies.

- Video adapters based on the 3dfx Voodoo Banshee chip set (such as the Diamond
  Monster Fusion video adapter)

  If a video adapter that is based on the 3dfx Voodoo Banshee chip set is
  installed in your computer, the textures in the game may appear distorted,
  may flash white, or may be missing.

  To resolve this issue, download and install the latest Voodoo Banshee
  reference driver from the following 3dfx Web site:

  http://www.3dfx.com

- Video adapters based on the 3DLabs Permedia 2 chip set based video adapter

  If a video adapter that is based on the 3DLabs Permedia 2 chip set is
  installed in your computer, you may experience the following issues:
   - Shadows may be distorted.

   - Menus in the game may be distorted.

   - Cockpit instruments may be oddly colored.

  To resolve the issue with distorted shadows, turn off the Shadows option in
  the Video menu in the game.

  To resolve the other issues with the 3DLabs Permedia 2 chip set, contact your
  hardware manufacturer for the latest video driver, or download and install
  the latest reference driver from the following 3DLabs Web site:

  http://www.3dlabs.com

- ATI 3D Rage II+ video adapter

  If an ATI 3D Rage II+ video adapter is installed in your computer, textures in
  the game may not be displayed correctly, visual anomalies may occur in the
  menus in the game, or lines may be displayed on the screen.

  To resolve these issues, install the latest video driver from the following
  ATI Web site:

  http://www.ati.com

- ATI Rage 128 or ATI Rage Fury Maxx video adapter

  If an ATI Rage 128 or an ATI Rage Fury Maxx video adapter is installed in your
  computer, you may experience intermittent problems with the screen.

  To resolve this issue, install the latest video driver from the following ATI
  Web site:

  http://www.ati.com

- ATI Rage GL AGP video adapter

  If an ATI Rage GL AGP video adapter is installed in your computer, the game
  may stop responding (hang) with a black screen when you start a mission.

  To resolve this issue, install the latest video driver from the following ATI
  Web site:

  http://www.ati.com

- ATI Rage Pro video adapter

  If an ATI Rage Pro video adapter is installed in your computer, multiple
  aircraft shadows may be displayed for one aircraft in a multiplayer game.

  To resolve this issue, install the latest video driver from the following ATI
  Web site:

  http://www.ati.com

- ATI Rage Pro turbo AGP video adapter

  If an ATI Rage Pro turbo AGP video adapter is installed in your computer, the
  game may hang with a white screen after you start the game.

  To resolve this issue, install the latest video driver from the following ATI
  Web site:

  http://www.ati.com

- Creative Labs 3D Blaster Riva TNT2 Ultra video adapter

  If a Creative Labs 3D Blaster Riva TNT2 Ultra video adapter is installed in
  your computer, you may not be able to use a screen resolution in the game
  that is higher than 640 x 480.

  To resolve this issue, download and install the latest video driver for your
  video adapter from the following Creative Labs Web site:

  http://www.creative.com

- Creative Labs Graphics Blaster Riva TNT video adapter

  If a Creative Labs Graphics Blaster Riva TNT video adapter is installed in
  your computer, you may experience the following issues:

   - Textures on the screen may be unusually dark or flash in the distance.

   - In Microsoft Windows Millennium, the main menu screen may be distorted.

To resolve this issue, download and install the latest reference driver from
nVIDIA from the following nVIDIA Web site:

  http://www.nvidia.com

- Creative Labs Voodoo 2 video accelerator

  If a Creative Labs Voodoo 2 video accelerator is installed in a Microsoft
  Windows 98-based computer, the game may not start properly.

  To resolve this issue, download and install the latest Voodoo2 reference
  driver from the following 3dfx Web site:

  http://www.3dfx.com

- Diamond Viper TNT AGP video adapter

  If a Diamond Viper TNT AGP video adapter is installed in your computer, your
  aircraft may roll uncontrollably, and the game may be minimized unexpectedly
  when you are playing a mission.

  To resolve this issue, download and install the latest reference driver from
  nVIDIA from the following nVIDIA Web site:

  http://www.nvidia.com

- Guillemot Maxi Gamer Cougar video adapter

  If a Guillemot Maxi Gamer Cougar video adapter is installed in your computer,
  the game may hang when you start a mission.

  To resolve this issue, download and install the latest video driver for your
  video adapter from the following Guillemot Web site or the latest reference
  driver from the following nVIDIA Web site:

  http://www.guillemot.com

  http://www.nvidia.com

- Guillemot Xentor 32 video adapter with a DVD-ROM drive

  If a Guillemot Xentor 32 video adapter and a DVD-ROM drive are installed in
  your computer, you may experience problems if you attempt to start Crimson
  Skies during the opening video clip.

  To resolve this issue, upgrade your DVD-ROM software. If the problems continue
  to occur, uninstall the software DVD player that is included with your video
  adapter.

- Hercules Power Drive Terminator Beast video adapter

  If a Hercules Power Drive Terminator Beast video adapter is installed in your
  computer, multiple aircraft shadows for one aircraft may be displayed on the
  ground.

  To resolve this issue, download and install the latest video driver for your
  video adapter from the following Hercules Web site or the latest reference
  driver for the S3 Savage 3D chip set from the following S3 Web site:

  http://www.hercules.com

  http://www.diamondmm.com

- Matrox Millennium II video adapter

  The Matrox Millennium II video adapter does not have sufficient 3D feature
  support to run Crimson Skies in hardware mode.

  For example, targeting reticules are not correctly blended.

  If the Matrox Millennium II video adapter is installed in your computer, use
  Software mode when you play Crimson Skies.

- Number 9 Revolution IV video adapter

  If a Number 9 Revolution IV video adapter is installed in your computer, you
  may experience the following issues:

   - When you are killed in Crimson Skies, the game may be minimized, and to
     restart the mission you may have to use the options screen.

   - When you quit an Instant Action mission, the game may hang.

To resolve these issues, install the latest video driver from for your video
adapter from the following Number 9 Visual Technology Web site:

  http://www.nine.com

- Number 9 SR-9 Pro video adapter

  If a Number 9 SR-9 Pro video adapter is installed in your computer, the
  terrain in the game may be distorted.

  To resolve this issue, download and install the latest reference driver for
  the Savage 4 chip set from the following Diamond Multimedia Web site:

  http://www.diamondmm.com

- nVIDIA chipset (TNT, TNT2, and GeForce)

  If a video adapter that is based on the nVIDIA TNT, TNT2, or the GeForce chip
  set is installed in a Windows 2000-based computer, you may not be able to
  play the game in accelerated hardware mode.

  This issue can occur if you use the nVIDIA video driver that is included with
  Windows 2000.

  To resolve this issue, contact your hardware manufacturer for the latest video
  driver for your video adapter, or download and install the latest reference
  driver from the following nVIDIA Web site:

  http://www.nvidia.com

- STB Velocity 128 3D AGP video adapter

  If a STB Velocity 128 3D AGP video adapter is installed in your computer, the
  game may hang when you navigate through the user interface.

  To resolve this issue, download and install the latest reference driver from
  nVIDIA from the following nVIDIA Web site:

  http://www.nvidia.com

- STB Velocity 4400 video adapter

  If an STB Velocity 4400 video adapter is installed in your computer, you may
  experience the following issues:

   - The computer may hang at the Main Menu.

   - Textures may become distorted after a few minutes of game play.

   - An error message may be displayed and the game may not start.

To resolve these issues, download and install the latest retail driver for your
video adapter from the following STB Web site or install the latest reference
driver from the following nVIDIA Web site:

  http://www.stb.com/drivers

  http://www.nvidia.com

Contact the Manufacturer of Your Video Adapter
----------------------------------------------

For information about how to contact your hardware manufacturer, click the
appropriate article number below to view the article in the Microsoft Knowledge
Base:

  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K

  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P

  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z

To identify the manufacturer and model of your video adapter:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click System.

3. If you are using a Microsoft Windows 95-based or a Windows 98-based computer,
  click the Device Manager tab.

  If you are using a Windows 2000-based computer, click the Hardware tab, and
  then click Device Manager.

4. Click the plus sign (+) next to "Display adapters" to expand the branch.

5. Under the "Display adapters" branch, note the manufacturer and model of your
  video adapter, and then click OK.

6. Close all open windows.

NOTES:

Reference drivers are drivers that are released by the manufacturers of a
specific chip set that is used by specific video adapters.

Retail drivers are drivers that are released by the manufacturer of a video
adapter or audio card that use a specific chip set.

If you experience issues when you use the most recent retail drivers supplied by
the video adapter's or sound card's manufacturer, use the latest reference
driver from your video adapter's or sound card's chip set manufacturer.

Microsoft recommends that you do not install beta video drivers.
