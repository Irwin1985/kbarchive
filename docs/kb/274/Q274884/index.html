Q274884: CFS2: Cannot Host a Multiplayer Game on Zone.com

Article: Q274884
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbenv kbnetwork kbimu
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to host a multiplayer game of Microsoft Combat Flight Simulator
2 on Zone.com, you may be unable to start the game.

CAUSE
=====

This behavior can occur if you start Combat Flight Simulator 2 before you sign
on to the Zone.

RESOLUTION
==========

To resolve this issue:

1. Quit Combat Flight Simulator 2.

2. Sign off from the Zone.

3. Sign on to the Zone.

4. Enter the Combat Flight Simulator 2 game room, and then host or join a
  multiplayer game.

When the host starts the multiplayer game, the Zone automatically starts Combat
Flight Simulator 2.

Additional query words: 1.00 msgame combatfs cfs2 msngz igz
