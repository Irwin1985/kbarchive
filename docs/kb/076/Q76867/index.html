Q76867: MIDIMAP.INI Edited Through Control Panel

Article: Q76867
Product(s): Miscellaneous Windows Products
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 30-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows with Multimedia Extensions, version 1.0 
-------------------------------------------------------------------------------

The MIDIMAP.CFG file is not a standard profile (.INI) file. This file
was not meant to be edited directly with a text editor like a standard
.INI file.

To change the current patch map for the driver, use the MIDI Map
Editor (in the Control Panel).

Documenting the MIDIMAP.CFG file format was not feasible because of
the number of combinations, error conditions, and so on, involved.

Additional query words: MMWIN 1.0
