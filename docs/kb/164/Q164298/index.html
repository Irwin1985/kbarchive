Q164298: Writer 2: How to Preview and Publish Your Web Pages

Article: Q164298
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kbusagekbfaq
Last Modified: 12-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Creative Writer for Windows, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

After you create your Web pages in Creative Writer 2, you are ready to let the
world see your work. This article describes how to use the Web Publishing Wizard
to post your Web pages to your Internet Service Provider (ISP).

The Web Publishing Wizard is included as an installation option in Creative
Writer Setup.

MORE INFORMATION
================

To preview and publish your Web pages, use the following methods.

Preview Your Web Pages
----------------------

To preview your Web pages in a Web browser before you publish them:

1. On the Creative Writer 2 toolbar, click Ideas (the Red Toolbox), and then
  click Web Page (the WWW button).

  NOTE: The Web Page (WWW) button is only available when you are working with a
  Web Pages project. If the Web Page (WWW) button is unavailable, you need to
  copy your work into a new Web Pages project.

2. Click "Look at Web Page".

3. When you finish previewing your Web pages, close your Web browser.

Publish Your Web Pages
----------------------

To publish your Web pages:

1. On the Creative Writer 2 toolbar, click Ideas (the Red Toolbox), and then
  click Web Page (the WWW button).

  NOTE: The Web Page (WWW) button is only available when you are working with a
  Web Pages project. If the Web Page (WWW) button is unavailable, you need to
  copy your work into a new Web Pages project.

2. Click the Publish Web Page button.

3. Follow the instructions in the Web Publishing Wizard to publish your Web
  pages.

For information about how to publish Web pages from Creative Writer, please see
the online Help file for the Web Publishing Wizard, or the "Making Web Pages"
and "Web Publishing" sections in the Readme.doc file located on the Creative
Writer 2 CD-ROM.

NOTE: To view the Web Publishing Wizard Help file, you must insert the Creative
Writer 2 CD-ROM into the CD-ROM drive.

If you experience problems when you attempt to publish your Web pages, call
Microsoft Windows technical support at (425) 637-7098 to request assistance with
the Web Publishing Wizard.
