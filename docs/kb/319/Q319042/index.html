Q319042: How To Modify Location Settings In MSN Internet Access for Macin

Article: Q319042
Product(s): The Microsoft Network
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 12-APR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Internet Access for Macintosh, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to modify Location Settings in MSN Internet Access
version 1.0 for Macintosh.

MORE INFORMATION
================

The minimum requirements for connecting to MSN Internet Access are as follows:

- Macintosh Operating System 9.x

- A paying MSN Internet Access account

- A modem with the minimum baud rate of 28.8 Kbps

- Internet Explorer 5.x

- Outlook Express 5.0.4

To modify Location Settings in MSN Internet Access version 1.0 for Macintosh

1. Start MSN Internet Access.

2. On the MSN Internet Access window, click Change Settings.

To edit a current Location

1. Select the Location you want to modify (for example, Home).

2. Click Edit Location.

3. After Location name:, type the label you wish to use to identify the
  Location.

4. Select your Connection Type.
  If you are using a broadband connection, click Broadband connection, and then
  click OK to close the Edit Location window.

5. Click Dial-up connection, by using a phone line and a modem.

6. Type the number of the telephone line from which you will be dialing.

7. Select any additional dialing properties as necessary, such as Dial 9, to
  reach an outside line or Dial *70, to disable call waiting.

8. Click OK to close the Edit Location window.

9. Click OK to close the Change Settings window.

Your MSN software will locate the best local number ("poptimize") the next time
you attempt sign in to MSN. To add a location

1. Click Add Location.

2. After Location name:, type the label you wish to use to identify the
  Location.

3. Select your Connection Type.
  If you are using a broadband connection, click Broadband connection and then
  click OK to close the Edit Location window.

4. Click Dial-up connection, by using a phone line and a modem.

5. Type the number of the telephone line from which you will be dialing.

6. Select any additional dialing properties as necessary, such as Dial 9, to
  reach an outside line or Dial *70 to disable call waiting.

7. Click OK to close the Edit Location window.

8. Click OK to close the Change Settings window.
  Your MSN software will locate the best local number ("poptimize") the next
  time you attempt sign in to MSN.
