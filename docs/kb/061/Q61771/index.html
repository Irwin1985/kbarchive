Q61771: Hewlett-Packard TextEquations Font Cartridge Description

Article: Q61771
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a description of the capabilities of the Hewlett-Packard (HP)
TextEquations Font Cartridge.

The font cartridge supports a print resolution of 300 x 300 dots per inch (dpi)
and has an average lifetime of 500 insertions. Separate fonts are required for
landscape and portrait. Some fonts may not be available in both orientations on
a specific cartridge.

For more information about this font cartridge, please contact Hewlett-Packard.

MORE INFORMATION
================

The following explains TextEquations Cartridge capabilities:

Typeface        Type    Orient.   Pitch          Point  Symbol Set
--------        ----    -------   -----          -----  ----------

Prestige Elite  Medium  Portrait  16.66          7      Roman-8,Math-8

Prestige Elite  Medium  Portrait  12             10     Roman-8,Math-8
                                                        Math-7,Pi Font

Prestige Elite  Bold    Portrait  12             10     Roman-8

Prestige Elite  Italic  Portrait  12             10     Roman-8

CG Times        Medium  Portrait  Proportional   8      Roman-8,Math-8

CG Times        Medium  Portrait  Proportional   10     Roman-8,Math-8
                                                        Math-7,Pi Font

CG Times        Bold    Portrait  Proportional   10     Roman-8

CG Times        Italic  Portrait  Proportional   10     Roman-8
