Q189925: Golf 2.0: Update Available to Improve Swing Control

Article: Q189925
Product(s): Microsoft Home Games
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kbenv kbimu msgame
Last Modified: 21-DEC-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf for Windows, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

While you are playing Microsoft Golf, you may experience inconsistent swing
control.

CAUSE
=====

This problem can occur for either of the following reasons:

- Golf does not correctly identify the operating system you are using.
- You are using a power-saving feature on your notebook computer.

RESOLUTION
==========

An updated Glfani32.dll file is available that corrects this problem.

The following file is available for download from the Microsoft Download Center:

  Gm1170.exe
  (http://download.microsoft.com/download/golf20win/Update/1/WIN98/EN-US/Gm1170.exe)

Release Date: May-22-1995

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

MORE INFORMATION
================

To install the updated Glfani32.dll file, follow these steps:

1. Quit Golf if it is running.

2. Click Start, point to Find, and then click Files Or Folders.

3. In the Named box, type "glfani32.dll" (without the quotation marks), and then
  click Find Now.

4. In the list of found files, right-click the Glfani32.dll file, and then click
  Delete.

5. Double-click the Gm1170.exe file to extract the updated Glfani32.dll file.

6. Drag the updated Glfani32.dll file into the Msgolf2 folder.

  NOTE: The default location of this folder is C:\Msgolf2.

7. Start Golf. When you start Golf, the game profiles your computer for maximum
  swing efficiency.

  NOTE: Golf profiles your computer each time you change the screen resolution
  or make a major system modification.

Additional query words: 2.00 swings downloads fix patch
