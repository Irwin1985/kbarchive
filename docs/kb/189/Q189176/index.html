SUMMARY
=======

The following article lists available resources for BackOffice Customer Support.
Included is information on Web links from which to download and view information
on products, services, and troubleshooting.

MORE INFORMATION
================

Almost anything that Microsoft BackOffice customers need they can find at the
following Web site. Information about products, sales, services, support, and
the latest Microsoft news. From here, customers have information at their
fingertips.

  http://www.microsoft.com/servers/

The following URL takes customers directly to the Microsoft Personal Support
Center Web page. From here, you can search the Microsoft Knowledge Base, use
troubleshooters, download files, view FAQs and Support Highlights, and join
newsgroups.

  http://msdn.microsoft.com/support

Also, customers can sign up for the Support News Watch service. After you sign
up, Microsoft sends you e-mail notifications of the most critical support
issues.

Current Support News Watch categories:

  BackOffice
  Developer
  Games, Reference, and Kids' Products
  Office
  Windows and Internet Explorer

The URL below takes you to a page that allows customers to subscribe to a variety
of newsletters that are pushed to their desktops through e-mail. The BackOffice
News subscription sends you additional information and Web links, special
promotions, conference information, and tips.

  http://register.microsoft.com/regwiz/forms/pic.asp

The following is a URL that will take you to the Microsoft Servers Web page. This
page has updated news about BackOffice products, product information, a download
and trial center, and access to a BackOffice community.

  http://www.microsoft.com/servers/

Click the link below to go to the Seminar Online Web page. From here, you can
attend a seminar on the Internet. You choose which seminars you want to view,
right from your desktop. Some examples are Microsoft Windows NT Security
Architecture and Directions, Microsoft BackOffice Server 4.0 Overview, and
Microsoft Windows NT and UNIX interoperability: Tools and Technologies.

  http://www.microsoft.com/seminar/

The following URL points you a Web page dealing with the top BackOffice issues
and Support Highlights. The following can be accessed from this page:

- Microsoft BackOffice

- Link to the Year 2000 Resources Available from Microsoft

- Microsoft Windows NT

- Links to the top issues for Microsoft Windows NT Setup, Macintosh,
  Applications, Printing, and IPX, Domains, and TCP/IP and RAS

- Microsoft Internet Information Server

- Links to the top issues for Microsoft Internet Information Server (IIS)

- Microsoft Exchange

- Links to the top issues for Microsoft Exchange clients, administration, and
  connectors

- Microsoft SQL Server

- Links to the top issues for Microsoft SQL Server

- Microsoft Systems Management Server

- Links to the top issues for Microsoft Systems Management Server

  http://support.microsoft.com/support/BOTopIssues/

This is the Technet IT appfarm, which has lots of Exchange Server tools and
programming tips. This site provides a central clearinghouse for over 40 custom
Microsoft Exchange applications.

  http://www.microsoft.com/ithome/resource/exchange/default.htm

For Microsoft Certified Partners only
-------------------------------------

This secured page contains information about products, marketing promotions, and
a user group of other Microsoft Certified Partners. To access this page, the
Microsoft Certified Partner must use his or her Microsoft Certified Partner ID#
for the User Name and the last name of his or her primary contact as the
password.

  http://members.microsoft.com/certpartner/
