Q105924: Determining Version of Windows Sound System 1.0x Software

Article: Q105924
Product(s): Miscellaneous Windows Products
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 20-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Sound System, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

There are three ways to determine which version of the Windows Sound System
software you are running:

1. Start Windows and open the Windows Sound System group. If you have a Games
  Configuration icon, you are using version 1.0a of Windows Sound System
  software.

-or-

2. Start Windows and open the Windows Sound System group.

  a. Start Quick Recorder.

  b. From the Help menu, choose About. The dialog box will report either
     version 1.0 or 1.0a.

  c. Start Volume Control, Recording Control, or Sound Finder.

  d. From the System menu on the top-left corner, choose About. The dialog box
     will report either version 1.0 or 1.0a

-or-

3. Compare the file and file sizes (the following file details are for 1.0a).

  a. Check the Windows Sound System Setup Disk 1 for the file:

        GAMECNFG EX_     11796 02-17-93   1:00a

  b. Check the C:\SNDSYS directory for the file:

        GAMECNFG.EXE     23049 02-17-93   1:00:10am

Additional query words: 1.00 1.00a 1
