Q197682: WD97: After Install of ATI Driver, Text Looks Blocked or Corrupt

Article: Q197682
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

After you install or upgrade your ATI display driver, text in your Word document
may appear blocked or corrupted. This problem may occur with certain fonts and
more frequently in larger font sizes.

CAUSE
=====

You may have installed ATI Rage II / II+ display driver version 5.21 or 5.24.
This is a known problem with these ATI display drivers. For more information,
please contact ATI Technical Support.

WORKAROUND
==========

To work around this problem, turn off the "Smooth edges of screen fonts" option
in Windows.

To turn off the "Smooth edges of screen fonts" option, use the following steps:

1. Quit Word.

2. On the Start menu, point to Settings and then click Control Panel.

3. Double-click the Display icon.

4. On the Effects tab, click to clear the "Smooth edges of screen fonts" check
  box, and then click OK.

STATUS
======

This problem may occur when you use Microsoft Word 97 for Windows on either
Microsoft Windows 95 or Microsoft Windows 98, and ATI Rage II/II+ display driver
versions 5.21 or 5.24.

NOTE: This problem may also occur in other Windows applications.
