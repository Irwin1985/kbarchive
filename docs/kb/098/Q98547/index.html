Q98547: Cannot Print More Than 99 Copies with Some Printer Drivers

Article: Q98547
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print using one of the printer drivers listed below, 99 is the maximum
number of copies you can print using the Copies setting in the printer driver
Setup dialog box. Although you can type a larger number in the Copies box, a
maximum of 99 copies actually print.

  Driver           Versions
  ------           --------
  HPPCL.DRV        1.0, 1.1, and 2.0
  HPPCL5A.DRV      31.3.89
  HPPCL5MS.DRV     2.0 and 2.1

STATUS
======

We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

When you use the Microsoft PostScript, Hewlett-Packard (HP) DeskJet, Epson
dot-matrix, and Panasonic printer drivers, you can print more than 99 copies
using the Copies setting in the printer driver Setup dialog box.
