Q98643: Paste Command Dimmed After Choosing Copy or Cut in Encarta

Article: Q98643
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
- Microsoft Multimedia Viewer 
-------------------------------------------------------------------------------

SUMMARY
=======

If the Paste command is unavailable (dimmed) after you copy an image from
Microsoft Encarta, you may be experiencing a video driver conflict.

RESOLUTION
==========

To avoid this problem, use any of the following three workarounds:

- If the video driver is from a third party video card manufacturer, verify
  that it is the latest version of the driver.

- Use an alternate driver supplied by the manufacturer. Check Windows Setup for
  available video drivers.

- Install the Windows VGA driver.
