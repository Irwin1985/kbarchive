SUMMARY
=======


You can use keywords to easily query for specific topics in the Microsoft
Knowledge Base. This article contains keywords for Microsoft Mail and related
products. By using these keywords, you can quickly access common topics. These
keywords are commonly found at the bottom of articles on the line that begins,
"KBSubcategory."

In addition to these keywords, all article titles for Microsoft Mail and related
products are in the following format

  <prod> <category>: <title>

where <prod> is PC or Mac, and <category> is one of the categories
listed below. For example:

  PC Win: <title>
  Mac Srv: <title>
  PROFS: <title>
