Q106067: PC WRmt: Using Mail Remote with Postoffice in Another Language

Article: Q106067
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:3.2
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Remote for Windows, version 3.2 
-------------------------------------------------------------------------------

SUMMARY
=======

To use version 3.2 of Microsoft Mail Remote for Windows with a postoffice in a
different language, the foreign postoffice must first install the Microsoft Mail
for PC Networks MS-DOS workstation in the same language as Mail Remote.

For example, to use the French version of Mail Remote on a German postoffice,
perform the following steps:

1. Install the French version of MS-DOS client on the German postoffice.

2. Use the Regenerate command on the Remote menu before creating the data disk
  for the French version of Mail Remote.

This is required in order for Mail Remote to find the correct template files.
