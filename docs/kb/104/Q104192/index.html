Documentation Error: The PACKING.LST file included on Disk #1 of the 720K disk
set incorrectly lists the following files on Disk #2 as well as on Disk #1:

  AUTOEXEC.BAT
  CONFIG.SYS
  SETUP.MSG

The above files are on Disk #1 only.

The following files are not included on the 720K or 360K disk sets:

  COUNTRY.ICE
  KEYBOARD.ICE
  EGA.ICE
  SMARTMON.EXE
  SMARTMON.HLP
