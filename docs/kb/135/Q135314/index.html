Q135314: Windows 95 Driver Library

Article: Q135314
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): kbfile win95 kbgraphxlinkcritical
Last Modified: 02-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

The Windows 95 Driver Library driver information in this article was revised May
17, 1996. Obtain the most recent version of the Windows 95 Hardware
Compatibility List, Hcl95.exe, from the following site:

  http://www.microsoft.com/hwtest/

Obtain the most recent versions of the Windows
95 Driver Library drivers from the Microsoft Download Center.

SUMMARY
=======

Microsoft has made the Windows 95 Driver Library (W95DL) available. The W95DL
contains drivers for many printers, display adapters, sound cards, and network
adapters not provided with Windows 95. If you have a modem, the drivers are
available for free on the Microsoft Download Center. Please note that standard
connect-time fees and long-distance telephone charges, if any, apply during
downloading.

NOTE: The drivers included in this library are intended for use with Windows 95.
These drivers are not recommended for use with earlier versions of Windows.

MORE INFORMATION
================

Instructions for Downloading
----------------------------

1. Locate the device you want in the listing included at the end of the
  W95dl.txt file. Note the name of the file listed next to the device.

2. If you are downloading to a floppy disk, you need a formatted, blank disk. If
  you are downloading to your hard disk, create a new folder in which you can
  place the file.

  CAUTION: Do not download files directly into your Windows folder. Doing so
  could overwrite files essential to the proper operation of your system.

3. Follow the downloading procedure used by your service to retrieve the file
  you identified in step 1. This file contains all the files you need to
  support your device.

If you have problems extracting downloaded files, try downloading them again.

How to Install New Drivers
--------------------------

To install a new driver (never before installed), follow these steps:

1. Download a file using the instructions in the "Availability" section of this
  article.

2. Use Windows Explorer or My Computer to double-click the file you downloaded.
  Double-clicking the file extracts the files it contains.

3. Click the Start button, point to Settings, and then click Control Panel.

4. Double-click the Add New Hardware icon, and then click Next.

5. Click the No option button, and then click Next.

6. Click the type of hardware for the driver you are installing, and then click
  Next.

7. Click the Have Disk button. Type the location of the files you extracted in
  step 2, click OK, and then follow the instructions on the screen.

How to Upgrade Existing Drivers
-------------------------------

To upgrade an existing driver with a Windows 95 driver, follow these steps:

1. Download a file using the instructions in the "Availability" section of this
  article.

2. Use Windows Explorer or My Computer to double-click the file you downloaded.
  Double-clicking the file extracts the files it contains.

3. Use the right mouse button to click My Computer, and then click Properties on
  the menu that appears.

4. On the Device Manager tab, double-click the branch containing the device you
  want to update, and then double-click the device.

5. On the Driver tab, click the Change Driver button.

6. Click the Have Disk button. Type the location of the files you extracted in
  step 2, click OK, and then follow the instructions on the screen.

NOTE: To upgrade your network adapter driver, you must use the Network tool in
Control Panel.
