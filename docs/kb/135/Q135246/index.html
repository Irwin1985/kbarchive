SUMMARY
=======

"Mastering Microsoft Visual FoxPro" (MVFP) is a comprehensive computer- based
training (CBT) tool for experienced database developers that includes narrated
demonstrations. This product is not an introduction to database design,
programming or the Xbase language. The lessons in this CBT assume some expertise
in these areas.

MVFP is an add-on sale in the Visual FoxPro 3.0 direct mailing. It is designed as
a single-user product. It can be run over a network only if the appropriate
number of client licenses have been purchased.

It will be available internationally through an international Store-Keeping Unit
(SKU), but it will not be localized.

The content of the course applies to Microsoft Visual FoxPro for Macintosh, with
the exception of the information relating to OLE Automation. That is
accomplished on the Macintosh using AppleScript.

The course runs only on personal computers compatible with the System
Requirements described in this article.

Product Description
-------------------


A content-rich course and sample code library that helps you try out new concepts
as you learn them. Paste useful sample code into your own applications, and use
a flexible browsing and searching system to navigate right to the most relevant
reference material, such as articles, frequently asked questions (FAQs), and
technical notes.

You can even test your knowledge with self-tests of various tasks. It is
excellent preparation for the Microsoft Certified Professional exam. The result
is less time spent learning and more time putting Visual FoxPro to work on your
sophisticated business solutions.

- Apply the core set of concepts, commands, and functions used in
  object-oriented programming.

- Create classes and subclasses visually.

- Manage and control data in multiuser situations.

- Build client-server solutions that query and update remote data.

- Drive Microsoft Office applications from Visual FoxPro using OLE Automation.

- Create tables and databases visually, with Table and Database Designers.

- Master design principles to create smart, efficient queries, views, and
  reports.

- Drag and drop classes onto forms for easier use and better design.

- Enhance grid functionality.

- Manage all application components efficiently with the Visual FoxPro Project
  Manager.

Topics Covered in the Course
----------------------------

- Application Designer, Table Designer, and Database Designer.

- Object Model.

- Visual Classes.

- Creating Forms.

- Creating Reports.

- Creating Queries and Views.

- OLE and OLE Automation.

- Client-Server Model.

- Error Handling and Debugging.

- Multiuser Databases.

- Creating Menus and Toolbars.

- Project Manager; Converting 2.x to 3.0; Data Dictionary.

- DLLs and .API Files.

- Wizards and Builders.

To get the most out of this comprehensive training tool, you'll want Microsoft
Visual FoxPro, Professional Edition.

System Requirements
-------------------

- Personal computer with a 486DX or higher processor running Microsoft Windows
  version 3.1, Windows 95, or Microsoft Windows NT Workstation operating system
  version 3.5 or later.

- 8 MB of memory (16 MB recommended if running this application simultaneously
  with Microsoft Visual FoxPro, Professional Edition).

- 10 MB of available hard-disk space.

- MPC2-compatible CD-ROM drive.

- Super VGA or higher-resolution video adapter capable of displaying 256 colors
  or more.

- Microsoft Mouse or compatible pointing device.

- MPC2-compatible audio board required for audio and video instruction.

Optional:

- Microsoft Visual FoxPro, Professional Edition, version 3.0 strongly
  recommended.

Product Support Policy
----------------------

If you have problems installing or setting up MVFP, call Microsoft Technical
Support at (425) 635-7033. This is the Microsoft CD-ROM Setup Support line.
There is no phone support for post-installation questions, such as usage of
MVFP.

MVFP is considered a multimedia title. It is a training product--not a
development tool. The Help file and guided tour will answer most of your
questions on functionality. You can get peer-to-peer support for usage questions
on the Internet at the following email address:

  devtrain@microsoft.com
