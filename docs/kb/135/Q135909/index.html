Q135909: Flight Simulator Display Options

Article: Q135909

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, versions 5.0, 5.0a, 5.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article contains a list of the display options available in Flight
Simulator.

To change the current display option, do the following:

1. Start Microsoft Flight Simulator.

2. Click Options.

3. Click Preferences, or use the DOWN ARROW to select the option, and press the
  ENTER key.

4. Click Display.
