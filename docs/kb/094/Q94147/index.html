Q94147: HPPCL5E: Printer Intellifont Fonts Are Not Listed

Article: Q94147
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 07-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

When you use the printer driver for the Hewlett-Packard (HP) LaserJet 4/4M
(HPPCL5E.DRV) and the TrueType Screen Fonts Installed check box located in the
driver's setup dialog box is selected, the driver will not enumerate the
internal Intellifont fonts for an application's font list.

MORE INFORMATION
================

This option is provided in the driver so that if equivalent TrueType fonts are
installed, the fonts do not show up twice in the font list. If the equivalent
TrueType fonts are not installed and the TrueType Screen Fonts check box is
selected, no Intellifont fonts will be displayed in the font list.

Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
