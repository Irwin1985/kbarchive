Q94801: Windows for Workgroups 3.1 Retail Package Options (SKUs)

Article: Q94801
Product(s): Microsoft Windows 3.x Retail Product
Version(s): 1.0,3.1
Operating System(s): 
Keyword(s): 
Last Modified: 20-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.1 
- Microsoft Workgroup Connections, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes the various packages and configurations that are
available for Windows for Workgroups version 3.1. Each stock keeping unit (SKU)
is listed below with a description and list of contents:

- Windows for Workgroups

- Workgroup add-on for 1 user of Microsoft Windows version 3.1 One-User
  Edition

  -or-

  Workgroup add-on for 5 users of Microsoft Windows version 3.1 Five-User
  Edition

- Workgroup upgrade for Microsoft Windows version 3.0 users

- Windows for Workgroups Starter Kit

- Windows for Workgroups User Kit

- Workgroup add-on for 2 users of Microsoft Windows version 3.1 Add-on Starter
  Kit

- Workgroup add-on for 1 user of Microsoft Windows version 3.1 Add-on User Kit

- Workgroup Connections

- Mail & Schedule+ Extensions for Windows for Workgroups

MORE INFORMATION
================

Windows for Workgroups
----------------------

This is the complete software package for machines that do not have Microsoft
Windows operating system version 3.0 or 3.1. It contains all the Microsoft
Windows for Workgroups software, including Windows 3.1 functionality, file
sharing, printer sharing, network dynamic data exchange (DDE), Microsoft Mail,
and Microsoft Schedule+. This package does not include any hardware.

Workgroup Add-on: One-User Edition

Workgroup Add-on: Five-User Edition
----------------------------------------------------------------------

This package is for machines that already have Microsoft Windows version 3.1
installed. It upgrades Windows 3.1 by adding those files required for the new
features of Windows for Workgroups. It includes no hardware.

This package also comes in a five pack, which includes one copy of the software
and licenses for using it on five workstations.

Workgroup Upgrade for Microsoft Windows Version 3.0 Users
---------------------------------------------------------

This package is for machines that are currently running Microsoft Windows version
3.0. It upgrades Windows 3.0 to Windows 3.1, including all of those features
specific to Windows for Workgroups. It includes no hardware.

Windows for Workgroups Starter Kit
----------------------------------

This package is designed to be a complete solution for two users without previous
installations of Windows on their machines. It includes the complete Windows for
Workgroups software for two machines as well as two network adapter cards and
the necessary connectors and cabling (25 feet).

Windows for Workgroups User Kit
-------------------------------

This package is similar to the Starter Kit, except it includes only what is
needed to add another machine to your Windows for Workgroups network. It also
comes with 25 feet of cabling.

Workgroup Add-on Starter Kit
----------------------------

This package is the same as the Windows for Workgroups Starter Kit, except that
it is designed to upgrade over Windows 3.1; Windows 3.1 must be installed on the
machine. It includes two network adapter cards and the necessary connectors and
cabling (25 feet).

Workgroup Add-on User Kit
-------------------------

This package is the same as the Windows for Workgroups User Kit, except that it
is designed to upgrade over Windows 3.1; Windows 3.1 must be installed on the
machine.

Workgroup Connections
---------------------

This package is for the MS-DOS user who wants to be a client in a Windows for
Workgroups network. Workgroup Connections allows you to connect to files and
resources shared by Windows for Workgroups machines on the same network as well
as a Mail package and a network connection pop-up program. It includes no
hardware.

Mail & Schedule+ Extensions for Windows for Workgroups
------------------------------------------------------

This package upgrades Windows for Workgroups Mail and Schedule+, enabling
workgroup-to-workgroup communications, access to external mail systems, and
remote access to Mail.
