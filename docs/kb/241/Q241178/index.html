Q241178: NBA Inside Drive 2000: Court Textures Are Not Displayed Properly

Article: Q241178
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 01-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft NBA Inside Drive 2000, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft NBA Inside Drive 2000, the court textures may appear
stretched or distorted.

CAUSE
=====

This behavior can occur if an STB Velocity 3D video adapter is installed in your
computer.

RESOLUTION
==========

To resolve this issue, use the following methods in the order in which they are
presented.

Install the Latest Video Driver for Your Video Adapter
------------------------------------------------------

Download and install the latest video driver for your video adapter from the
following STB Web site:

  http://www.stb.com

If the issue continues to occur, proceed to the next method.

Bypass the Introductory Video
-----------------------------

For information about how to bypass the introductory video, please click the
article number below to view the article in the Microsoft Knowledge Base:

  Q241012 NBA Inside Drive 2000: Introductory Video Is Not Played Properly
