Q172757: Several NT 4.0 Resource Kit Utilities Missing

Article: Q172757
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Windows NT Server 4.0 Resource Kit ISBN 1-57231-344-7 
- MSPRESS Microsoft Windows NT Workstation 4.0 Resource Kit ISBN 1-57231-343-9 
-------------------------------------------------------------------------------

SUMMARY
=======

The following Resource Kit utilities are mentioned in the RKTools.hlp file, but
are not present on the Microsoft Windows NT 4.0 Resource Kit CD-ROM:

- Global.exe

- Local.exe

- Grpcpy.exe

- Showmbrs.exe

MORE INFORMATION
================

These utilities were planned for release with the Microsoft Windows NT 4.0
Resource Kit, but were either incomplete or not stable when the Resource Kit was
released.

All of these utilities are available on the Microsoft Windows NT 4.0 Resource Kit
Supplement One and Supplement Two CD-ROMs. For more information on the Microsoft
Windows NT 4.0 Supplements, please check the Microsoft Press Web site at:

  http://mspress.microsoft.com/

Intel versions of Global.exe, Local.exe, and Showmbrs.exe are available at the
following FTP address:

  ftp://ftp.microsoft.com/bussys/winnt/winnt-public/reskit/nt40/

Additional query words: mspress ms_press press ntrk ntreskit 1-57231-344-7 1-57231-343-9
