Q172492: WD97: Templates and Wizards Not Included with SR-1 Patch

Article: Q172492
Product(s): Word 97 for Windows
Version(s): Service Release 1 (SR-1)
Operating System(s): 
Keyword(s): kbfile kbdta kbofficeupdatekbfaq
Last Modified: 30-APR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows, version Service Release 1 (SR-1) 
-------------------------------------------------------------------------------

SUMMARY
=======

The following updated template files are included only with the Microsoft Office
97 Service Release 1 (SR-1) Enterprise Update or the Microsoft Word Service
Release 1 (SR-1) Enterprise Update:

  Pleading Wizard.wiz
  Fax Wizard.wiz
  Support8.dot
  Macros8.dot

Because these files are not available with the SR-1 Patch, they are available for
download from the Microsoft Download Center.

The following file is available for download from the Microsoft Download Center:

  Wdtlupd.exe (
  http://download.microsoft.com/download/word97win/addin/1.0/W9XNT4/EN-US/Wdtlupd.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. After it is posted, the file is housed on
secure servers that prevent any unauthorized changes to the file.


MORE INFORMATION
================

For information about how to obtain other templates or wizards, please see the
following Microsoft Office Web site:

  http://office.microsoft.com/ (http://office.microsoft.com/)

To obtain the Calendar Wizard, see the following Office Web site:

  http://office.microsoft.com/downloads/9798/wdtmgmt.aspx
  (http://office.microsoft.com/downloads/9798/wdtmgmt.aspx)

For additional information about how to install wizards and templates that appear
to be missing from your installation, click the article number below to view the
article in the Microsoft Knowledge Base:

  Q139868 WD: How to Install Missing Templates and Wizards
