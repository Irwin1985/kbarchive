Q172187: WD97: Corel Square Fountain Fill Changed to Radial

Article: Q172187
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kb3rdparty kbgraphic word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you insert a graphic from CorelDRAW version 5 or 6 (*.cdr) into a Microsoft
Word 97 document, square fountain fills are converted to radial fountain fills.

CAUSE
=====

This problem occurs when you insert the graphic into Word using the Insert
Picture From File command. (On the Insert menu, point to Picture and click From
File).

WORKAROUND
==========

To work around this problem, follow these steps to copy and paste the graphic
into Word:

1. Open the CDR graphic in Corel Draw or other graphics program.

2. On the Edit menu, click Select All, and then click Copy.

3. Switch to Word.

4. Place the insertion point where you want to put the graphic.

5. On the Edit menu, click Paste.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
