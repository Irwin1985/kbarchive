Q172179: WD97: Saving to a Banyan Vines Server Hangs Word

Article: Q172179
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kb3rdparty kbnetwork
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you save a text (.txt) file using a Banyan Vines client version 5.56(5)
with Windows NT 3.51, Microsoft Word stops responding (hangs).

CAUSE
=====

This problem occurs when you open a text file in Word that was created
previously with Windows Notepad and then re-save the file.

WORKAROUND
==========

Contact Banyan Vines for the latest client version. As of June 1997, the current
release of the Banyan client for Windows NT was release 4, version 7.26.
Microsoft has not tested this version and therefore cannot guarantee its
results.

For information about how to contact Banyan Vines, query in the Knowledge Base
for the following article:

  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K


MORE INFORMATION
================

warranty, implied or otherwise, regarding this product's performance or
reliability.
