Q111041: Games Tested with WSS (Hardware & Software) Version 2.0

Article: Q111041
Product(s): Miscellaneous Windows Products
Version(s): 2.0
Operating System(s): 
Keyword(s): 
Last Modified: 28-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Sound System, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of games tested with the Windows Sound System (WSS)
version 2.0 sound card and software. (Note that the sound card is identical to
that of version 1.x). MS-DOS games were tested using the WSSXLAT.EXE driver as
set up by WSSCONFIG.EXE and Windows games were tested using the Windows Sound
System version 2.0 Windows drivers.

For more information about using WSSCONFIG.EXE and WSSXLAT.EXE, and running
MS-DOS applications using the Windows Sound System board, see the "Using Sound
Blaster-Compatible MS-DOS-Based Games and Applications" section on pages 71-82
of the "Microsoft Windows Sound System User's Guide."

MORE INFORMATION
================

The games are listed in alphabetical order in three sections: "Works Well,"
"Works with Some Dropouts," and "Does Not Work." If the name of the game is
entirely capitalized, there will be a note about it below the list in the "Game
Notes" section at the bottom of this article.

GAME NOTES
----------

ALPHABET BLOCKS = Set "AcceptCloseSamplingRates=TRUE" in the
    [SNDSYS.DRV] section of the SYSTEM.INI file for MSDVM.
LOST FILES OF SHERLOCK HOMES = Won't play digitized sounds due to
    Sound Blaster ADPCM format.
MIGHT & MAGIC : CLOUDS OF XEEN = Set at least "DMABufferSize=42" in
    the [386ENH] section of the SYSTEM.INI file for MSDVM.
PLAYROOM, THE = Set "VirtualHDIRQ=FALSE" in the [386ENH] section of
    the SYSTEM.INI file for MSDVM.
SAN DIEGO ZOO : THE ANIMALS! = There are some clicks in the background
    during speech.
SPACE ADVENTURE = Set at least"DMABufferSize=64" in the [386ENH]
    section of the SYSTEM.INI file for MSDVM.
STAR TREK 25TH ANNIVERSARY = Configure the game for "SoundBlaster" not
    "SoundBlaster PRO".
TREEHOUSE = Set "VirtualHDIRQ=FALSE" in the [386ENH] section of the
    SYSTEM.INI file for MSDVM.
ULTIMA UNDERWORLD - STYGIAN AB. = Configure the game for "SoundBlaster
    not "SoundBlaster PRO".
WHERE USA CARMEN SANDIEGO? DELX = Set "VirtualHDIRQ=FALSE" in the
    [386ENH] section of the SYSTEM.INI file for MSDVM.
