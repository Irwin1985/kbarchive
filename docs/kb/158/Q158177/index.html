Q158177: Games: How to Turn off Screen Savers

Article: Q158177
Product(s): Microsoft Home Games
Version(s): 1.0,2.0,3.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Arcade for the Macintosh, version 1.0 
- Microsoft Arcade for Windows, versions 1.0, 2.0 
- Microsoft Close Combat for Windows 1.0 
- Microsoft Close Combat for the Macintosh 1.0 
- Microsoft Deadly Tide for Windows 1.0

SYMPTOMS
========

If you have screen savers on your computer, they may display while you are
playing games if you don't move on the screen for a few minutes.

RESOLUTION
==========

If your screen saver appears while you are playing a game, simply turn off or
disable the screen saver. To disable Windows 95/98 screen savers:

1. Click Start, point to Settings, and then click Control Panel.

2. In Control Panel, double-click the Display icon.

3. Click the Screen Saver tab.

4. Under Screen Saver, click (None).

To disable AFTERDARK screen savers:

1. Activate the AFTERDARK program.

2. In the upper-right corner of the AFTERDARK control panel, click the
  activation switch to Off.
