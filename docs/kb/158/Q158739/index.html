Q158739: How to Obtain 128-bit Encryption for Windows NT Service Packs

Article: Q158739
Product(s): Internet Information Server
Version(s): WINDOWS:3.0; winnt:2.0,4.0
Operating System(s): 
Keyword(s): kbother
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
- Microsoft Internet Information Server 2.0 
- Microsoft Internet Explorer version 3.0 for Windows NT 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain 128-bit encryption for Windows NT service
packs and networking clients.

MORE INFORMATION
================

You can download 128-bit versions of service packs or networking updates from
the following Microsoft Web site:

  http://support.microsoft.com/support/ntserver/128Eula.asp

The licensing agreement for 128-bit downloads is displayed on this page. After
you read the agreement, click I Agree to accept and acknowledge that you have
read and understand the agreement.

After you accept the licensing agreement, the 128-bit download page is displayed
and you can download a service pack or an update package. To download the
128-bit version of the latest service pack, click the appropriate link at the
top of the page. To download 128-bit hotfixes and updates, click any of the
following items displayed in the drop-down list box:

- Windows NT 4.0 Post-SP3 Hotfix: Updated SChannel.dll -x86

- Windows NT 4.0 Post-SP3 Hotfix: Updated SChannel.dll -Alpha

- PPTP Performance and Security Upgrade for Windows NT 4.0 -x86

- PPTP Performance and Security Upgrade for Windows NT 4.0 -Alpha

- PPTP Performance and Security Upgrade Symbols for Windows NT 4.0 -x86

- PPTP Performance and Security Upgrade Symbols for Windows NT 4.0 -Alpha

- 128-bit version of the Windows 98 Dial-Up Networking Security Upgrade

- Windows 95 Dial-Up Networking 1.3 Utility to enable 128-bit encryption

NOTE: The last two items in the list are not rolled into Windows NT 4.0 Service
Pack 4 (SP4) because they are updates to Microsoft Windows 95 and Microsoft
Windows 98. For all other items, it is recommended that you install SP4 or later
to correct these problems.
It is also important to note that the latest service packs include additional
fixes and service-related updates, including Microsoft Internet Information
Services (IIS) 4.0 components and Microsoft Internet Explorer components. Please
refer to the Readme.txt file of the latest service pack to obtain a more
complete list of fixes that are available.
