Q158228: Mac Encarta 97: Remove Option Removes Update/Notemark Files

Article: Q158228
Product(s): Microsoft Home Multimedia Titles
Version(s): 1997 edition
Operating System(s): 
Keyword(s): 
Last Modified: 24-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 97 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

When you remove Encarta using the Remove option in Setup, everything related to
Encarta 97 is removed from your hard disk. This includes:

- Encarta 97 Encyclopedia folder

- Internet Config 1.3 folder

- Internet Explorer 2.1 Folder

- Yearbook Update files

- Notemark contents

These items would otherwise unnecessarily consume space on your hard disk.

NOTE: Fonts and other shared system components are not removed.
