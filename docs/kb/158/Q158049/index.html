Q158049: Fonts Missing or Damaged After Installing Microsoft Office 95

Article: Q158049
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kbenv kbsetup
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Microsoft Office 95, it may seem that several fonts, including
Arial, are no longer installed. When you open the Fonts folder, there are not
many fonts listed.

Or, after installing Microsoft Office 95, some of the fonts installed on your
computer are not displayed correctly in programs.

CAUSE
=====

The Microsoft Office 95 Setup program removes the registry keys for the fonts
that seem to be missing or damaged.

RESOLUTION
==========

To restore the missing or damaged fonts, restart the computer.


Additional query words: prodnt corrupt
