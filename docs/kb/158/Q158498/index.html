Q158498: PPT7: Files Installed by Stand-Alone PowerPoint

Article: Q158498
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbsetup kbdta kbdiskdir kbdir
Last Modified: 31-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a list of the files that PowerPoint Setup copies to your
hard drive if you do a Complete setup of PowerPoint for Windows 95, version 7.0.
It also lists the folders these files are copied to.
