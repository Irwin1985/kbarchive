Q86961: X400: Documentation Index for X.400 and X.25 Protocols

Article: Q86961
Product(s): Microsoft Mail For PC Networks
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 04-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to 3Com 3+Mail 
-------------------------------------------------------------------------------

SUMMARY
=======

The reference material for the Consultative Committee for International
Telephone and Telegraph (CCITT) X.400, X.200, and X.25 for the 1984 is the Red
books for the X.400 series. Here is a list of the contents:

X.400     System Model -- Service Elements
X.401     Basic Service Elements Optional User Facilities
X.408     Encoded Information Type Conversion Rules

         F.1                 Telex
         T.50                International Alphabet
         T.60, T.61, F.200   Teletex
         T.30                G3 Facsimile
         T.73                Text Interchange Format 0
         T.100, T.101        Videotext
         pending             Voice
         X.420, T.61         Simple Formatted Document
         T.73                Text Interchange Format 1

X.409     Syntax and Notations
X.410     Remote Operations and Reliable Transfer Server
X.411     Message Transfer
X.413     Message Store
X.419     MHS Application Protocols
X.420     IPM User Agent Layer

X.200     Basic Reference Model
X.208     Abstract Syntax Notation One (ASN.1)
X.209     Basic Encoding Rules for Abstract Syntax Notation One (ASN.1)
X.217     Association Control: Service Definition
X.218     Reliable Transfer: Model & Service Definition
X.219     Remote Operations: Model, Notation, & Service Definition
X.227     Association Control: Protocol Specification
X.228     Reliable Transfer: Protocol Specification
X.229     Remote Operations: Protocol Specification

X.25      Interface Between DTE/DCE Operating in a Packet Mode
X.3       Packet Assembly Disassembly (PAD) in a Public Data Network
X.28      DTE/DCE Interface for a Packet Assembly Disassembly (PAD)
X.29      Procedures for the Exchange of Control Information and User
         Data Between a Packet X.28 Packet Assembly Disassembly (PAD)
         and a DTE or Another PAD

X.121     Specifications for IP Addresses
