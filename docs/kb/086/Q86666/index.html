Q86666: PPT: Pantone(R) Color Conversion

Article: Q86666
Product(s): Microsoft PowerPoint for Windows
Version(s): MACINTOSH:2.0,3.0,4.0; WINDOWS:2.0,3.0,4.0,4.0a,4.0c,7.0
Operating System(s): 
Keyword(s): kb3rdparty kbgraphic kbdta
Last Modified: 19-APR-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
- Microsoft PowerPoint for Windows, versions 2.0, 3.0, 4.0, 4.0a, 4.0c 
- Microsoft PowerPoint for Macintosh, versions 2.0, 3.0, 4.0

For a Microsoft PowerPoint 2002 version of this article, see Q291907.

SUMMARY
=======

Many graphics programs enable you to specify color values using the Pantone
system. Microsoft products cannot provide the red, green, and blue (RGB) colors
associated with the Pantone colors or the percentages of cyan, magenta, yellow,
and black (CMYK) used by the Pantone system.

If you would like an RGB-to-Pantone color conversion table or a cyan, magenta,
yellow, and black (CMYK) percentage table, please see the following World Wide
Web site:

  http://www.pantone.com/catalog/officecolor_ss.htm

If you have any questions relating to Pantone, contact Pantone, Inc. at the
following address or phone number:

  Pantone, Inc.
  590 Commerce Boulevard
  Carlstadt, New Jersey 07072-3098
  Tel: (201) 935-5500
  Fax: (201) 896-0242
