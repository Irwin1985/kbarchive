Q153779: Close Combat: CD-ROM, Right Mouse Button Options

Article: Q153779
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Close Combat for Windows 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following options are available when you use the right mouse button to click
Closecombat (D:) from the My Computer window (where D refers to the drive letter
of your CD-ROM drive). (To open the My Computer window, double-click My Computer
on the Windows desktop.)

Open
----

Opens a window with the contents of the Close Combat compact disc.

Explore
-------

Opens the Exploring - Closecombat (D:) window with the Windows Explorer
interface.

Find
----

Opens the "Find: All Files" window, which allows you to search your hard drive
for specific files.

AutoPlay
--------

Opens the Microsoft Close Combat Autorun window. Re-install, Play, Game Demos,
Explore CD, and Quit are all options that are available from this window.

Install Close Combat
--------------------

Opens Close Combat Setup.

Install DirectX
---------------

Opens the DirectX Setup window. "ReInstall DirectX" and "Restore Windows95
drivers" are options that are available from this window.

Launch MS Info
--------------

Opens Microsoft System Information, version 2.5.

Install Internet Explorer
-------------------------

Opens Microsoft Internet Explorer version 2.0 Installation Kit, which give you
the option to install Microsoft Internet Explorer, version 2.0.

Install Indeo 4.1
-----------------

Opens Setup for Indeo video interactive, which installs Indeo video drivers on
your system.

Install Windows95 Service Pack 1
--------------------------------

Opens the Windows 95 Service Pack 1 Update window, which installs the Windows 95
Service Pack 1 on your system.

Sharing
-------

Opens the Sharing tab of the Close Combat Properties window, which allows you to
specify the sharing properties for the Close Combat compact disc.

Eject
-----

Ejects the Close Combat compact disc.

Create Shortcut
---------------

Gives you the opportunity to create a shortcut to the Close Combat compact disc
on your Windows 95 desktop.

Properties
----------

Opens the Close Combat Properties window.
