Q63696: Hardware Options Offered When Installing Windows

Article: Q63696
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 09-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the hardware options offered during the installation of
Microsoft Windows version 3.0 or 3.0a. If the Setup program offered any other
choices for hardware, it is not Microsoft Windows but an OEM (original equipment
manufacturer) version of Windows; additional support must be obtained from the
manufacturer.

MORE INFORMATION
================

The following choices are listed during the Setup process.

Computer
--------

  All 80386- and 80486-based AST machines
  All 80386-based Zenith machines
  All Hewlett-Packard machines
  Everex Step 386/25
  MS-DOS or PC-DOS system
  NCR PC386SX
  NCR PC 925
  NEC PowerMate SX Plus
  NEC ProSpeed 386
  Toshiba 1600
  Toshiba 5200
  Other (Requires disk provided by a hardware manufacturer)

Display
-------

  8514/a
  CGA
  Compaq Portable Plasma
  EGA
  Hercules Monochrome
  Olivetti/AT&T Monochrome or PVC Display
  QuadVGA, ATI VIP VGA, 82C441 VGAs
  VGA
  VGA with Monochrome display
  Video Seven VGA with 512K
  Other (Requires disk provided by a hardware manufacturer)

Mouse
-----

  HP Mouse (HP-HIL)
  Logitech serial mouse
  Logitech bus or PS/2-style
  Microsoft or IBM PS/2
  Mouse Systems (or VisiOn) connected to COM1
  Mouse Systems (or VisiOn) connected to COM2
  No mouse or other pointing device
  Olivetti/AT&T Keyboard Mouse
  Other (Requires disk provided by a hardware manufacturer)

Keyboard
--------

  All AT type keyboards (84 - 86 keys)
  AT&T '301' keyboard
  AT&T '302' keyboard
  Enhanced 101- or 102-key US and Non-US keyboards
  Hewlett-Packard Vectra keyboard (DIN)
  Olivetti 101/102 A keyboard
  Olivetti 83-key keyboard
  Olivetti 86-key keyboard
  Olivetti M24 102-key keyboard
  PC-XT 83-key keyboard
  PC/XT - Type keyboard (84 keys)
  Other (Requires disk provided by a hardware manufacturer)

---------------

  Belgian
  French Canadian
  Danish

  Finnish
  German
  Icelandic
  Italian
  Latin American

  Portuguese
  Swedish
  Swiss French
  Swiss German

  British
  US-Dvorak
  US-International
  US
  Other (Requires disk provided by a hardware manufacturer)

Language
--------

  Danish
  Dutch
  English (American)
  English (International)
  Finnish
  French
  French Canadian
  German
  Icelandic
  Italian
  Norwegian
  Portuguese
  Spanish
  Swedish
  Other (Requires disk provided by a driver manufacturer)

Network
-------

  3Com 3+Open LAN Manager (XNS only)
  3Com 3+Share
  Banyan VINES 4.0
  LAN Manager 1.x (or 100 percent compatible)
  LAN Manager 2.0 Basic (or 100 percent compatible)
  LAN Manager 2.0 Enhanced (or 100 percent compatible)
  No Network Installed
  Novell NetWare 2.10 or later or Novell NetWare 386
  IBM PC LAN Program
  Microsoft Network or 100Other (Requires disk provided by a network
  manufacturer)
