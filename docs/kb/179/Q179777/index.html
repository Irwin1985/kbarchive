Q179777: Encarta 98: Lookup Reference Feature Not Available in Word

Article: Q179777
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbfile kbinterop kbmm kbimukbfaq
Last Modified: 21-DEC-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 98 Encyclopedia for Windows 
- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install Encarta 98 Encyclopedia, you cannot access the Lookup Reference
feature from inside Microsoft Word.

CAUSE
=====

The behavior occurs because the required files and files and settings are not
added by the Encarta 98 Encyclopedia installation program.


RESOLUTION
==========

To resolve this problem, download and install the Enc98lur.exe patch file.


MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  Enc98lur.exe
  (http://download.microsoft.com/download/encarta98encdlx/Update/1/WIN98/EN-US/Enc98lur.exe)

Release Date: Jan-22-1998

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
