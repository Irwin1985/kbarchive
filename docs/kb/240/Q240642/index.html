Q240642: World Atlas: Quits Unexpectedly When Screen Resolution Changed

Article: Q240642
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kb3rdparty kbenv kbimu
Last Modified: 25-JUN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Interactive World Atlas 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you change the screen resolution in the Display tool in Control Panel while
Microsoft Encarta World Atlas 2000 is running, the program may quit
unexpectedly.

CAUSE
=====

This behavior can occur if a Matrox Millennium G200 AGP video adapter is
installed in your computer.

RESOLUTION
==========

To work around this issue, change the screen resolution before you start Encarta
World Atlas 2000:

1. Quit Encarta World Atlas if it is running.

2. Click Start, point to Settings, and then click Control Panel.

3. Double-click Display.

4. Click the Settings tab.

5. Drag the "Display area" or "Screen area" slider to the screen resolution you
  want.

6. Click OK, and then click OK again.

7. Close Control Panel.

8. Start Encarta World Atlas 2000.
