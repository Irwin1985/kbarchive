Q186213: Outwars: List of Enhancement (Cheat) Codes

Article: Q186213
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbimu msgame outwars
Last Modified: 26-MAY-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outwars, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Outwars enhancement codes allow you to have unlimited fuel, unlimited
ammunition, and to be immortal. This article lists the enhancement codes
available in Outwars.

MORE INFORMATION
================

To use the enhancement (cheat) codes, press the ENTER key, and then type the
code word in the Chat box.

Outwars Cheat List - Retail Version
-----------------------------------

  Enhancement Code     Function
  -----------------------------
  Bigears              Placeholder sounds On/Off

  Buzz                 Glider wings (as in Buzz Lightyear)

  Dirtyharry           Unlimited ammunition

  Framerate            Framerate counter

  Gohome               Reset to the start position

  Keymaster            Unlimited jet pack fuel

  Jump<level>, where   Jumps to a specific level. For example, if you
  <level> is a level   type "jumpoasis," you jump to the Oasis level.
  name.                Known levels that work with this code include:

                       Oasis, Anubis, Ragnarok, Juggernaut, and Dead.
                       There may be secret levels you can jump to, such
                       as "jumpsecret."

  Macleod              Immortality (Highlander)

  Phantom              Spy on creatures using the F11 and F12 keys

  Singletrac           SingleTrac message

  Sneakers             Sneaker mode

  Snipsnip             Change your player character's gender

  Thrasher             Thrasher mode. Shows all your enemies on radar

  Timewarp             Mission timer On/Off

  Weaponcam            Weapon optics display. Follows your fired rounds
                       (like the cameras in smart bombs).

  Warpme               Mission warp? Possibly with an argument, such as
                       "warpme dead."

Outwars Cheat List - Trial Version
----------------------------------

  Enhancement Code   Function
  ---------------------------

  Macleod            Invincibility

  Singletrac         Displays a message from SingleTrac
