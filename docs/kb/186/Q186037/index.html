Q186037: OFF97: Alki Language Proofing Tools Available

Article: Q186037
Product(s): Word 97 for Windows
Version(s): 97
Operating System(s): 
Keyword(s): kb3rdparty kbdta kbproof word97
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Access 97 
- Microsoft Excel 97 for Windows 
- Microsoft Outlook 97 
- Microsoft PowerPoint 97 for Windows 
- Microsoft Word 97 for Windows 
- Microsoft Office 97 Professional Edition for Windows 
- Microsoft Office 97 Standard Edition for Windows 
- Microsoft Office 97 Small Business Edition for Windows, version 2.0 
- Microsoft Office 97 Small Business Edition for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

Microsoft proofing tools packages are available from Alki Software for languages
from six different language groups: Western European, Central European, Baltic,
Cyrillic, Greek, and Turkish.


MORE INFORMATION
================

Each language group uses its own set of characters. By default, you can proof
text in any of the languages within your system's language group. For example,
if you are using the English (U.S.) version of Microsoft Windows 95, Microsoft
Windows 98, Microsoft Windows NT, or Microsoft Windows 2000, you should have no
problem proofing text in other Western European languages. However, if you want
to proof text in a different language group, you may need to adjust your
operating system to turn on (enable) your applications to display and proof text
correctly in the new language.

For additional information about how to turn on support for multiple languages in
Windows, please see the following articles in the Microsoft Knowledge Base:

  Q141306 How to Enable Support for Multiple Languages in Windows


  Q177561 How to Add and Enable Additional Languages in Windows NT


NOTE: The spelling check feature does not work in Office 97 applications, other
than Word 97 for Windows, in languages outside your Windows 95 operating
system's language group. You must upgrade to Windows NT 4.0 or later. Please
contact Alki Software for more information.

-------------------

- Microsoft Windows 95 or Microsoft Windows NT Workstation 3.51 or later

- One or more of the Microsoft Office applications listed at the beginning of
  this article

your version of Windows.
