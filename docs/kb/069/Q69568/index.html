Q69568: Disabling Shadow RAM on Leading Technology 386-16SX

Article: Q69568
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 14-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Leading Technology's 386-16SX does not work in enhanced mode with Microsoft
Windows version 3.0 if the ROM BIOS or VIDEO BIOS is being shadowed. To disable
this shadowing, do the following:

1. Boot the PC.

2. At the DOS prompt, press CTRL+ALT+S.

3. Press PGUP.

4. Highlight the Shadow BIOS option using the vertical arrow keys.

5. Press either the RIGHT or LEFT ARROW key to toggle between enable and
  disable. Choose Disable.

6. Highlight the Shadow RAM option using the UP and DOWN ARROW keys.

7. Press either the RIGHT or LEFT ARROW key to toggle between Enable and
  Disable. Choose Disable.

8. Press ESC.

================
