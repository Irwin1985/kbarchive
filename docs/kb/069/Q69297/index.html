Q69297: Windows Err Msgs: Paintbrush Message List

Article: Q69297
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 08-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a list of the possible error messages you may receive from the
Paintbrush application (PBRUSH.EXE) included with Microsoft Windows version
3.00.

NOTE: The "(null)" and "0%" symbols in a message indicate a variable string used
by the application.

MORE INFORMATION
================

+--------------------------------------------------------------------------------------------------------------------------------------------------------+

   more than 8 characters followed by a period and a 3-letter
   extension. | 
+--------------------------------------------------------------------------------------------------------------------------------------------------------+
| 3  | (null) was created using a previous version of Windows; Paintbrush
  cannot load this file.                                                      | 
+--------------------------------------------------------------------------------------------------------------------------------------------------------+

+--------------------------------------------------------------------------------------------------------------------------------------------------------+

+--------------------------------------------------------------------------------------------------------------------------------------------------------+

   available memory.   |
