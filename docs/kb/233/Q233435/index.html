Q233435: Business Planner: Black Bar Is Printed Beneath Heading of Page

Article: Q233435
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbprint kbimu
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Business Planner 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a Web Links or Reference Directory page from Microsoft Business
Planner, a black bar may be printed beneath the heading on the printed page.

CAUSE
=====

This behavior can occur if you attempt to print an entire Web Links or Reference
Directory page.

RESOLUTION
==========

To work around this issue, select the text you want to print, and then print the
selected text. To do this:

1. Use your pointing device to select the text you want to print from the Web
  Links page or Reference Directory page.

2. On the File menu, click Print.

3. Under Print Range, click Selection.

4. Click Print.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Business Planner.

MORE INFORMATION
================

The heading at the top of a Web Links or Reference Directory page contains
graphics that may be printed differently from the way they are displayed on the
screen.
