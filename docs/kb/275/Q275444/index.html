Q275444: INFO: Online Resources for Visual FoxPro Support

Article: Q275444
Product(s): Microsoft FoxPro
Version(s): 3.0,3.0b,5.0,5.0a,6.0,7.0
Operating System(s): 
Keyword(s): kbvfp300 kbvfp300b kbvfp500 kbvfp500a kbvfp600 kbGrpDSFox kbDSupport
Last Modified: 24-OCT-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a, 6.0, 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

There are many online resources where you can get support for Visual FoxPro. The
following list is not meant to be exhaustive.

MORE INFORMATION
================

The main resource for Visual FoxPro on the World Wide Web is the Microsoft
Visual FoxPro Home Page:

  http://msdn.microsoft.com/vfoxpro/ (http://msdn.microsoft.com/vfoxpro/)

Microsoft also runs a news server for peer-to-peer support at the following
address:

  news://msnews.microsoft.com/microsoft.public.fox
  (news://msnews.microsoft.com/microsoft.public.fox)

For other peer-to-peer support resources, the following list provides some good
starting points:

- The Universal Thread (http://www.universalthread.com/), which has Basic
  (free) and Premier memberships

- DevX, which has Web
  (http://www.devx.com/gethelp/gethelp.asp?Area=Visual+FoxPro) and News
  (news://news.devx.com/vfp.general) resources available.

- The FoxForum (http://www.foxforum.com/) and FoxForum Wiki
  (http://fox.wikis.com)

- The West Wind Message Board (http://www.west-wind.com/wwthreads/)

- The ProFox mailing list (http://www.leafe.com/)

- The FoxShare mailing list (http://www.foxshare.com/h/fs/)

- The Virtual FoxPro User Group (http://www.vfug.org/)

- The MS Developer Applications Forum on CompuServe
  (http://forums.compuserve.com/vlforums/default.asp?SRV=MSDevApps)

All of these resources are frequented by Microsoft Visual FoxPro MVPs
(http://support.microsoft.com/support/mvp/program.asp), who are community
members recognized by Microsoft for their outstanding contributions.

For additional information on FoxPro newsgroups, click the article number below
to view the article in the Microsoft Knowledge Base:

  Q192825 INFO: Searching FoxPro Newsgroups
