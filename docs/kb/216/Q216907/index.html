Q216907: HOW TO: Obtain a Test or Test Client Authentication Certificate

Article: Q216907
Product(s): Internet Information Server
Version(s): 4.0,5.0
Operating System(s): 
Keyword(s): kbAudDeveloper kbAudITPro kbHOWTOmaster
Last Modified: 30-JUL-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Internet Information Server version 4.0 
- Microsoft Internet Information Services version 5.0 
-------------------------------------------------------------------------------


IN THIS TASK
------------

- SUMMARY

   - To Obtain Test SSL Certificates

SUMMARY
=======

This step-by-step article describes how to obtain Secure Sockets Layer (SSL)
certificates for testing purposes.

To Obtain Test SSL Certificates
-------------------------------

Certification authorities do, occasionally, offer short term certificates for
testing purposes. Microsoft recommends that you contact the certification
authority of your choice to ascertain whether the certification authority offers
this service.

For additional information about how to set up your own certificate server to
create test certificates, click the article number below to view the article in
the Microsoft Knowledge Base:

  Q299525 HOWTO: Set Up SSL Using IIS 5.0 and Certificate Server 2.0
