Q216223: Windows NT 4.0 Service Pack 4 Information

Article: Q216223
Product(s): Microsoft Windows NT
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): 
Last Modified: 19-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Internet Explorer 4.01 Service Pack 1a (SP1a) is now included in the
full download version of Windows NT 4.0 Service Pack 4 (SP4) available on the
Microsoft Web site and CD-ROM. Please note that this is not a new version of
Windows NT 4.0 SP4.

MORE INFORMATION
================

This version of Windows NT 4.0 SP4 also includes a fix to the Y2ksetup.exe
program that resolves an issue affecting Microsoft Site Server Express 2.0 and
Microsoft Message Queue Server (MSMQ) 1.0 users. This fix to Y2ksetup.exe, which
affects only a small segment of customers, is also available as a workaround for
customers who previously downloaded the full version of Windows NT 4.0 SP4. If
you already have Windows NT 4.0 SP4 successfully installed, you can continue
using your existing software.

For additional information, please see the following articles in the Microsoft
Knowledge Base:

  Q197533 BUG: Applying the Full Download of WinNT 4.0 SP4 Removes MSMQ

  Q197534 BUG: Recovering Messages and Redefining Queues after Applying SP4

For information about how to obtain the Windows NT 4.0 SP4 update, please see the
following article in the Microsoft Knowledge Base:

  Q152734 How to Obtain the Latest Windows NT 4.0 Service Pack


Additional query words: update re-release rerelease
