Q243525: XCLN: Inbox Assistant Rule on Custom Field Does Not Start

Article: Q243525
Product(s): Microsoft Exchange
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): 
Last Modified: 22-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook 97 
- Microsoft Outlook 98 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

An Inbox Assistant rule that is based on a custom field of one-off form does not
work.

WORKAROUND
==========

To work around this problem, publish the form to the Organization Forms Library.

MORE INFORMATION
================

This behavior is by design. Inbox Assistant rules do not work for custom
properties on one-off forms.


Additional query words: trigger, OL97, OL98
