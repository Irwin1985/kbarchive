Q277792: Reader: "Open With" Dialog Box Appears When Opening eBook File

Article: Q277792
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.5
Operating System(s): 
Keyword(s): kbfile kbimu
Last Modified: 31-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Reader, version 1.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to download or open an eBook (.lit) file in Microsoft Reader,
you may experience one of the following symptoms:

- An Open With dialog box appears on the screen, prompting you to choose the
  program that you want to use to open the file.

- A Save As dialog box appears on the screen, prompting you to choose a
  location in which to save the file.

CAUSE
=====

This behavior can occur if Microsoft Internet Explorer 5.5 is installed on your
computer.

RESOLUTION
==========

To resolve this issue, update the eBook (.lit) file association on your
computer.

The following file is available for download from the Microsoft Download Center:

  Rdrmime.exe
  (http://download.microsoft.com/download/msreader/Patch/1.0/W9X2KMe/EN-US/Rdrmime.exe)

Release Date: Oct-24-2000

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
