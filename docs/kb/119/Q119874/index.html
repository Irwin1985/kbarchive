Q119874: Lost Colors with Cirrus Logic Based VLB Video Adapter

Article: Q119874
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Windows 95 startup logo is distorted, and the interface appears in black and
white during startup and while running Windows 95.

CAUSE
=====

This problem occurs with a VLB video adapter using the Cirrus Logic 5426, 5428,
or 5434 video chip on a Pentium-based computer that has both VLB sockets and PCI
sockets on the motherboard.

RESOLUTION
==========

To correct this problem, replace the VLB video adapter with a PCI video adapter
using the Cirrus Logic 5426, 5428, or 5434 video chip or a video adapter using a
different video chip, such as the S3 or one manufactured by Tseng Labs.

MORE INFORMATION
================

The same VLB adapters function correctly in computers with only VLB sockets.
