Q264388: Allegiance Error Message: Unable to Create a Game

Article: Q264388
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Allegiance, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Training Mission 7 in Microsoft Allegiance, you may receive the
following error message:

  Unable to create a game. The home server may not be properly installed.
  Install the Home Server from your Allegiance CD.

CAUSE
=====

This behavior can occur if either of the following conditions is true:

- The Allegiance server is not installed.

- The Allegiance server is a different version than the Allegiance client.

RESOLUTION
==========

To resolve this issue, use the following methods in the order in which they are
presented.

Install the Allegiance Server
-----------------------------

To install the Allegiance server, follow these steps:

1. Press and hold down the SHIFT key as you insert the Allegiance CD-ROM into
  the CD-ROM drive.

2. Click Start, and then click Run.

3. In the Open box, type the following line, and then click OK

  <drive>:\server\setup.exe

  where <drive> is the drive letter of your CD-ROM drive.

4. Follow the instructions on the screen to install the Allegiance server.

Update Allegiance
-----------------

To update Allegiance, follow these steps:

1. Start Allegiance.

2. Click Free Games.

3. Click Games List.

4. If the latest update of Allegiance is not installed on your computer, the
  update is downloaded and installed automatically, and then Allegiance
  restarts.

Update the Allegiance Server
----------------------------

To update the Allegiance server, follow these steps:

1. Start Allegiance server.

2. In the Zone Free Games list, click Public Game.

3. Click Create Game.

If the latest update of the Allegiance server is not installed on your computer,
the update is downloaded and installed automatically, and then Allegiance
restarts.
