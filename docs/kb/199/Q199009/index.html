Q199009: EVG99: Video Playback Is Choppy or Quits Before Clip Is Finished

Article: Q199009
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu
Last Modified: 31-JAN-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
- Microsoft Encarta Reference Suite 99 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play a video clip in Microsoft Encarta Virtual Globe 99 on a Microsoft
Windows 95-based computer, the video playback may be broken up or choppy, or may
quit before the video clip is finished playing.

CAUSE
=====

This behavior can occur if an ATI Rage Pro video card is installed in your
computer.

RESOLUTION
==========

To resolve this issue, download and install the latest video driver for the ATI
Rage Pro video card from the following ATI Web site:

  http://www.atitech.com

If the issue continues to occur, download and install the latest Intel Indeo
video codec from the following Intel Web site:

  http://www.intel.com/ial/indeo/video/driver.htm

MORE INFORMATION
================

For information about problems playing video clips in Microsoft Encarta
Encyclopedia, please see the following article in the Microsoft Knowledge Base:

  Q197041 Encarta Encyclopedia 99: Video Clips Are Not Played Smoothly
