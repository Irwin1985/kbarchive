Q199032: EVG99: "Error Reading CD-ROM" When You Zoom In or Out on the Map

Article: Q199032
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 30-JAN-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Reference Suite 99 
- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you zoom in or out on the map in Microsoft Encarta Virtual Globe 99, you
may receive the following error message:

  Error Reading CD-ROM in Drive <cd-rom>:

where <cd-rom> is the drive letter of your CD-ROM drive.

CAUSE
=====

This behavior can occur if the Encarta Virtual Globe 99 CD-ROM is not inserted
in the CD-ROM drive.

RESOLUTION
==========

To prevent this issue from occurring, insert the Encarta Virtual Globe 99 CD-ROM
in the CD-ROM drive when you run Encarta Virtual Globe 99.

MORE INFORMATION
================

The Encarta Virtual Globe 99 CD-ROM must be inserted in the CD-ROM drive at all
times for the program to run properly.
