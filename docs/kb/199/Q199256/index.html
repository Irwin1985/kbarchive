Q199256: Networking Essentials Education Edition Comments And Corrections

Article: Q199256
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 22-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Academic Learning Series Networking Essentials Second Edition ISBN 1-57231-909-7 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book "Networking Essentials 2nd Edition:
Education Edition," ISBN 1-57231-909-7.

Contents:

- CD-ROM: Student.doc Shows Revisions Marks In WordPad

- Page 230: Table Incorrect

- Page 777: Description of Gateway Incorrect

MORE INFORMATION
================

CD-ROM: Student.doc Shows Revisions Marks In WordPad
----------------------------------------------------

The Student.doc file, contained on the Supplemental Course Material CD-ROM, shows
revision marks when opened in WordPad. These revisions marks do not appear if
the file is opened in Microsoft Word 97.

If you would like to receive a copy of this file that does not show revision
marks when opened in WordPad, please email tkinput@microsoft.com.


Page 230: Table Incorrect
-------------------------

On page 230 of the "Acaademic Learning Series Networking Essentials, Second
Edition," the table is incorrect.

In the "Demand Priority" column and the "Type of access method" row, change:
"Contention"

TO:
"Non-contention"


Page 777: Description of Gateway Incorrect
------------------------------------------

On page 777, the description of Gateway is incorrect.

Change:
"Gateways functions at the Network layer of the OSI model."

To:
"Gateways typically function at the application layer of the OSI model. However,
the level of functionality varies widely between different types of gateways."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
