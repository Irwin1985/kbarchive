Q288120: Combat Flight Simulator: No Bombs or Rockets in Multiplayer Mode

Article: Q288120
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbenv kbimu
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start a multiplayer game of Microsoft Combat Flight Simulator, the
bombs and the rockets options are unavailable when you configure your aircraft.

CAUSE
=====

This behavior occurs because bombs and rockets are intended for air-to-ground
targets and ground structures and those objects are not present in multiplayer
mode.
