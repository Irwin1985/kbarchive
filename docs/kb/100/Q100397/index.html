Q100397: SoundBits: Sounds from Two Schemes Are Merged

Article: Q100397
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0; MACINTOSH:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Sound Bits (Hanna-Barbara) version 1.0 
- Microsoft Sound Bits (Movie Clips) version 1.0 
- Microsoft SoundBits, Musical Instruments, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you select the Windows default sound scheme after selecting a Sound Bits
sound scheme, the two schemes will both run simultaneously. In other words,
choosing the default sound scheme does not cancel your selection of the Sound
Bits sound scheme. You will hear sounds from both schemes. This behavior is by
design.

To reset your sound scheme to the Windows default scheme only, follow these
steps:

1. From the Windows Control Panel, choose Sound.

2. From the Sound Schemes list box, select None.

3. Choose OK.

4. Run Sound again.

5. From the Sound Schemes menu, select Windows Default.

6. Choose OK.

This resets your sound scheme to the Windows default sounds only.
