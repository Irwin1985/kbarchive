Q262249: StarLancer: Cannot Play Game in Direct 3-D Render Mode

Article: Q262249
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbimu kbHardware
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft StarLancer, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you configure Microsoft StarLancer to use 3D Render Mode for your Direct 3D
video adapter, you may be unable to play the game.

CAUSE
=====

This behavior can occur if a video adapter based on the Intel I740 chip set is
installed in your computer.

RESOLUTION
==========

To work around this issue, configure StarLancer to use the software renderer:

1. On the StarLancer startup screen, click Game Options.

2. Click Video.

3. Click the RIGHT ARROW or the LEFT ARROW next to 3D Render Mode to change the
  setting to Software Renderer.

4. Click Main Menu.
