Q118674: Gtwy: Gateway Connectivity Information

Article: Q118674
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:2.1,3.0,3.2
Operating System(s): 
Keyword(s): 
Last Modified: 30-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for PC Networks, versions 2.1, 3.0, 3.2 
-------------------------------------------------------------------------------

SUMMARY
=======

The following list identifies the Microsoft Mail gateways that can be used to
integrate Mail in a heterogeneous mail environment. Additional gateways that use
the Microsoft Mail File Format API (FFAPI) may also be available.

NOTE: All gateways in the following list are Microsoft gateways unless otherwise
noted.

 Message system              Gateway
  -----------------------------------------------------------------------

  3+ Mail                      3Com
  ARCOM 400 Swiss PTT          X.400
  AT&T Easylink                X.400
  AT&T Mail                    X.400 or AT&T Gateway
  Atlas 400                    X.400 (NF mark from Afnor)
  Banyan Mail                  MHS or X.400
  Beyond Mail                  MHS
  cc:Mail                      X.400, MHS, SMTP
  CompuServe                   SMTP or MHS
  The Coordinator              MHS
  Data General CEO             X.400
  DaVinci                      MHS
  DEC All-In-1                 X.400, PROFS, SNADS, or SMTP
  DEC VMS Mail                 X.400 or SNADS
  Dutcha PTT                   X.400
  Envoy 100/Gemdes             X.400
  Fax                          Fax
  Fischer International(EMC2)  X.400 or SNADS
  GEIS                         QuickComm gateway from GE
  Gold 400 UK                  X.400
  Higgins                      MHS
  HP Desk                      Walker, Richer, Quinn HP Desk gateway
  IBM AS/400 Office            SNADS, or through PROFS Distribution
                               Manager, if you have a VM host
  IBM System 36                PROFS/OV
  IBM CMS NOTES                PROFS
  IBM DISOSS                   SNADS
  IBM OfficeVision MVS         SNADS, or through PROFS Distribution

  MCI Paper Mail               MCI MAIL

  NCR Corporation              X.400, or in some cases, the AT&T Gateway
