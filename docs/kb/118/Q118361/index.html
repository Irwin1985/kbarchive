Q118361: Mac Bookshelf: Quickshelf Does Not Recognize Words in Works

Article: Q118361
Product(s): Microsoft Home Multimedia Titles
Version(s): 1994 edition,1996-1997; MACINTOSH:3.0
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 11-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf version 1996-1997 for the Macintosh 
- Microsoft Bookshelf for the Macintosh 1994 edition 
- Microsoft Works for the Macintosh, version 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Bookshelf QuickShelf in conjunction with Works 3.0, QuickShelf does
not respond.

RESOLUTION
==========

Upgrading to Works version 4.0 resolves this problem.

Microsoft has confirmed this to be a problem in Works version 3.0. This problem
has been corrected in Works version 4.0.

Microsoft Works 2.0 is also compatible with QuickShelf.

MORE INFORMATION
================

QuickShelf is a utility included with Bookshelf that allows you to access the
Bookshelf Dictionary and Encyclopedia from within other applications.

Steps to Reproduce Problem
--------------------------

1. Type a word in a Works 3.0 Word Processor document and highlight it.

2. From the QuickShelf icon in the menu bar, choose either Dictionary Contents
  or Encyclopedia Contents. Nothing will happen.
