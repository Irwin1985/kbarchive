Q145609: Pregnancy & Child Care: Setup Does Not Start

Article: Q145609
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 01-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Pregnancy and Child Care, Preview on MSN 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Pregnancy and Child Care requires that you have MSN, the Microsoft
Network online service version 1.2 installed. If this version is not installed,
then Pregnancy and Child Care Setup does not start.

MORE INFORMATION
================

Use one of the following two methods for downloading version 1.2 of MSN:

1. MSN 1.2 can be downloaded from MSN Today at no additional charge.

2. On the Edit menu in MSN, click Go To and, and then click Other Location.

3. Type "MSN12" (without the quotation marks) and click OK.

4. Double-click the MSN 1.2 Updates folder to download MSN version 1.2.

Additional query words: msn multi media multimedia multi-media mmtitles kbmm installation install initiate startup start up doesn't run
