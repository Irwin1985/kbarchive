Q145896: Cinemania: Unable to Add Custom Items to List Maker

Article: Q145896
Product(s): Microsoft Home Multimedia Titles
Version(s): 1996 edition,1997 edition
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 15-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Cinemania for Windows versions 1996 edition, 1997 edition 
- Microsoft Cinemania for the Macintosh versions 1996 edition, 1997 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are using the List Maker Find button, and type the name of a movie and
click Add To List, you may receive the following Cinemania message:

  No item is selected.

CAUSE
=====

This occurs when the name you type in the Find field is misspelled or not
included in the Cinemania list of movies.

RESOLUTION
==========

Check the spelling of the movie or make sure the movie is contained in the list.

MORE INFORMATION
================

If the entry is spelled correctly and you receive this error, the movie does not
exist in the list of movies in Cinemania. You cannot add an item to the list
included in Cinemania.

NOTE: Adding a custom item was available in previous versions of Cinemania.
