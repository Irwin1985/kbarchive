Q145674: How Long Does it Take to Set Up Pregnancy and Child Care

Article: Q145674
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 01-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Pregnancy and Child Care, Preview on MSN 
-------------------------------------------------------------------------------

SUMMARY
=======

The first time you install and start Microsoft Pregnancy and Child Care takes
between 4 and 15 minutes. The exact time depends on the speed of your modem and
if shared system files are required.

If you use a 28.8K bits-per-second (bps) modem the process takes approximately
four minutes. A 9600 bps modem may take as long as 15 minutes.

MORE INFORMATION
================

Downloading and installing the user interface software is a one-time procedure.
The installation consumes approximately 4 MB of disk space
