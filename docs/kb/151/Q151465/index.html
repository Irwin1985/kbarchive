Q151465: MSB Ocean: How to Determine Which Version You Have

Article: Q151465
Product(s): Microsoft Home Kids Products
Version(s): 1.0,1.01
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Scholastic's Magic School Bus series: Explores the Ocean for Windows, versions 1.0, 1.01 
-------------------------------------------------------------------------------

SUMMARY
=======

There is an updated version of Magic School Bus Explores the Ocean for Windows
(MSB Ocean): version 1.01. To determine which version of MSB Ocean you have, use
the following steps:

Windows 95
----------

1. Insert MSB Ocean into the CD-ROM drive. Click Cancel when it asks you if you
  want to run MSB Ocean.

2. Click the Start button, point to Programs, click Windows Explorer.

3. Locate the CD-ROM drive in the left pane of Windows Explorer. The volume
  label is different, depending on the version of MSB Ocean that you have.

  MSB Ocean version 1.0 has volume label MSEO100__01.

  MSB Ocean version 1.01 has volume label MSEO101__01.

Windows 3.1
-----------

1. Insert MSB Ocean into the CD-ROM drive.

2. In Program Manager, go to the Main group and run the MS-DOS prompt.

3. In MS-DOS, type "dir D:" (without the quotation marks), and where D: is the
  letter of the CD-ROM drive) and press "ENTER" (without the quotation marks).

4. At the top of the directory listing, you should see "Volume in drive D is."

  MSB Ocean version 1.0 has volume label MSEO100__01.

  MSB Ocean version 1.01 has volume label MSEO101__01.

5. To return to Windows, type "exit" (without the quotation marks) at the C:\
  prompt.

You can also determine which version you have by comparing file sizes. To do so,
display the file properties of the following file:

  Mskids\Msbocean\System\Msbsea.exe

In MSB Ocean version 1.0, Msbsea.exe is 195,984 bytes.

In MSB Ocean version 1.01, Msbsea.exe is 196,864 bytes.
