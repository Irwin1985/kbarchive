Q151975: INFO: List of SourceSafe Error Messages Documented on the Web

Article: Q151975
Product(s): Microsoft SourceSafe
Version(s): WINDOWS:4.0,5.0,6.0
Operating System(s): 
Keyword(s): kberrmsg kbSSafe400 kbSSafe500 kbSSafe600
Last Modified: 04-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual SourceSafe for Windows, versions 5.0, 6.0 
- Microsoft Visual SourceSafe, 16-bit, for Windows, version 4.0 
- Microsoft Visual SourceSafe, 32-bit, for Windows 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Use the "List of SourceSafe and Visual SourceSafe Error Messages Documented in
the Microsoft Knowledge Base" on the following Web page to find the SourceSafe
and Visual SourceSafe error messages that are documented in the Microsoft
Knowledge Base:

  http://support.microsoft.com/support/ssafe/content/q151975/default.asp

You can use the Find in your Browser to locate error messages quickly on this
page.
