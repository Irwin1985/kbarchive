Q296256: SMS: Online Resources for the Seagate Crystal Reports Tool

Article: Q296256
Product(s): Microsoft Systems Management Server
Version(s): 2.0
Operating System(s): 
Keyword(s): kbinterop kbDatabase kbsms200 kbQuery kbReport kbsmsUtil
Last Modified: 04-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the online resources that are available from Seagate Software
for the Seagate Crystal Reports tool, which is bundled with Systems Management
Server (SMS) version 2.0.
