Q272658: MSN Companion Member Name and Password Confirmation

Article: Q272658
Product(s): The Microsoft Network
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, versions 1.0, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you first connect to MSN, The Microsoft Network, using the MSN Companion,
the MSN Web Companion asks for a member name and password.

MORE INFORMATION
================

You sign up for MSN Internet Access when you purchase the MSN Web Companion
unit. At that time you will choose a member name.

A confirmation letter with this member name and an initial password is given to
you. You must enter that name and password on the member name and password
confirmation screen.

If the system is unable to confirm the name and password, a Call Member Support
option appears.

Call call Customer Service at (888) 765-2411

REFERENCES
==========

  Q272409 MSN Companion Password and Member ID Requirements
