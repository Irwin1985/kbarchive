Q191086: XFOR: NDR Not Sent When "Report to DL Owner" Option Checked

Article: Q191086
Product(s): Microsoft Exchange
Version(s): winnt:5.0,5.5
Operating System(s): 
Keyword(s): 
Last Modified: 23-MAR-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, versions 5.0, 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When a mail client user sends messages to a distribution list (DL) with invalid
foreign Simple Mail Transfer Protocol (SMTP) recipients, the non-delivery
reports (NDRs) are routed only to the sender, even if the "Report to DL Owner"
option on the Advanced tab is selected.


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
versions 5.0 and 5.5. Microsoft has targeted this problem to be fixed in the
next version of Exchange Server. Contact Microsoft Technical Support for more
information.
