Q191755: MCM: Program Is Minimized When You Click Options or Multiplayer

Article: Q191755
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgamekbfaq
Last Modified: 21-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Motocross Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Motocross Madness, and then click either Multiplayer
Game or Options on the Motocross Madness startup screen, Motocross Madness may
be minimized.

NOTE: Motocross Madness is designed to be minimized when you click Help on the
Motocross Madness startup screen.

CAUSE
=====

This behavior can occur if an STB Symmetric GLyderMAX-2 video adapter is
installed in your computer.


RESOLUTION
==========

To resolve this issue, download and install the latest version of the video
driver for the STB Symmetric GLyderMAX-2 video adapter from the following
Symmetric Web site:

  http://www.symmetric.com

MORE INFORMATION
================

The STB Symmetric GLyderMAX-2 display adapter is based on the 3Dlabs Permedia-2
chipset.
