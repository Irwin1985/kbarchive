Q191478: INFO: Palm-size PC 1.0 Does Not Support VBCE Applications

Article: Q191478
Product(s): Microsoft Visual Basic for Windows
Version(s): 
Operating System(s): 
Keyword(s): kbAddIn kbToolkit kbVBp500 kbVBp600 kbOSWinCE100 kbGrpDSVB
Last Modified: 16-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows CE Toolkit for Visual Basic 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The implementation of the Windows CE 2.0 operating system on the Palm-size PC
1.0 devices does not include sufficient COM support for Windows CE Toolkit for
Visual Basic 6.0 (VBCE) applications. However, the Palm-size PC devices running
the Windows CE operating system 2.11 do provide support for VBCE applications.

MORE INFORMATION
================

The Windows CE Toolkit for Visual Basic 6.0 (VBCE6) can create applications for
the following classes of devices:

- Handheld PC (H/PC - running Windows CE 2.0)

- Handheld PC Pro (H/PC Pro - running Windows CE 2.1x)

- Palm-size PC (PPC - running Windows CE 2.1x)

Please note that the implementation of the Windows CE operating system 2.0 on the
Palm-size PC device does not support VBCE6 applications. The Palm-size PC
Platform SDK version 1.2 has been made available to assist you with writing
applications targeting Palm-size PC's (running Windows CE 2.11 or later). To
download this SDK, please refer to the following Web site:

  http://www.microsoft.com/msdownload/cetools/ppcsdk_color.asp

For current lists of Windows CE devices, please see the following Web sites:
Handheld PC devices:

  http://www.microsoft.com/windowsce/hpc/partner/suppliers.asp

Handheld PC Pro devices:

  http://www.microsoft.com/windowsce/hpcpro/partner/suppliers.asp

Palm-size PC devices:

  http://www.microsoft.com/windowsce/products/ppc/manufacturers.asp

The Windows CE Toolkit for Visual C++ (VCCE) supports the computers listed in
these Web site as well as the Palm-size PC (PPC) class of devices running the
Windows CE operating system version 2.0, such as the Philips Nino 300 and Everex
Freestyle.
