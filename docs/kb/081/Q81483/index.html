Q81483: Jetpage Cartridge Printer Memory Requirements for Windows

Article: Q81483
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

According to Computer Peripherals, Inc. (CPI), to use the Jetpage printer
cartridge with Microsoft Windows, printers must have a minimum of 1.5 megabytes
of memory. Printers may require 2.5 megabytes to process some complicated
graphic files.

The Jetpage cartridge is shipped with a driver for the Windows operating system
version 3.00. A driver is not yet available for Windows version 3.1.

MORE INFORMATION
================

The Jetpage PostScript language emulation cartridge emulates the Hewlett-Packard
(HP) LaserJet Series II, IID, IIP, and III within the Windows system. According
to CPI, the manufacturers of Jetpage, the cartridge is compatible with the
following drivers, devices, and programs:

- PCL Language

- Adobe Type 1 and Type 3 Downloadable fonts

- Jetware Font Cartridge

- Apple Laserwriter II NT fonts

For additional information, Contact Computer Peripherals.
