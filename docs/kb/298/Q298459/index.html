Q298459: Frequently Asked Questions for Windows NT 3.51

Article: Q298459
Product(s): Microsoft Windows NT
Version(s): 3.51
Operating System(s): 
Keyword(s): kbWinNT351
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 3.51 
- Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

This list of frequently asked questions (FAQ) provides information about some
Windows NT 3.51 and describes some common issues and workarounds or resolutions
that may occur. The following subjects are included in the article:

- Dial-Up Networking and Remote Access Service (RAS) Questions
- Hardware and Multimedia Questions
- Setup and Upgrading Questions
- Printing Questions
- Miscellaneous Questions

MORE INFORMATION
================
