Q298714: MSN Explorer (MSN Version 6.1) Sign-in And Sign-up Error Message

Article: Q298714
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 23-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SUMMARY
=======

For any of the specific sign-in and sign-up error messages below, use the
corresponding solution. In this KB article, the sign-in and sign-up error
messages are categorized by type, and are listed alphabetically.

MORE INFORMATION
================

Error Message: E-mail names can only contain letters (a-z), numbers (0-9), the underscore (_), and no spaces.
-------------------------------------------------------------------------------------------------------------

This error occurs when a user attempts to place an invalid character in their
e-mail address. The e-mail address can only contain the characters referenced in
the error message. This is often caused by the user typing @msn.com in the user
name box. Inform the user that they do not need to type @msn.com in this box.
Enter a valid e-mail name, and then click Continue.

Error Message: E-mail names cannot start with a dash '-'.
---------------------------------------------------------

or

Error Message: E-mail names cannot start with a dash (-).
---------------------------------------------------------

This error occurs when the user attempts to create an e-mail name that begins
with the '-' character. Inform the user that new e-mail names must begin with a
letter. The e-mail can contain a '-', but it cannot begin with one. Enter a
valid e-mail name and click Continue.

Error Message: E-mail names cannot start with a number.
-------------------------------------------------------

This error occurs when the user attempts to create an e-mail name that begins
with a letter. E-mail names must begin with a letter. The e-mail name can
contain numbers, but it cannot begin with one. Enter a valid e-mail name, and
then click Continue.

Error Message: E-mail names need to be at least three characters long.
----------------------------------------------------------------------

This error occurs when the user attempts to enter an e-mail name that is less
than three characters long. Enter a valid e-mail name, and then click Continue.

Error Message: Please enter your e-mail name.
---------------------------------------------

This error occurs when the user has left the E-mail name box blank. Specify the
desired e-mail name, and then click Continue.

Error Message: Please enter your new e-mail name here.
------------------------------------------------------

This error occurs when the user has not specified an e-mail name. Type an e-mail
name, and then click Continue.

Error Message: Please choose an option or enter your new e-mail name here.
--------------------------------------------------------------------------

This error occurs when the user has neither chosen one of the suggested e-mail
names nor has specified an e-mail name of their own. Select one of the suggested
names or type an e-mail name of your choice in the provided box.

Error Message: Please choose an option or enter your new e-mail name here.
--------------------------------------------------------------------------

This error occurs when the user has neither selected a suggested e-mail name nor
typed in an e-mail name. Either select an e-mail name, or enter a custom e-mail
name in the box provided, and then click Continue.

Error Message: Please select an option before continuing.
---------------------------------------------------------

This error occurs when a user does not choose an option on the e-mail name page.
Select one of the suggested e-mail names or type an e-mail name in the space
provided, and then click Continue.

Error Message: There is already an account for this user on this computer.  Please try again with another e-mail name.
----------------------------------------------------------------------------------------------------------------------

This error occurs when the user enters an e-mail name that has already been set
up on the computer. In this case, use the pre-existing user tile.

Error Message: This e-mail name contains a forbidden word, please select a new e-mail name.
-------------------------------------------------------------------------------------------

This error occurs when the user tries to set up an e-mail name that contains
either an obscene word or a word relating to the service (such as MSN). Enter a
valid e-mail name, and then click Continue.

Error Message: This email name has already been taken, please choose another one and continue.
----------------------------------------------------------------------------------------------

This error occurs when the user is trying to create a new e-mail name, but the
e-mail name they wish to set up his being used by another user. E-mail names are
available on a first-come, first-serve basis, and that there is no way to assume
ownership of an e-mail name if it is already being used by another user. Choose
a different e-mail name (consider adding a number to the end, or some other
slight variation), and then click Continue.

Error Message: Your e-mail name, user@msn.com is too long. It needs to be less than 64 characters total.  Please try again.
---------------------------------------------------------------------------------------------------------------------------

This error occurs when the user attempts to create an e-mail name that is over
the 64 character limit. Enter a shorter e-mail name, and then click Continue.
