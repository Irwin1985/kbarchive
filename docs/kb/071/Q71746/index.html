Q71746: Getting the Correct Size MS-DOS Disks

Article: Q71746
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:3.30a,4.01a,5.0
Operating System(s): 
Keyword(s): 
Last Modified: 22-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 3.30a, 4.01a, 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you get the Microsoft MS-DOS Packaged Product in the wrong disk size, you can
obtain the correct media through Microsoft Customer Service by sending in the
disk set and requesting the other format.

For example, if you purchased the 5.25-inch Microsoft MS-DOS Packaged Product
version 4.01a for a machine that has only 3.5-inch disk drives, send in the
5.25-inch disks and request the 3.5-inch disk set.

This information applies only to Microsoft MS-DOS Packaged Product versions
3.30a, 4.01a, and 5.0.

If you do not have the Microsoft MS-DOS Packaged Product (for example, you have
COMPAQ MS-DOS), contact your dealer or OEM (original equipment manufacturer).

MORE INFORMATION
================

If you purchased the product within 90 days of contacting Microsoft, you will
not be charged. Otherwise, the cost is $15.00, plus shipping, handling, and
applicable sales tax.

Pricing information is subject to change and is provided only for the customer's
convenience. For more information, contact Microsoft Sales Information Center
(MSIC) at (800) 426-9400.
