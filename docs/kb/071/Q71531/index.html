Q71531: STAR Color Printers Print in Color from Paintbrush

Article: Q71531
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SUMMARY
=======

STAR Micronics offers printer drivers for its new line of 24-pin printers that
can print in color from the Paintbrush program packaged with Windows 3.0.

This information applies to Microsoft Windows versions 3.0, and 3.0a. This
information does not apply to later versions of Windows.

MORE INFORMATION
================

The color drivers available from STAR are for its line of new 24-pin color
printers. The following are the printers:

  Model XB2410 with color option
  Model XB2420 Rainbow
  Model NX2420 Rainbow

You can obtain the drivers from the STAR support department.


The only other known printers to work with Paintbrush in color are color
PostScript printers, Hewlett-Packard's (HP) PaintJet printer, and Kodak's
Diconix Color 4 printer (in PaintJet emulation mode).

For more information, query on:

  "paintbrush" (without the quotation marks) and "color" (without the quotation
  marks) and "printers" (without the quotation marks)

The products discussed here are manufactured by vendors independent of Microsoft;
we make no warranty, implied or otherwise, regarding these products' performance
or reliability.

NOTE: Microsoft has not tested the new STAR drivers.
