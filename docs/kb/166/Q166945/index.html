Q166945: DocErr: Microsoft BackOffice Resource Kit

Article: Q166945
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 08-JAN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft BackOffice Resource Kit, Part 1 ISBN 1-55615-932-3 
- MSPRESS Microsoft BackOffice Resource Kit, Part 2 ISBN 1-57231-534-2 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information on known errors
relating to the "Microsoft BackOffice Resource Kit" series of Microsoft Press
books. The following topics are covered:

- ActiveLU missing from BackOffice Resource Kit CD-ROM

- Part 1, Page 212: DMI45.DOC Not On CD

MORE INFORMATION
================

ActiveLU Missing From BackOffice Resource Kit CD-ROM
----------------------------------------------------

The help file for SNA Server contains a topic for a utility called ActiveLU.
However, this utility was not included on the "BackOffice Resource Kit, Part 2"
CD-ROM.

ActiveLU is currently planned for a future release of the BackOffice Resource
Kit. Please disregard any references to ActiveLU in this release of the
BackOffice Resource Kit.

Microsoft Press will post any updates to this information in the BackOffice
Resource Kit Knowledge Base article. This and other Microsoft Press Knowledge
Base articles are available via the World Wide Web by connecting to:

  http://mspress.microsoft.com/support/support.htm

Part 1, Page 212: DMI45.DOC Not On CD
-------------------------------------

Page 212, Paragraph 1 states that the DMI45.doc (Desktop Management Interface
Specification 4.5) can be found on the BackOffice Resource Kit CD. However, this
file was not included with the CD-ROM.

DMI45.doc can be found on the Microsoft BackOffice SDK CD.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
