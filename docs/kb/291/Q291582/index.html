Q291582: Games Err Msg: Version Testing Date has Expired

Article: Q291582
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kbimu
Last Modified: 28-MAR-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MechWarrior 4: Vengeance 
- Microsoft Crimson Skies 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start one of the games listed at the beginning of this
article, you may receive the following error message:

  Version testing date has expired.

CAUSE
=====

This behavior can occur if either of the following conditions is true:

- You have outdated video codecs installed on your computer.

  -or-

- You are using third-party digital video disc (DVD) software that controls
  MPEG playback, such as WinDVD.

RESOLUTION
==========

To resolve this issue, use the method appropriate for your situation.

If the video codecs are outdated
--------------------------------

Update the Indeo video playback codecs to the latest version. For the latest
updated codecs, visit the following Web site:

  http://www.ligos.com/index.phtml?pi=103
  (http://www.ligos.com/index.phtml?pi=103)

If you have third-party DVD software controlling MPEG playback
--------------------------------------------------------------

Remove the third-party DVD software controlling MPEG playback. For more
information about how to do this, contact the manufacturer of your DVD software
or refer to the program documentation.
