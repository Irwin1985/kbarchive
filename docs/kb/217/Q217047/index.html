Q217047: PRB: CD Key Invalid for Visual Basic 6.0 Deluxe Learning Edition

Article: Q217047
Product(s): Microsoft Visual Basic for Windows
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): kbsetup kbVBp600 kbLEdition kbGrpDSVB
Last Modified: 06-JUN-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Setup program of the Microsoft Visual Basic 6.0 Deluxe Learning Edition from
Microsoft Press requires a 10-digit CD key. However, the CD key sticker on the
back of your jewel case may contain an 11-digit CD key.

NOTE: This applies to the Microsoft Visual Basic 6.0 Deluxe Learning Edition
only.

CAUSE
=====

Initially, the wrong type of CD key stickers were used with this product.

RESOLUTION
==========

It is possible to use the 11-digit CD key that was included with the Visual
Basic 6.0 Deluxe Learning Edition CD-ROM. To do this, you must remove the last
digit from the first group of four numbers. In other words, if the CD key
reads:

  1234-5678907

Remove the "4" and enter the CD key as follows:

  123-5678907

This allows the product to install correctly.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft Visual Basic 6.0
Deluxe Learning Edition.

REFERENCES
==========

For additional information about the Visual Basic 6.0 Deluxe Learning Edition,
please see the following article(s) in the Microsoft Knowledge Base:

  Q192469 Can't Find Visual Basic 6 Step by Step Files



Additional query words: ISBN 1-57231-873-2 bad
