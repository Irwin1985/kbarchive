Q322034: How To Import E-mail From Netscape Mail To Outlook Express 5.0.4

Article: Q322034
Product(s): The Microsoft Network
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-APR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Internet Access for Macintosh, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to import e-mail from Netscape Mail to Outlook
Express 5.0.4

MORE INFORMATION
================

To import e-mail from Netscape Mail to Outlook Express 5.0.4

1. Open Outlook Express.

2. On the File menu, click Import.

3. In the Import window, click Import from an application, and then click the
  right arrow to continue.

4. Select Netscape Communicator 4.0 or higher as the application from which you
  want to import, and click the right arrow to continue.

5. Select Mail Messages as the data you wish to import, and click the right
  arrow to continue.

6. Click OK.
  Your imported e-mail messages will be in the Communicator Import folder.
