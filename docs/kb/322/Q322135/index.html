Q322135: Pocket Streets 2002: Err Msg: Error Extracting Support Files

Article: Q322135
Product(s): Microsoft Automap
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 09-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Streets & Trips 2002, version 1.0 
- Microsoft MapPoint 2002 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Setup.exe to start the installation of Microsoft Pocket Streets
2002, you may receive an error message similar to the following:

  Error extracting support files: Server execution failed.

CAUSE
=====

This problem may occur if InstallShield is not completely installed.

RESOLUTION
==========

To resolve this problem, download and install the most recent version of
Microsoft Reader, and then install Pocket Streets. To do this, follow these
steps in the order in which they are presented.

Download and Install Microsoft Reader
-------------------------------------

1. Connect to the Internet, and then visit the following Microsoft Web site:

  Microsoft Reader for PC
  http://www.microsoft.com/reader/downloads/pc.asp

2. Click to download the most recent version of Reader.

3. When the File Download dialog box appears, click Open.

4. Follow the on-screen instructions to complete the installation process.

Install Pocket Streets
----------------------

NOTE: Before you install Pocket Streets, make sure that your Pocket PC is
connected to your desktop computer and that Microsoft Windows CE Services or
ActiveSync is installed on the desktop computer.

1. Insert the Streets & Trips 2002 or MapPoint 2002 Setup CD into the CD-ROM
  drive or DVD-ROM drive. Hold down the SHIFT key to prevent Setup from
  starting automatically.

2. Double-click My Computer.

3. Right-click the CD-ROM drive, and then click Explore.

4. Double-click Pstreets.

5. Double-click Setup.exe.

6. Follow the on-screen instructions to install Pocket Streets.

MORE INFORMATION
================

For more information about InstallShield, visit the following InstallShield Web
site:

  InstallShield
  http://www.installshield.com

For more information about ActiveSync, visit the following Microsoft Web site:

  ActiveSync
  http://www.microsoft.com/mobile/downloads/activesync.asp
