Q91006: Packing List: Windows 3.1 SDK Part 5 (5.25-Inch Disks)

Article: Q91006
Product(s): Microsoft Windows Software Development Kit
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): kb16bitonly
Last Modified: 06-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Software Development Kit (SDK) 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is the packing list for the Microsoft Windows Software Development
Kit (SDK) version 3.1:

  Disk 6
    ------

    MPPRINT.C          MULTIPAD Sample Printing Code
    MAKEFILE           OUTPUT Sample Program Makefile
    OUTPUT.C           OUTPUT Sample Program Source Code
    OUTPUT.DEF         OUTPUT Sample Program Definitions File
    OUTPUT.H           OUTPUT Sample Program Include File
    OUTPUT.RC          OUTPUT Sample Program Resource Compiler File
    MAKEFILE           OWNCOMBO Make File--Owner Drawn Combo Boxes
    OWNCOMBO.C         OWNCOMBO Sample Program Source File
    OWNCOMBO.DEF       OWNCOMBO Sample Program Definitions File
    OWNCOMBO.DLL       OWNCOMBO Sample Program Dynamic Link Library
    OWNCOMBO.H         OWNCOMBO Sample Program Include File
    OWNCOMBO.ICO       OWNCOMBO Sample Program Icon
    OWNCOMBO.RC        OWNCOMBO Sample Program Resource Compiler File
    README.TXT         Readme File for OWNCOMBO Sample Program
    MAKEFILE           PRNTFILE Sample Program Makefile
    PRNTFILE.C         PRNTFILE Sample Program Source File
    PRNTFILE.DEF       PRNTFILE Sample Program Definitions File
    PRNTFILE.H         PRNTFILE Sample Program Include File
    PRNTFILE.RC        PRNTFILE Sample Program Resource Compiler File
    DEMO.C             Sample Program Used to Access SELECT.DLL
    DEMO.DEF           Definitions File Used to Access SELECT.DLL
    DEMO.H             Include File Used to Access SELECT.DLL
    DEMO.RC            Resource Compiler File Used to Access SELECT.DLL
    MAKEFILE           SELECT Sample Program Make File--Windows DLL
    SELECT.C           SELECT Sample Program Source Code
    SELECT.DEF         SELECT Sample Program Definitions File
    SELECT.H           SELECT Sample Program Include File
    DIB.C              DIBVIEW Sample Program DIB Manipulation Source Code
    DLGOPEN.C          MCITEST Sample Program Dialog Box Source Code
    MAKEFILE           SHOWDIB Sample Program Makefile
    PRINT.C            DIBVIEW Sample Program DIB Printing Source Code
    SHOWDIB.C          SHOWDIB Sample Program Source Code
    SHOWDIB.DEF        SHOWDIB Sample Program Definitions File
    SHOWDIB.DLL        SHOWDIB Sample Program Dynamic Link Library
    SHOWDIB.H          SHOWDIB Sample Program Include File
    SHOWDIB.ICO        SHOWDIB Sample Program Icon File
    SHOWDIB.RC         SHOWDIB Sample Program Resource Compile File
    API31.HLP          Windows Multimedia API Quick Help File (part 1)
    MCISTRQH.HLP       Windows Multimedia API Quick Help File (part 2)
    PENAPIQH.HLP       Pen Windows API Quick Help File

    Disk 7
    ------

    MSGS31   HLP       Windows API Overview Quick Help File (part 1)
    OVER31   HLP       Windows API Overview Quick Help File (part 2)
    WINHELP  HLP       Windows Help File on Using Windows Help

    Disk 8
    ------

    WIN31MQH HLP       Windows API Overview Quick Help File (part 3)
    WIN31WH  1HP       Windows SDK Overview Windows Help File (part 2)

    Disk 9
    ------

    SAMPLES  HLP       Windows Sample Program Windows Help File
    WIN31WH  0HP       Windows SDK Overview Windows Help File (part 1)
    TIPS     HLP       Windows Programming Tips Windows Help File

    Disk 10
    -------

    MCISTRWH HLP       Windows Media Control Interface Windows Help File
    PENAPIWH HLP       Pen Windows API Windows Help File
    WIN31MWH HLP       Windows Multimedia Reference Windows Help File
    WIN31WH  2HP       Windows SDK Overview Windows Help File (part 3)

Additional query words: 3.00 no32bit packing.lst 3.10
