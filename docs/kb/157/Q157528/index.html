Q157528: WD97: "Outline Effect" Text Garbled or in All Caps When Printed

Article: Q157528
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbprint word97 kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print a document, text formatted with the Outline font effect may be
garbled or may appear in all capital letters.

NOTE: To format text with the Outline font effect, click Font on the Format menu,
click the Font tab, and click to select the Outline check box under Effects.

CAUSE
=====

This problem may occur when you print to one of the following printers:

- Hewlett-Packard DeskJet 660C or Hewlett-Packard 1600 printers, using the
  legacy printer drivers.

- Hewlett-Packard LaserJet 4 or 5 series PostScript printers

- Hewlett-Packard Color LaserJet 5/5m PostScript printers

- Hewlett-Packard Color LaserJet 5/5m non-PostScript (HPPCL) printers

  NOTE: Text formatted as Outline is printed as solid text.

- Tektronix Phaser 550c

- Lexmark Optra PS


WORKAROUND
==========

To work around this problem, use a non-PostScript printer driver.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
