Q157883: How Can I Get VSS and Visual InterDev Integration Working?

SUMMARY
=======

Q. How can I get VSS and Visual InterDev integration working?

A. Visual InterDev uses FrontPage Server Extensions to integrate with Visual
SourceSafe through OLE Automation. It is important to note that the Visual
InterDev client does not need a Visual SourceSafe client for integration to
work. However, the web server does need to have the Visual SourceSafe and
FrontPage Server Extensions installed for the integration to work. For
additional information, please see the following article in the Microsoft
Knowledge Base:

  Q171116 HOWTO: Enable VSS Integration with FrontPage and Visual InterDev

  Q166259 INFO: Visual InterDev/Visual SourceSafe Integration Discussion

  Q165831 PRB: Anonymous User in NT Admin Group Breaks Source Control
