Q116156: Farallon Phone-Net Is Incompatible with 32-Bit File Access

Article: Q116156
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.11
Operating System(s): 
Keyword(s): 
Last Modified: 10-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

According to Farallon Technical Support, Farallon Phone-Net PC is incompatible
with Windows for Workgroups 3.11 32-bit file access.

MORE INFORMATION
================

The Farallon Phone-Net is a network that uses 10baseT and standard Ethernet
cards in IBM-compatible machines to allow connections to Macintosh computers.
According to Farallon Technical Support, the Farallon redirector is incompatible
with IFSHLP.SYS and 32-bit file access in Windows for Workgroups 3.11.

WORKAROUND
==========

At this time, to work around this problem, you must use the Farallon Desk
Accessories (DA.EXE) at the MS-DOS level outside Windows for Workgroups. This
allow you to download files from the Macintosh servers before entering Windows
for Workgroups.


Additional query words: 3rdparty 3.11 vfat Phone Net
