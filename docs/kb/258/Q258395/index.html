Q258395: Baseball 2001: Textures for the Players and Crowd Appear White

Article: Q258395
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 22-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Baseball 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Baseball 2001, you may receive an error message stating
that textures in the game may appear white.

When you play the game, the textures for the players, the crowd, and the stadium
appear white.

CAUSE
=====

This behavior can occur if the following conditions are true:

- Baseball 2001 is configured to run at a display resolution of 1024 x 768
  pixels.

- An ASUS 3Dexplorer 3000 video adapter or an ATI Rage IIc (8 MB) video adapter
  is installed in your computer.

- The video driver for your video adapter is outdated.

RESOLUTION
==========

To resolve this issue, use the appropriate method for your video adapter.

ASUS 3Dexplorer 3000
--------------------

Download and install the latest version of the video driver for the ASUS
3Dexplorer 3000 video adapter from the following ASUS Web site:

  http://www.asus.com

ATI Rage IIc
------------

Download and install the latest version of the video driver for the ATI Rage IIc
video adapter from the following ATI Web site:

  http://support.atitech.ca/drivers/index.html
