Q313576: XGEN: Exchange Server 5.5 Post-SP4 Outlook Web Access Fixes

Article: Q313576
Product(s): Microsoft Exchange
Version(s): 5.5,5.5 SP1,5.5 SP2,5.5 SP3,5.5 SP4
Operating System(s): 
Keyword(s): kbExchange550preSP5fix
Last Modified: 20-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, versions 5.5, 5.5 SP1, 5.5 SP2, 5.5 SP3, 5.5 SP4 
-------------------------------------------------------------------------------


SUMMARY
=======

This article lists Microsoft Knowledge Base articles for Exchange Server 5.5
Microsoft Outlook Web Access (OWA) bugs that were fixed after Exchange Server
5.5 Service Pack 4 (SP4) was released.

Contact Microsoft Product Support Services to obtain this rollup. For a complete
list of Microsoft Product Support Services phone numbers and information about
support costs, please go to the following address on the World Wide Web:

  http://support.microsoft.com/default.aspx?scid=fh;EN-US;CNTACTMS

NOTE: In special cases, charges that are normally incurred for support calls may
be canceled if a Microsoft Support Professional determines that a specific
update will resolve your problem. Normal support costs will apply to additional
support questions and issues that do not qualify for the specific update in
question.

Exchange Server 5.5 fixes for a particular component are cumulative and contain
all of the earlier fixes for that component. Fixes with a particular version
number contain all of the fixes that have an earlier version number.

Fixes Released on December 3, 2001
----------------------------------

The following files are modified:

- Cdohtml.dll, Htmlsnif.dll, and Safehtml.dll are incremented to version
  5.5.2655.73
- Fumsg.asp is incremented to version 5.5.2655.39
- Global.asa is incremented to version 5.5.2655.41
- Read.asp is incremented to version 5.5.2655.12

The following fixes are included:

  Q296689 XWEB: Exchange Server 5.5 Outlook Web Access Generates Errors on OWA
  Servers with No Daylight Savings Time

  Q301361 XGEN: Incorrect Attachment Processing in Exchange Server 5.5 Outlook
  Web Access Can Run Script

  Q307195 XWEB: OWA Function Allows Unauthenticated User to Enumerate Global
  Address List
