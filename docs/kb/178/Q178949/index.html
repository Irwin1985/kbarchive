Q178949: MS Graph 97: Cannot Scroll One Window Left or Right in Datasheet

Article: Q178949
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbui
Last Modified: 16-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
- Microsoft Graph 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you use the ALT+PAGE DOWN or ALT+PAGE UP keys in a Microsoft Graph
datasheet, the shortcut keys may not scroll one window right or one window left.

WORKAROUND
==========

Use the horizontal scroll bar. To scroll one window left or right, click to the
left or right of the scroll box on the horizontal scroll bar.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

MORE INFORMATION
================

In a Microsoft Graph datasheet, you can scroll left or right to the next window
of cells. When you use the ALT+PAGE DOWN shortcut keys, the Graph datasheet
should scroll right one window of cells. When you use the ALT+PAGE UP shortcut
keys, the Graph datasheet should scroll left one window of cells.

For more information about moving and scrolling through a Graph datasheet, click
the Office Assistant, type "move in datasheet" (without the quotation marks),
click Search, and then click to view "Move and scroll through a datasheet."

NOTE: If the Assistant is hidden, click the Office Assistant button on the
Standard toolbar. If Microsoft Help is not installed on your computer, please
see the following article in the Microsoft Knowledge Base:

  Q120802 Office: How to Add/Remove a Single Office Program or Component

REFERENCES
==========

For additional information, please see the following article(s) in the Microsoft
Knowledge Base:

  Q178969 MS Graph 97: Keyboard Shortcuts for Microsoft Graph 97
