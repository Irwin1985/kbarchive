Q178715: Zone: How to Obtain Help with the Zone Web Site

Article: Q178715
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kbref kbimu msgamekbfaq
Last Modified: 30-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zone.com 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain help with the Zone Web Site.

MORE INFORMATION
================

Microsoft Product Support Web Site:

Visit the FAQs & Highlights for Games Web page for technical information and
additional support resources at the following Microsoft Web site:

  http://support.microsoft.com/highlights/gms.asp

Ask a Microsoft Support Professional for help

  http://go.microsoft.com/fwlink/?LinkId=376

Tutorial:

For information concerning how to play games on the Zone.com web site, please
visit the Zone.com tutorial:

  http://zone.msn.com/tutorial/tutorial_1of15.asp

Game Rules and Objectives:

To obtain information about game rules and objectives for games you can play on
Zone.com, follow these steps:

1. Connect to Zone.com at the following Web site:

  http://zone.msn.com

2. Click Games.

3. Click the game you want to play.

4. Click Game Help.

5. When the game Help window is displayed on the screen, click the link to the
  appropriate Help topic to obtain the information you need.

Zone Newsgroup:

You can obtain assistance from other Zone users on the Zone newsgroup:

  news://msnews.microsoft.com/microsoft.public.games.zone

Billing of Zone Premium Games:

If you have a question related to the billing of Zone premium games, visit the
following Web site:

  http://zone.msn.com/billing/

If you would like to view the answers to commonly reported billing problems or to
report a billing problem, please visit the following Web site:

  http://zone.msn.com/zqdb/help/zscreen.asp

Banned Account:

If you have been banned, you are denied access to the Zone. To view the details
of your ban, please visit the following Zone Web site:

  https://zone.msn.com/services/MyZone.asp

If you are trying to play a Premium game and have been banned, please contact
Billing Support at the following Zone Web site if you need to cancel your
subscription renewal until you are allowed access again:

  http://zone.msn.com/billing

Ratings:

If you have a question about your Rating in a Zone game, please send a message to
the following e-mail address:

  zratings@microsoft.com

How to Handle Abusive or Disruptive Players:

If you need to report a problem with another member in the Zone, please contact a
Member Plus member for immediate help. Member Plus members are people who are
available to help with community issues in the Zone. This includes problems
between members, such as a player who uses offensive language or harasses other
players. In a game room, you can identify a Member Plus member by the PLUS SIGN
(+) to the left of their Zone name.

To report a community issue to a Member Plus member:

- If you are already in a game lobby, follow these steps:

  a. From the Zone menu, click Contact Sysop...

  b. Under "Member Plus Help Room", click "I ACCEPT the above terms. Let me go
     chat!"

  c. On the Room menu, click Ask Question.

- If you are not already in a game lobby, follow these steps:

  a. On the Zone home page (http://zone.msn.com), click Chats & Events.

  b. Click Chat Rooms.

  c. Under "Member Plus Help Room", click "I ACCEPT the above terms. Let me go
     chat!"

  d. On the Room menu, click Ask Question.

For more information about how to report a problem with another member in the
Zone, please see the following Zone Web page:

  http://zone.msn.com/zzzz/help/zonehelpdisruptiveplayers.asp
