-------------------------------------------------------------------------------

SUMMARY
=======

This article contains the text from the FIXLIST.TXT file that accompanies the
Microsoft Systems Management Server version 1.1 U.S. Service Pack 1. For
information on obtaining this update, query on the following word in the
Microsoft Knowledge Base (without the spaces):

  " S E R V P A C K " (without the quotation marks)
