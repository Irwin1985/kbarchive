Q131655: Mac Wine Guide, Ancient Lands: Play Button Sounds Don’t Work

Article: Q131655
Product(s): Microsoft Home Multimedia Titles
Version(s): MACINTOSH:1.0
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Ancient Lands for the Macintosh, version 1.0 
- Microsoft Wine Guide for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click a Play button in one of the programs listed at the top of this
article, you don't hear sounds.

CAUSE
=====

Your computer probably has file sharing enabled.

RESOLUTION
==========

Microsoft has confirmed this to be a problem in Microsoft Wine Guide for the
Macintosh, version 1.0. This problem was corrected in product name version
1.0a.

To correct this in Microsoft Ancient Lands for the Macintosh or Microsoft Wine
Guide version 1.0, use the steps below to stop file sharing and restore Play
button functionality.

1. Open the Control Panels.

2. Select Sharing Setup.

3. In the File Sharing section, click Stop.

4. Close the Sharing Setup dialog box.

5. Play your sound.

MORE INFORMATION
================

Sounds produced using Play buttons are in compressed format. Other sounds you
hear throughout the applications are generated using a different technology.
Although you can hear the narration, with file sharing on you cannot hear any
Play button sounds.
