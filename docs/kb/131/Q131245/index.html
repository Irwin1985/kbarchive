Q131245: Setting Up The Microsoft Network on a Second Computer

Article: Q131245
Product(s): The Microsoft Network
Version(s): WINDOWS:1.3,2.0,2.5
Operating System(s): 
Keyword(s): kbsetup kbmsn
Last Modified: 12-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.3, 2.0, 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to set up MSN, The Microsoft Network, on a second
computer if you are already an MSN member.

MORE INFORMATION
================

MSN 1.x
-------

Run the Sign-up program by clicking the The Microsoft Network icon on the
desktop.

If this opens the Sign-In screen instead of starting the Sign-up program, follow
these steps:

1. Click Cancel.

2. Click Start, and then click Run.

3. In the Open box, type "signup.exe" (without quotation marks), and then click
  OK.

MSN 2.0 and 2.5
---------------

1. Press and hold down the SHIFT key while you insert the MSN CD-ROM.

2. Click Start, and then click Run.

3. In the Open box, type "msnstart.exe" (without quotation marks), and then
  click OK.

4. Click Go To MSN Setup to install all the MSN components.

5. After all components are installed, you are prompted whether you need to set
  up a new account. Click "No, I already have an MSN account."

Windows 98 and MSN 2.5
----------------------

1. Insert the Windows 98 CD-ROM into the CD-ROM drive, and then close the
  Windows 98 Welcome screen if it opens on the desktop.

2. Click Start, point to Find, and then click Files Or Folders.

3. In the Named box, type "msnstart.exe" (without quotation marks).

4. In the Look In box, click the CD-ROM drive containing the Windows 98 CD-ROM,
  and then click Find Now.

5. In the list of found files, double-click the Msnstart.exe file.

6. Follow the instructions on the screen to reinstall MSN.
