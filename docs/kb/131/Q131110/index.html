Q131110: Writer/Artist: How to Change User Names

Article: Q131110
Product(s): Microsoft Home Kids Products

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
- Microsoft Creative Writer for the Macintosh, versions 1.0, 1.1, 1.1a 
- Microsoft Fine Artist for Windows, versions 1.0, 1.1, 1.1a 
- Microsoft Fine Artist for the Macintosh, versions 1.0, 1.1, 1.1a 
-------------------------------------------------------------------------------

SUMMARY
=======

It is possible to change your user name in Creative Writer or Fine Artist.

To change your user name, follow the instructions appropriate for your Windows or
Macintosh operating system.

WINDOWS
-------

These instructions involve copying and deleting files and folders. For more
information about how to accomplish these tasks in Windows, see your Windows
printed documentation or online help.

1. Start Creative Writer or Fine Artist.

2. Click I'm New. Type the information you want for your new user name.

3. Exit the program.

4. In the Mskids\Users folder, open the directory with the old user name.

5. Copy all files, except ~user.inf, ~maguser.inf, and ~maxuser.inf to the new
  user's directory.

6. Delete the folder with the old user name.

When you restart Creative Writer or Fine Artist, the old user name is removed and
you should follow the instructions for updating the new user.

MACINTOSH
---------

1. Start Creative Writer or Fine Artist.

2. Click I'm New. Type the information you want for your new user name.

3. Exit the program.

4. From your hard disk drive, open the MSKids folder, then open the USERS
  folder.

5. Open the folder with the old user name. Drag all files EXCEPT ~User Profile,
  ~Fine Artist Preferences, and ~Creative Writer Preferences into the new user
  name folder.

6. Drag the old user name folder to the Trash.

When you restart Creative Writer or Fine Artist, the old user name is removed and
you should follow the instructions for updating the new user.
