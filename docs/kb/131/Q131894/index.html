Q131894: Delta Knowledge Base Available as Help File (June 1995)

Article: Q131894
Product(s): Miscellaneous Software Development Kits
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 25-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Delta for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Knowledge Base for Microsoft Delta has been compiled into a help
file.

MORE INFORMATION
================

The following file is available for download from the Microsoft Software
Library:

  DELTA.EXE (http://support.microsoft.com/download/support/mslfiles/DELTA.EXE)

For more information about downloading files from the Microsoft Software Library,
please see the following article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Delte.exe contains the following Microsoft Knowledge Base articles:

  Q130883 - PRB: "Invalid Handle" Error with Delta
  Q131222 - Microsoft Delta Tips and Tricks
  Q115699 - PRB: Unexpected Network Error 15
  Q111824 - Microsoft Delta README.TXT File
  Q110979 - PRB: Persistent Network Connection Problems w/ WFW 3.1 or 3.11
  Q111403 - Hosting a Project on PC-NFS
  Q118571 - PRB: Using NetWare Versions with the VLM Redirector
  Q112531 - PRB: ERROR: <filename> Isn't a Valid File or Directory Name
  Q111548 - PRB: Delta and Bad Command or Filename Message
  Q130887 - Advantages of Saving VB Forms as Text When Using Delta
  Q130881 - How to Update a Project Using Delta as Bugs Are Resolved
  Q115305 - PRB: TimerCriticalSection and ASPI4DOS.SYS Stopping Windows
  Q111547 - PRB: Delta and Reading or Writing to Temp Files
  Q130886 - Parallel Development for Visual Basic Files w/ Microsoft Delta
  Q118389 - PRB: Temporary Variable Causes "Error Executing Command"
  Q130888 - PRB: Project Reports Corrupt Status Files with Delta
  Q130885 - How to Use Netware Remote Drives as Local Project Directories
  Q130884 - Delta for Windows Does Not Work with Windows NT
  Q130879 - How to Use the LOCALDRIVES Variable with Microsoft Delta
