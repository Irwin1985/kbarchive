Q131754: Mac Encarta 1995: Slide Show Stops After 100 Slides

Article: Q131754
Product(s): Microsoft Home Multimedia Titles
Version(s): 1995 edition
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 24-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you view approximately 100 slides from the Encarta Media Gallery slide
show, the show stops and returns to the first slide.

CAUSE
=====

The problem occurs because of a limitation in the number of slides Encarta can
show.

WORKAROUND
==========

When the slide show stops, click the play button to restart it.

MORE INFORMATION
================

Microsoft has confirmed this to be a problem in the Encarta 1995 edition for the
Macintosh. This was corrected with Encarta Encyclopedia 1996 edition.
