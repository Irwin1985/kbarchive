Q247133: XCON: List of KB Articles on MTA Concepts and Troubleshooting

Article: Q247133
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): exc55
Last Modified: 14-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article provides a list of KB articles that may help you understand some
basic message transfer agent (MTA) concepts. Articles are also listed that may
be useful for initial basic MTA troubleshooting. This is not a complete list; it
is simply a reference to some potentially useful KB articles about the MTA.
