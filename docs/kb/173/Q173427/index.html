Q173427: Error Message New MSN Error 87

Article: Q173427
Product(s): The Microsoft Network
Version(s): WINDOWS:2.0,2.5,95
Operating System(s): 
Keyword(s): kberrmsg kbmsn
Last Modified: 16-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- the operating system: Microsoft Windows 95 
- The Microsoft Network versions 2.0, 2.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you connect to MSN, The Microsoft Network, you may receive the following
error message:

  New MSN Error 87.

CAUSE
=====

This issue can occur if the Phone.pbk file is damaged or missing.

RESOLUTION
==========

To resolve this issue, replace the Phone.pbk file. To do so, use the following
steps:

1. Insert the MSN CD-ROM in the CD-ROM drive.

2. If the Welcome To MSN Setup window appears, click Cancel, and then click Yes.

3. Click Start, point to Find, and then click Files Or Folders.

4. In the Named box, type "phone.pbk" (without the quotation marks), and then
  click Find Now.

5. In the list of found files, right-click the Phone.pbk file, and then click
  Rename. NOTE: If the Phone.pbk file is not found, skip to step 7.

6. Type "phone.old" (without the quotation marks), and then press ENTER.

7. In the Look In box, click the CD-ROM drive that contains the MSN CD-ROM, and
  then click Find Now.

8. In the list of found files, right-click the Phone.pbk file, and then click
  Copy.

9. Close the Find dialog box.

10. Click Start, point to Programs, and then click Windows Explorer.

11. Click the Program Files\The Microsoft Network folder.

12. On the Edit menu, click Paste, and then close Windows Explorer.

13. Restart the computer. To restart the computer click Start, click Shut Down,
  click Restart The Computer, and then click Yes.
