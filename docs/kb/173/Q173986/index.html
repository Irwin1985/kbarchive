Q173986: Flight Simulator 98/2000: Product Identification (PID) Location

Article: Q173986
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbusage kbtlc fs98 kbimu msgamekbfaq
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
- Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to locate the Product Identification Number (PID) in
Microsoft Flight Simulator 98 and Microsoft Flight Simulator 2000.

MORE INFORMATION
================

To locate the PID number in Flight Simulator 98 or Flight Simulator 2000:

1. On the Help menu in Flight Simulator 98 or Flight Simulator 2000, click About
  Microsoft Flight Simulator.

2. In the Product ID box or on the Product ID line, note the PID number.

3. Click Close.
