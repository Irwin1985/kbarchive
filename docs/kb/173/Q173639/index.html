Q173639: Support for Localized or International Versions of Windows NT

Article: Q173639
Product(s): Microsoft Windows NT
Version(s): 2000,3.5,3.51,4.0
Operating System(s): 
Keyword(s): 
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
- Microsoft Windows 2000 Advanced Server 
- Microsoft Windows 2000 Server 
- Microsoft Windows 2000 Professional 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes support options available to users of localized (foreign
language) or international versions of Windows NT.

MORE INFORMATION
================

The following support options are available to users of localized or
international versions of Windows NT:

- Contact the international subsidiary responsible for the localized version.
  For information about locating an international subsidiary, see the Microsoft
  World Wide Offices Web site at the following location:

  http://www.microsoft.com/worldwide/default.htm

  Or, see the following article in the Microsoft Knowledge Base:

  ARTICLE ID: Q49868
  TITLE : Microsoft International Subsidiary Technical Support


- Contact the vendor that supplied the localized version of Windows NT.
