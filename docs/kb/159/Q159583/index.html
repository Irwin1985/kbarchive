Q159583: Flight Simulator 95 and the SideWinder 3D Pro Joystick

Article: Q159583
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for Windows 95, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Microsoft SideWinder 3D Pro joystick may exhibit erratic behavior in Flight
Simulator for Windows 95. For example, when you move the joystick to the left,
the view changes on the screen.

RESOLUTION
==========

To resolve this issue, ensure that you calibrate your SideWinder Joystick for
Flight Simulator for Windows 95:

1. On the Options menu in Flight Simulator, click Preferences, and then click
  Controls.

2. In the Control Type box, click Joystick.

3. Click Calibrate and follow the prompts.

Enabling Joystick Features
--------------------------

In Flight Simulator for Windows 95, you can enable all joystick features
including the Rudder, as follows:

1. On the Options menu, click Preferences, and then click Controls.

2. Make sure the Joystick On button is lit, and then click OK.

3. On the Aircraft menu, click Realism & Reliability. Turn off Auto
  Coordination.

NOTE: The Rudder may be inactive if the Auto Coordination button is left on.

Downloading Drivers
-------------------

If the erratic behavior continues, download the Microsoft SideWinder Game
Controller software version 3.02 from the following Microsoft Web site:

  http://www.microsoft.com/products/hardware/sidewinder/downloads/default.htm
