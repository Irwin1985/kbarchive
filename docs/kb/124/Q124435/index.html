Q124435: Windows 3.1 Resource Kit Files Available

Article: Q124435
Product(s): Microsoft Windows 3.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): kbfile kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups 
-------------------------------------------------------------------------------

SUMMARY
=======

The file RESKIT.EXE contains the Windows 3.1 Resource Kit. For information about
obtaining RESKIT.EXE, refer to the MORE INFORMATION section below. RESKIT.EXE
contains the following files:

RESKIT   EXE     1,785,913 12-29-94  11:45a
DISK_DRV EXE       154,018 04-03-92  11:54a
DISK_IPX EXE        44,979 04-10-92   5:18p
DISK_WRK EXE       593,062 04-03-92  11:55a
DISKROOT EXE       163,872 04-03-92  11:53a
WRK_DOC  EXE       877,801 04-03-92  11:57a
DESCRIP  TXT         1,070 05-07-92  10:31a
LICENSE  TXT        11,113 05-07-92   8:53a
KIT      TXT             0 12-29-94  11:57a
      11 file(s)      3,631,828 bytes

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  DownloadDownload Reskit.exe now
  (http://download.microsoft.com/download/win31/wrk/1/W31/EN-US/reskit.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
