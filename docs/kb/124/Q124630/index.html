Q124630: After Dark Err Msg: "An Error Has Occurred in Your Program…"

Article: Q124630
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When After Dark version 2.0 attempts to display a screen saver, you receive the
following error message:

  An error has occurred in your program and it will be terminated. If the
  problem persists, contact the program vendor.

If you click the Details button, you receive the following information:

  AD caused a general protection fault in module AD_RSRC.DLL at
  000c:0000028f.

CAUSE
=====

This error can occur if you shut down After Dark from the Close Program window
that appears when you press the CTRL+ALT+DEL key combination and then restart
After Dark or if After Dark is configured to run with no desktop icon.

RESOLUTION
==========

Shut down and then restart Windows 95. After Dark will function correctly after
Windows 95 is restarted.
