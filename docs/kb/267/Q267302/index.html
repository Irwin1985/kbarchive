Q267302: Expedia Streets & Trips 2000: Minimum System Requirements

Article: Q267302
Product(s): Microsoft Automap
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbref kbimu
Last Modified: 27-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Expedia Streets and Trips 2000 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a list of the minimum system requirements that are needed
to run Microsoft Expedia Streets & Trips 2000.

MORE INFORMATION
================

To run Microsoft Expedia Streets & Trips 2000, you need:

- Multimedia PC with Pentium-class processor
- Microsoft Windows 95 operating system, or Windows 98 or later operating
  system, or Windows NT Workstation operating system version 4.0 or later (will
  not run on earlier versions)
- 24 megabytes (MB) of RAM for Windows 95 or Windows 98 (32 MB of RAM
  recommended); 32 MB of RAM for Windows NT Workstation 4.0
- 120-170 MB of available hard disk space (170 MB required if Microsoft
  Internet Explorer 4.01 Service Pack 1 or later is not already installed and
  Microsoft Data Access Components 2.1 is not already installed); Internet
  Explorer is required and is included on the Setup CD-ROM
- Internet Explorer 4.01 Service Pack 1 (included) or later; users can maintain
  other default browsers after installation. Internet Explorer 4.01 Service
  Pack 1 is compatible with Netscape Navigator 2.0 or later; for Windows NT
  Workstation 4.0, Service Pack 3 is required.
- Double-speed or faster CD-ROM drive
- Super VGA, 256-color monitor
- Microsoft Mouse or compatible pointing device

Optional:

- Internet access to online features and functionality requires an Internet
  service provider (ISP), which may require payment of a separate fee, and
  browser software (Internet Explorer 4.01 Service Pack 1 provided). All
  software required is included in Expedia Streets and Trips 2000. Compatible
  with all Windows-compatible ISP's.
