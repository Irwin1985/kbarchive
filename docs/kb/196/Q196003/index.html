Q196003: How to Show Text in Different Fonts in Notepad

Article: Q196003
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): Windows:95,98; WinNT:4.0
Operating System(s): 
Keyword(s): 
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Windows 98 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

There are different versions of Notepad shipped with Windows 95 and Windows NT,
each having different functionality.

MORE INFORMATION
================

Notepad is shipped with Windows 95, Windows98, and Windows NT.

The Notepad shipped with Windows 95 does not have the capability of showing the
text in different fonts. This is the version of Notepad that is identified by
file version 4.00.950.

The version of Notepad shipped with Windows 98, identified by file version
4.10.1998, and the version shipped with Windows NT, identified with file version
4.00, has the capability of showing text in different fonts. In both these
versions of Notepad, the capability is limited to changing font text for the
entire document. Both these versions do not have the capability of selecting
portions of text and modifying the font for selected text.

To show text in different fonts, follow these steps:

1. On the Edit menu, click Set Font. This will bring up the font selection
  dialog box.

2. Select the preferred font from the Font list, and then select the preferred
  Font Style and Font Size settings.

3. Look at the text in the sample window to verify the correct font has been
  selected.

NOTE: The next time Notepad is used, it will uses the most recently used font.
