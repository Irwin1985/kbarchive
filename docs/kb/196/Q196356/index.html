Q196356: Visual J++ Deluxe Learning Edition Comments and Corrections

Article: Q196356
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 09-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Visual J++ 6.0 Deluxe Learning Edition ISBN 1-57231-930-5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Visual J++ 6.0 Deluxe Learning
Edition, ISBN 1-57231-930-5.

The following topics are covered:

- Pages xxi-xxii: Installing sample files for Learn VJ Now

- Page xxiii, Learn J++ Now: Software is not a trial version

- Support clause missing from introduction

MORE INFORMATION
================

Pages xxi-xxii: Installing sample files for Learn VJ Now
--------------------------------------------------------

Pages xxi and xxii, of Learn Visual J++ Now include instructions for "Installing
the Sample Projects on Your Computer". These instructions need to be modified
for the issue of Learn VJ Now that is included with the Visual J++ Deluxe
Learning Edition.

Installation instructions should read:

To install the samples

1. Insert the Visual J++ Starts Here compact disc into your CD-ROM drive.

2. Close the Installation window that appears.

3. Click Start, point to Programs, and the click Windows Explorer. If you are
  using Windows NT, click Windows NT Explorer.

4. In the All Folders area, click the CD-ROM drive. The content of the CD-ROM
  will appear.

5. Double click the Learnvj folder to open it.

6. Copy the Learnvj.exe file to your desktop.

7. Double click on the Learnvj.exe file to extract the sample code and lab
  projects. This will also save the files to your hard drive.

You should now be able to access files that correspond directly with the book.
The default location of those files will be C:\Learnvj.

Please Note:
These changes are only for the copy of Learn Visual J++ Now that is included with
the Visual J++ 6.0 Deluxe Learning Edition, ISBN 1-57231-930-5. These changes do
not apply to Learn Visual J++ Now, ISBN 1-57231-923-2.


Page xxiii, Learn J++ Now: Software is not a trial version
----------------------------------------------------------

Page xxiii, Learn Visual J++ Now:

Under the heading titled "Installing Visual J++ on Your Computer," the second
sentence indicates that the software included is a 90-day trial version. This is
inaccurate only when the book is included with the Deluxe Learning Edition
package. If the book was bought by itself, the software is a 90-day trial
edition.

Change:

  "This is a 90-day trial version of the Standard Edition of Microsoft
  Visual J++."

To:

  "This is a Standard Edition of Microsoft Visual J++."

Please Note:
These changes are only for the copy of Learn Visual J++ Now that is included with
the Visual J++ 6.0 Deluxe Learning Edition, ISBN 1-57231-930-5. These changes do
not apply to Learn Visual J++ Now, ISBN 1-57231-923-2.
