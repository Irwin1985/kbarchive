Q196049: WD97: How to Use Examples Found in Style Gallery

Article: Q196049
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta winword word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

This article describes how to use the example and style samples in the Style
Gallery (Format menu). When you select a template and then select Example in the
Style Gallery, you see a sample document formatted with the styles from the
selected template. When you select a template and then select Style Samples, you
see a list of all of the styles in the selected template and samples of the
style formats. The example and style samples are stored as AutoText entries in
their respective templates.

MORE INFORMATION
================

To use the Style Gallery examples, use the appropriate method for your
situation.

Method 1: If the Template Does Not Contain Boilerplate Text
-----------------------------------------------------------

1. Create a new document based on a Word template.

2. On the Insert menu, click AutoText, select Gallery Examples, and then click
  Insert.

This places the entire example in your document.

Method 2: If the Template Contains Boilerplate Text
---------------------------------------------------

1. Create a new document based on the Normal template.

2. On the Tools menu, click Templates And Add-Ins.

3. Click Attach.

4. Select the template that contains the Gallery Examples you want to use, and
  then click OK.

5. Click OK.

6. On the Insert menu, click AutoText, select Gallery Examples, and then click
  Insert.

This places the entire example in your document.

You can modify the inserted text and store it as a new AutoText entry or save the
document.
