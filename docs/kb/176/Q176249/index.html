Q176249: Encarta: Sounds Distorted or Do Not Play Under Windows NT 4.0

Article: Q176249
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:; winnt:4.0
Operating System(s): 
Keyword(s): kbenv kbsound kbimukbfaq
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
- Microsoft Encarta 97 Encyclopedia for Windows 
- Microsoft Encarta 98 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 99 
- Microsoft Encarta Reference Suite 99 
- the operating system: Microsoft Windows NT 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Microsoft Encarta Encyclopedia for Windows under Microsoft Windows
NT 4.0, you may be unable to hear sounds. If you attempt to play wave audio
files directly from the Encarta compact disc using Windows NT Media Player, the
playback may be audible, but the sounds may be scratchy and distorted.

CAUSE
=====

This behavior can occur if you are using outdated or incompatible sound card
drivers designed for Windows NT 3.51.

RESOLUTION
==========

To resolve this issue, obtain and install updated sound card drivers designed
for Windows NT 4.0.

MORE INFORMATION
================

When you upgrade to Windows NT 4.0, unless you download updated sound card
drivers from the sound card manufacturer, you continue to use the Windows NT
3.51 sound card drivers.

For additional information, please see the following article in the Microsoft
Knowledge Base:

  Q156285 Windows NT Setup Does Not Detect or Install Sound Cards

This behavior has been reported with the following sound cards:

- Creative Labs Sound Blaster 16
- Media Vision Pro Audio Spectrum


Creative Labs has released updated sound card drivers for Windows NT 4.0. You can
obtain the updated sound card drivers from the following Creative Labs Web
site:

  http://www.soundblaster.com
