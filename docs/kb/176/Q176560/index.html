Q176560: How to Create an Inetnews.log File for Outlook Express

Article: Q176560
Product(s): The Microsoft Network
Version(s): WINDOWS:2.5,2.51,2.52,2.6,4.0,5,5.0
Operating System(s): 
Keyword(s): kbmsn
Last Modified: 21-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 2.5, 2.51, 2.52, 2.6, 5.0 
- Microsoft Outlook Express versions 4.0, 5 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to create an Inetnews.log file for Microsoft Outlook
Express 4.0, 5.0.

You can use the Inetnews.log file to troubleshoot problems that may occur when
you connect to a news server in Outlook Express using the Network News Transport
Protocol (NNTP).

MORE INFORMATION
================

Before you begin, first check which version of Outlook Express you are have by
clicking About Microsoft Outlook Express, on the Help menu. Then, to create an
Inetnews.log file, use the appropriate method:

Outlook Express 5.0
-------------------

1. Start Outlook Express.

2. On the Tools menu, click Options.

3. Click the Maintenance tab, under Trouble Shooting, click the News check box
  to select it. NOTE: You can also create log files for MAIL, IMAP and HTTP
  this way.

4. Click Apply, and then click OK. If you connect to multiple news servers, an
  Inetnews.log file is created for each news server.

Outlook Express 4.0
-------------------

1. Start Outlook Express.

2. On the Tools menu, click Options.

3. On the Advanced tab, under Logging Options, click the News Transport check
  box to select it.

4. Click Apply, and then click OK. If you connect to multiple news servers, an
  Inetnews.log file is created for each news server.
