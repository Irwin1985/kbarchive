Q176728: How to Reinstall The Microsoft Network and Microsoft Exchange

Article: Q176728
Product(s): The Microsoft Network
Version(s): WINDOWS:1.0,1.05,1.2,1.3,2.0
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 08-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Some issues with MSN, The Microsoft Network and Microsoft Exchange may require
you to remove and reinstall these programs. This article describes how to safely
remove and reinstall The Microsoft Network with Microsoft Exchange.

NOTE: In this article, the term "Microsoft Exchange" refers to both Microsoft
Exchange and Windows Messaging.

MORE INFORMATION
================

To safely remove and reinstall The Microsoft Network and Microsoft Exchange, use
the following steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. If Microsoft Plus! for Windows 95 is installed, follow these steps:
  a. On the Install/Uninstall tab, click Microsoft Plus! For Windows 95, and
     then click Add/Remove.

  b. Click Add/Remove.

  c. Click the Internet Jumpstart Kit check box to clear it, click Continue,
     and then click Yes.


1. On the Windows Setup tab, click the Microsoft Exchange check box to clear it.

2. Click Yes to remove The Microsoft Network, and then click OK.

3. Close Control Panel, and then restart the computer.

4. Click Start, point to Settings, and then click Control Panel.

5. Double-click Add/Remove Programs.

6. On the Windows Setup tab, click the Microsoft Exchange check box to select
  it.

7. Click the The Microsoft Network check box to select it, and then click OK.

8. Follow the instructions in the Inbox Setup Wizard to configure your Inbox.

9. Connect to The Microsoft Network, and then click Yes to start the MSN
  software upgrade. When the upgrade process is completed, close Control Panel,
  and then restart the computer.

10. If you have the MSN 2.0 CD-ROM, insert it into the CD-ROM drive. If MSN
  Setup starts automatically, follow the instructions on the screen to install
  MSN 2.0. If MSN Setup does not start automatically, use the following
  steps:
  a. Click Start, and then click Run.

  b. In the Open box, type the following line, and then click OK

  "d:\setup.exe" (without the quotation marks)

     where d is the drive letter of your CD-ROM drive.

  c. Follow the instructions on the screen to install MSN 2.0.
