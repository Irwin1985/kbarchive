Q176833: FS98 Is Minimized After You Choose Context-Sensitive Help

Article: Q176833
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbui kbusage
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a context-sensitive Help topic from a dialog box in Flight
Simulator 98, you minimize the program window, the Help topic does not open, and
you cannot restore Flight Simulator to the original screen size.

CAUSE
=====

This behavior can occur if you run Flight Simulator 98 in Full Screen mode.

RESOLUTION
==========

To work around this problem, do not click context-sensitive Help topics when you
run Flight Simulator 98 in Full Screen mode. To disable Full Screen mode, click
Full Screen on the View menu, or press ALT+ENTER on the keyboard.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

You may be able to restore Flight Simulator 98 if you use both mouse buttons at
the same time to click Microsoft Flight Simulator on the taskbar. If this does
not restore Flight Simulator 98, restart the computer.
