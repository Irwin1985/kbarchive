Q176756: How to Install the Internet Explorer Active Desktop

Article: Q176756
Product(s): The Microsoft Network
Version(s): WINDOWS:2.51,4.0
Operating System(s): 
Keyword(s): kbenv kbui
Last Modified: 05-JAN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.51 
- Microsoft Internet Explorer version 4.0 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install MSN, The Microsoft Network, Microsoft Internet Explorer is not
installed with the Active Desktop component. After you complete the installation
of MSN, you may want to add the Active Desktop component to Internet Explorer.

MORE INFORMATION
================

To add the Active Desktop component to Internet Explorer, follow these steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. On the Install/Uninstall tab, click Microsoft Internet Explorer 4.0, and then
  click Add/Remove.

4. Click "Add the Windows Desktop Update component from Web site," and then
  click OK.

5. When you are prompted "Active Setup is about to determine what components are
  installed on your computer. Is this OK?" click Yes.

6. Click the Windows Desktop Update component check box to select it, and then
  click Next.

7. In the Please Select A Download Site box, click a download site near you, and
  then click Install Now.

8. Click Yes to any additional prompts.

9. When you are prompted to restart the computer, click Yes.

For instructions on how to install the full installation of Internet Explorer 4.0
from the MSN 2.51 CD-ROM, please see the following article in the Microsoft
Knowledge Base:

  Q175750 How to Install the Full Installation of Internet Explorer 4.0
