Q176168: XADM: Schema Changes for Exchange Server 5.0 and 5.5

Article: Q176168
Product(s): Microsoft Exchange
Version(s): WinNT:5.5
Operating System(s): 
Keyword(s): kbusage
Last Modified: 14-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, versions 5.0, 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

New schema attributes have been added to the Microsoft Exchange Server 5.5
directory.

MORE INFORMATION
================

The following schema attributes were added to Microsoft Exchange Server version
5.5:

  ATTRIBUTE_CERTIFICATE
  DEFAULT_MESSAGE_FORMAT
  DELEGATE_USER
  DELTA_REVOCATION_LIST
  DISPLAY_NAME_SUFFIX
  DMD_NAME
  DO_OAB_VERSION
  EXTENSION_ATTRIBUTE_11
  EXTENSION_ATTRIBUTE_12
  EXTENSION_ATTRIBUTE_13
  EXTENSION_ATTRIBUTE_14
  EXTENSION_ATTRIBUTE_15
  FORM_DATA
  FORWARDING_ADDRESS
  GENERATION_QUALIFIER
  HOUSE_IDENTIFIER
  LABELEDURI
  LIST_PUBLIC_FOLDERS
  MAIL_DROP
  OWA_SERVER
  REPLICATED_OBJECT_VERSION
  RETURN_EXACT_MSG_SIZE
  SECURITY_POLICY
  SUPPORT_SMIME_SIGNATURES
  SUPPORTED_ALGORITHMS
  TYPE
  VOICE_MAIL_FLAGS
  VOICE_MAIL_GREETINGS
  VOICE_MAIL_PASSWORD
  VOICE_MAIL_RECORDED_NAME
  VOICE_MAIL_RECORDING_LENGTH
  VOICE_MAIL_SPEED
  VOICE_MAIL_SYSTEM_GUID
  VOICE_MAIL_USER_ID
  VOICE_MAIL_VOLUME

The following schema attributes were added in Microsoft Exchange 5.0:

  ASSOC_PROTOCOL_CFG_NNTP
  CLIENT_ACCESS_ENABLED
  COMPROMISED_KEY_LIST
  CONNECTION_LIST_FILTER_TYPE
  CONNECTION_TYPE
  DISABLE_DEFERRED_COMMIT
  DISABLED_GATEWAY_PROXY
  ENABLED
  ENABLED_PROTOCOL
  ENABLED_PROTOCOL
  ENABLED_PROTOCOL_CFG
  EXCLUDED_INBOUND_NEWSFEED
  EXCLUDED_OUTBOUND_NEWSFEED
  GROUP_BY_ATTR_1
  GROUP_BY_ATTR_2
  GROUP_BY_ATTR_3
  GROUP_BY_ATTR_4
  GROUP_BY_ATTR_VALUE_DN
  GROUP_BY_ATTR_VALUE_STR
  HTTP_PUB_AB_ATTRIBUTES
  HTTP_SERVERS
  INBOUND_ACCEPT_ALL
  INBOUND_DN
  INBOUND_NEWSFEED
  INBOUND_NEWSFEED_TYPE
  INCLUDED_INBOUND_NEWSFEED
  INCLUDED_OUTBOUND_NEWSFEED
  INCOMING_PASSWORD
  INSADMIN
  INTERNET_ENCODING
  LDAP_SEARCH_CFG
  MIME_TYPES
  MODERATED
  NEWSFEED_TYPE
  NEWSGROUP_LIST
  NNTP_DISTRIBUTIONS
  NNTP_DISTRIBUTIONS_FLAG
  NNTP_NEWSFEEDS
  OBJ_VIEW_CONTAINERS
  OUTBOUND_HOST_TYPE
  OUTBOUND_NEWSFEED
  OVERRIDE_NNTP_CONTENT_FORMAT
  PORT_NUMBER
  PRESERVE_INTERNET_CONTENT
  PROMO_EXPIRATION
  PROTOCOL_SETTINGS
  PROXY_GENERATION_ENABLED
  RAS_ACCOUNT
  RAS_PASSWORD
  REPLICATED_DISTRIBUTIONS
  REQUIRE_SSL
  ROOT_NEWSGROUPS_FOLDER_ID
  VIEW_CALCULATION_LAST_USN
  VIEW_CALCULATION_LAST_USN
  VIEW_CALCULATION_SCHEDULE
  VIEW_CALCULATION_SCHEDULE
  VIEW_CALCULATION_STYLE
  VIEW_CALCULATION_STYLE
  VIEW_CONTAINER_1
  VIEW_CONTAINER_2
  VIEW_CONTAINER_3
  VIEW_DEFINITION
  VIEW_FLAGS
  VIEW_SITE

Concatenating both of these lists will yield the attribute changes since
Microsoft Exchange Server version 4.0.
