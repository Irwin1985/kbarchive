Q317902: Games: Support for Tools and Third Party Products

Article: Q317902
Product(s): Microsoft Home Games
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires II: The Age of Kings, version 2.0 
- Microsoft Age of Empires II Expansion: The Conquerors 
- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
- Microsoft Dungeon Siege, version 1.0 
- Microsoft Flight Simulator 2002 Professional Edition 
- Microsoft Flight Simulator 2002 
- Microsoft Flight Simulator 2000 Professional Edition 
- Microsoft Flight Simulator 2000 
- Microsoft Links 2001 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the tools included with the Microsoft Game listed at the
beginning of this article.

Microsoft does not provide support for the use of these tools.

MORE INFORMATION
================

Combat Flight Simulator 2 Mission Builder
-----------------------------------------

To view Help, start Combat Flight Simulator 2 Mission Builder and click the
question mark on the top right corner, or locate the following folder:

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator
  2\Help\Mb.hlp

Flight Simulator Aircraft Editor
--------------------------------

To view Help, start Aircraft Editor, click Help, and then click Help Topics.

Train Simulator Editors and Tools
---------------------------------

To view Help, start Train Simulator Editors and Tools, and then click Help on the
top right corner.

Dungeon Siege Editor
--------------------

To view Help, click Start, point to Programs (or All Programs), click "Dungeon
Siege Tool Kit", and then click "Siege Editor Manual" or "Siege University".

Links 2001 Arnold Palmer Course Designer
----------------------------------------

To view Help, start Arnold Palmer Course Designer, click Help, and then click
Help Topics.

Age of Empires Map Editor
-------------------------

No Help file available

Third-Party Tools
-----------------

Gmax Support:

Flight Simulator 2002 came with a tool, GMAX, for creating aircraft, buildings,
and other 3-dimensional objects. To receive support for GMAX, visit the
following Discreet Webs site:

  http://www.discreet.com/

The GMAX Help files are located in the following folder on your computer:

  <Drive>:\Program Files\Microsoft Games\FS2002\Gmax\Help Install

where <Drive> is the drive on which Flight Simulator 2002 Professional
Edition is installed.

NOTE: This folder path may vary depending upon your computer, and whether or not
you installed the product with the default installation path.

For help installing GMAX Help please visit the GMAX support site.

NOTE: The Help file that is included with Flight Simulator 2002 Professional
Edition does not support the first-party and third-party tools that are included
with Flight Simulator 2002 Professional Edition.
