Q137312: Cannot Run The Microsoft Network After Installing HiJaak95

Article: Q137312
Product(s): The Microsoft Network
Version(s): 1.00 1.05
Operating System(s): 
Keyword(s): 
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install HiJaak95 on your computer, you see only a blank white MSN
Central window when you connect to MSN, The Microsoft Network. MSN Today appears
(if it is set to open automatically), but none of the shortcuts work.
Additionally, you cannot navigate anywhere on MSN.

CAUSE
=====

There is a conflict between MSN and the HiJaak95 Find feature.

RESOLUTION
==========

You can run MSN if you disable the HiJaak95 Find feature. To disable the Find
feature, follow these steps:

1. In Control Panel, double-click HiJaak Catalog Setup.

2. Click the Enable HiJaak Shell Extensions check box to clear it.

3. Close HiJaak Catalog Setup.

If you want to use the HiJaak95 Find feature when you are not using MSN, repeat
the steps above. In step 2, click the Enable HiJaak Shell Extensions check box
to select it.


MORE INFORMATION
================

To shut down MSN, press CTRL+ALT+DELETE, and then click End Task. You may have
to do this several times.
