Q137887: Fury Err Msg: The Fury3 CD Is Not Correctly Inserted

Article: Q137887
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 08-NOV-2001

1.00
WINDOWS
kbsetup kberrmsg kbbuglist kbprb

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Fury 3 for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive the following Fury3 error message when you run a minimal
installation:

  The Fury3 CD is not correctly inserted. Press OK to try again. Press Cancel
  to quit.

This occurs when you have the program running from the compact disc and remove
the disc from the CD-ROM drive.

If you click Cancel, you receive another message:

  Unable to open .POD file. Check to make sure that CD is in or try
  reinstalling.

If you click OK, your computer stops responding (hangs) even if you insert the
compact disc. You then may receive another error message:

  Fury3 caused Segment Load Failure in module Win32S16.dll at 0007:7DlF".

At this point, you must restart the computer.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

Workaround
----------

Do not remove the Fury3 compact disc when the program is running.
