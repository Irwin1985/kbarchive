Q137759: List of Reference Words Used by AutoClipArt

Article: Q137759
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbusage kbwizard
Last Modified: 16-APR-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------


SUMMARY
=======

AutoClipArt is a Wizard in PowerPoint for Windows 95 that assists you by
suggesting clip art images that complement keywords found in your presentation.

When you run the Wizard, the words in your presentation are scanned and compared
to a preset list. When a match is found, that word's related clip art is
suggested.

The AutoClipArt list is stored in a binary format and cannot be edited.
