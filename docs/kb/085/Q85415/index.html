Q85415: PPT: Graphic Formats Supported by PowerPoint

Article: Q85415
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:3.0,4.0,4.0a,4.0c,7.0
Operating System(s): 
Keyword(s): kbgraphic
Last Modified: 24-MAY-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
- Microsoft PowerPoint for Windows, versions 3.0, 4.0, 4.0a, 4.0c 
-------------------------------------------------------------------------------


SUMMARY
=======

The following table lists the graphic file formats tested and supported by
Microsoft PowerPoint for Windows:

  Format                             Extension    Versions of PowerPoint
  ----------------------------------------------------------------------
  Joint Photographic Experts Group   JPG          7.0
  Kodak Photo CD                     PCD          4.0, 7.0
  GIF                                GIF          4.0, 7.0
  CorelDRAW!                         CDR          4.0, 7.0
  Targa                              TGA          4.0, 7.0
  Macintosh PICT                     PCT          3.0, 4.0, 7.0
  Computer Graphics Metafile         CGM          3.0, 4.0, 7.0
  Windows Metafile                   WMF          3.0, 4.0, 7.0
  Micrografx Designer                DRW          3.0, 4.0, 7.0
  Encapsulated PostScript            EPS          3.0, 4.0, 7.0
  AutoCAD                            PLT          3.0, 4.0, 7.0
  AutoDesk Drawing Exc.              DXF          3.0, 4.0, 7.0
  HP Graphics Language               HGL          3.0, 4.0, 7.0
  Lotus 1-2-3                        PIC          3.0, 4.0
  WordPerfect Graphics               WPG          3.0, 4.0, 7.0
  PC Paintbrush                      PCX          3.0, 4.0, 7.0
  Tagged Image File                  TIF          3.0, 4.0, 7.0
  Windows Bitmap                     BMP          3.0, 4.0, 7.0

MORE INFORMATION
================

PowerPoint 3.0 and 4.0 place their graphic import filters in the following
folder:

  C:\Windows\Msapps\Grphflt.

If other programs install 16-bit graphics filters into this folder, PowerPoint
3.0 and 4.0 can use them.

PowerPoint for Windows 95 places it's graphic import filters into this folder:

  C:\Program Files\Common Filtes\Microsoft Shared\Grphflt

If other programs install 32-bit graphics filters into this folder, PowerPoint
for Windows 95 can use them.

Microsoft does not guarantee that PowerPoint will work correctly with additional
filters, however.
