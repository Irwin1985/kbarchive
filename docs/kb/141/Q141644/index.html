Q141644: Mac Encarta 96 & 97: Unable to Copy and Paste an Image

Article: Q141644
Product(s): Microsoft Home Multimedia Titles
Version(s): 1996 edition,1997 edition
Operating System(s): 
Keyword(s): kbinterop kbusage kbprb
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Macintosh 
- Microsoft Encarta 97 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you copy an image from Encarta 1996 or 1997, the image may not be
available to paste into another program. Either the paste option will not be
available, or no image will paste when the command is chosen.

CAUSE
=====

Encarta 1996 or 1997 may be low on memory and will not have sufficient resources
to copy and paste.


RESOLUTION
==========

Allocate more memory to Encarta.

To allocate more memory to Encarta, do the following:

1. Open your hard drive and open the Encarta 96 Encyclopedia or Encarta 97
  Encyclopedia folder.

2. Click on the Encarta 96 Encyclopedia or Encarta 97 Encyclopedia icon.

3. On the File Menu, click Get Info.

4. In the Memory Requirements section, increase the Preferred size.

  NOTE: A precise or exact number is difficult to determine. For testing
  purposes, initially increase the size by 500K and if this is not sufficient,
  increase the number again.

5. Close the Get Info window, and start Encarta 1996.

NOTE: Instead of modifying the memory requirements for Encarta, you may choose to
select a portion of the article and copy it into the destination document.

MORE INFORMATION
================

For additional information on how to make more memory available to Encarta 1996,
please see the "Encyclopedia & Memory ReadMe" file found on the Encarta '96
CD-ROM.

For additional information on how to make more memory available to Encarta 1997,
please see the "Encarta 97 Read Me" file found on the Encarta 97 Disc 1 or 2
CD-ROM.
