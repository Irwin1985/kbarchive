Q141748: Mac Encarta: Older Version of QuickTime Fails to Start

Article: Q141748
Product(s): Microsoft Home Multimedia Titles
Version(s): 1994 edition,1996 edition,1997 edition
Operating System(s): 
Keyword(s): 
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
- Microsoft Encarta 96 Encyclopedia for Macintosh 
- Microsoft Encarta 97 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Encarta 1994 for the Macintosh, you receive the
following error message:

  Your system does not have QuickTime 1.6.2 installed. Please run the installer
  again to fix this problem.

When you attempt to start Encarta 1997 with a version of QuickTime before 2.0,
you receive the following error message:

  Encarta requires QuickTime 2.0 or later (including QuickTime Musical
  Instruments and PowerPlug). Please reinstall from Encarta 97 Disc 1.

followed by:

  An unknown error has occurred while initializing Encarta Encyclopedia. Please
  restart your Macintosh and try again. If you still have problems, reinstall
  from Encarta 97 Disc 1.

When you attempt to start Encarta 1996, you receive the following error message:

  Missing or older QuickTime extension. Please reinstall Encarta 96
  Encyclopedia.

followed by:

  An unknown error has occurred while initializing Encarta Encyclopedia. Please
  restart your Macintosh and try again. If you still have problems, reinstall
  from Encarta 97 Disc 1.

CAUSE
=====

A version of QuickTime prior to 1.6.2 is installed and currently loaded in the
Extensions folder or QuickTime is not installed at all. Encarta 1994 requires
QuickTime 1.6.2 or later.

RESOLUTION
==========

Install QuickTime version 1.6.2 or later.

To install QuickTime version 1.6.2, use one of the following methods:

Method 1
--------

Run the Encarta 1994 installation program again.

To run the Encarta Installer program, do the following:

1. Open the Encarta 1994, compact disc.

2. Double-click the Install Encarta icon.

3. At the Encarta 1994 Edition title screen click OK.

4. At the main installation screen, click Customize.

5. When you are prompted, select QuickTime 1.6.2 and then click Install.

6. Click Continue. This automatically quits all other open programs.

7. After the installation completes, click Restart.

Method 2
--------

Copy QuickTime 1.6.2 from the Encarta 1994 CD-ROM and replace the existing
version of QuickTime in your Extensions folder.

To manually install the QuickTime 1.6.2 system extension do the following:

1. On your Macintosh, open the System Folder.

2. Open the Extensions Folder.

3. Select the QuickTime icon.

4. On the File menu, click Get Info. Check if the version of QuickTime is lower
  than 1.6.2.

5. Close the Get Info window.

6. Drag the QuickTime system extension into the Trash.

7. Open the Encarta 1994 compact disc.

8. Open the Installer Files folder.

9. Open the System Files folder.

10. Drag the QuickTime system extension from the compact disc into the
  Extensions folder.

11. On the Special menu, click Restart.

MORE INFORMATION
================

Encarta 1994 installs QuickTime version 1.6.2. Other applications may replace
this with an older version. The replacement of Quicktime is the most common
cause of the error message.

For additional information, please see the following article in the Microsoft
Knowledge Base:

  Q121037 Mac Encarta 1994: README Contents
