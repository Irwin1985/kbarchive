Q141196: Encarta 1995: Unable to Open Files from Media Gallery

Article: Q141196
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11; :1995 edition
Operating System(s): 
Keyword(s): 
Last Modified: 28-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
- Microsoft Windows for Workgroups versions 3.1, 3.11 
- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

This article contains information about using Windows and Windows for
Workgroups with third-party products or configurations that have not
been tested and are not supported by Microsoft.

SYMPTOMS
========

When you start Media Gallery, you receive the following error message:

  Multimedia Viewer [MCI] error. Unable to open file
  \encyc95\audiom12!t048110a.wav or one of its components.

You may also receive a similar error about opening the MM95BAAG.M12 file when
attempting to run Mind Maze.

RESOLUTION
==========

Install and use a supported version of Microsoft MS-DOS rather than Novell
DR-DOS.
