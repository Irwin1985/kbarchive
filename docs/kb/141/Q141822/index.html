Q141822: 3D Movie Maker: Default Text Color Not Available in Color List

Article: Q141822
Product(s): Microsoft Home Kids Products
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 3D Movie Maker for Windows, version 1.0 
- Microsoft Nickelodeon 3D Movie Maker for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you create a Story Box or Credits Box in 3D Movie Maker, the default color
for the text is bright yellow. Bright yellow, however, is not available in the
Letter Color list in the Words tools.

STATUS
======

Microsoft has confirmed this to be a problem in 3D Movie Maker, version 1.0. We
are researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.

STEPS TO REPRODUCE PROBLEM
--------------------------

1. Create a new movie.

2. Click the Words tab to display the Words tools.

3. Click Word Box tool, which is labeled "ABC."

4. Click the Story Box.

5. Click anywhere on the scene.

6. Type some text into the Story Box.

7. Click the Letter Color tool and choose a color.

8. Highlight the text to change its color.

9. Click the Letter Color tool again. Note that the default color for the text
  is not available in the list.
