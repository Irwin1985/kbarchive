Q141405: Mac Encarta 96: Images with Text Don’t Paste

Article: Q141405
Product(s): Microsoft Home Multimedia Titles
Version(s): MACINTOSH:
Operating System(s): 
Keyword(s): 
Last Modified: 18-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Macintosh 
- Microsoft Encarta 97 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you copy an image from Encarta you are not be able to paste the image into
Microsoft Word.

CAUSE
=====

Graphic and text combination images do not Paste into some word processing
programs, such as Microsoft Word.

Some Encarta images are composed of a graphic containing a text section. These
type of images do not paste directly into some word processing programs.

RESOLUTION
==========

To paste combination graphic and text image into a program such as Microsoft
Word, do the following:

1. Copy the Encarta image.

2. Start Microsoft Word.

3. From the Edit Menu, click Paste Special.

4. Click Picture, and then click OK.

The image appears in the document.

MORE INFORMATION
================

Microsoft Works versions 3.0 and 4.0 allow the image to be directly pasted into
the word processing section. In Works 3.0, you first need to turn on the Draw
Layer.

To turn on the Draw Layer, do the following:

- On the Window Menu, click Draw On.
