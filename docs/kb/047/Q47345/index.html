Q47345: Mac Form: HyperCard Needs 1000K RAM with Mail Form Designer

Article: Q47345
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

You may experience problems when using the Microsoft Mail Form Designer if you
assign HyperCard less than the suggested 1000K of random access memory (RAM)
under MultiFinder. To assign RAM to HyperCard, select the HyperCard application
icon and choose Get Info from the File menu. At the bottom of the Get Info
dialog box, there is an edit box in which you can enter the desired RAM
allocation.

MORE INFORMATION
================

If HyperCard is not assigned enough memory, you may encounter the following
problems:

1. If HyperCard is assigned 600K or less, HyperCard may not launch when you
  double-click the Form Designer icon.

2. If HyperCard is assigned 700K or less, you may get out-of-memory messages or
  the Macintosh may hang.
