Q224229: Streets and Trips 2000: Unable to Print Route Driving Directions

Article: Q224229
Product(s): Microsoft Automap
Version(s): 
Operating System(s): 
Keyword(s): kbprint kbui kbimu
Last Modified: 10-OCT-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Expedia Streets and Trips 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to print a route you planned in Microsoft Expedia Streets and
Trips 2000, the Driving Directions options in the Print dialog box may be
unavailable.

CAUSE
=====

This behavior can occur if you click the Map tab in the Print dialog box before
you attempt to print the route.

RESOLUTION
==========

To resolve this issue, click the Directions, Pictures, or Information tab in the
Print dialog box before you attempt to print the route.
