Q323942: Asheron’s Call: Err Msg: You Do Not Have the Current Version…

Article: Q323942
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 26-JUN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to play Asheron's Call in the Microsoft Gaming Zone, you may
receive an error message similar to the following:

  You do not have the current version of Asheron's Call installed.

CAUSE
=====

This problem may occur if you try to connect to the Asheron's Call game lobby
while Microsoft is updating the site. The game files may not yet be available.

RESOLUTION
==========

To resolve this problem, sign out, and then try to sign in later. To do this,
follow these steps:

1. In the Microsoft Gaming Zone, click Sign Out.

2. Later, click Sign In.

3. When you are prompted, enter your Microsoft .NET Passport credentials, and
  then click OK.

4. In the list of games, click Asheron's Call.

  If this does not resolve the problem, repeat these steps until the game files
  are available.

MORE INFORMATION
================

For information about the status of the Asheron's Call server, visit the
following Microsoft Web site:

  State of the Network for Asheron's Call
  http://zone.msn.com/asheronscall/news/ashenewsnetwork.asp

For more information about the Microsoft Gaming Zone, visit the following
Microsoft Web site:

  Microsoft Gaming Zone
  http://www.zone.com
