Q187396: Monster Truck Madness 2: Minimum System Requirements

Article: Q187396
Product(s): Microsoft Home Games
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kbref kbimu msgame mtm2kbfaq
Last Modified: 12-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Monster Truck Madness 2, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the minimum system requirements to play Microsoft Monster
Truck Madness 2.

MORE INFORMATION
================

To use Monster Truck Madness 2, you need:

Required
--------

- A multimedia PC with a Pentium 133 or faster processor.

- Microsoft Windows 95/98 or Microsoft Windows NT Workstation 4.0 with Service
  Pack 3.

- 16 megabytes (MB) of random access memory (RAM) (32 MB recommended).

- 30 MB of available hard disk space (110 MB recommended).

- A quad-speed or faster CD-ROM drive (6X or faster recommended).

- A Super VGA, 16-bit color monitor (Microsoft Direct3D compatible graphics
  accelerator card recommended).

- A Microsoft mouse or compatible pointing device (joystick or race car
  controller recommended).

- A DirectSound compatible sound card with speakers or headphones.

Optional
--------

- A 28,800 bits per second (bps) or faster modem for head-to-head play.

- A local area network (LAN) that supports either the TCP/IP or IPX/SPX network
  protocols for LAN play.

- An Internet service provider (ISP) and access to the Internet for Internet
  play.

  NOTE: Internet access may require payment of a separate fee to an Internet
  service provider. Connect time charges may apply.

- A microphone for voice communication during head-to-head play.

Compatibles
-----------

- Supports force-feedback hardware compatible with Microsoft DirectInput
  (Windows 95/98 only).

- Supports MMX technology.

- Supports Pentium II processor platform with AGP.
