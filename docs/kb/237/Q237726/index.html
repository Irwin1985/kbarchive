Q237726: NFL Fever 2000: Wrong Team Has Possession of the Ball When You R

Article: Q237726
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbimu msgame
Last Modified: 12-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft NFL Fever 2000, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you resume playing a saved game of Microsoft NFL Fever 2000, the wrong team
may have possession of the ball.

CAUSE
=====

This behavior can occur if you selected the same team to be both the home team
and the away team.

RESOLUTION
==========

To prevent this issue from occurring, do not select the same team to be both the
home team and the away team.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000,
version 1.0.
