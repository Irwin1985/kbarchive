Q136008: PC Ext: The F2 Key Does Not Display All Postoffices

Article: Q136008
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:3.0,3.0a,3.2,3.2a,3.5
Operating System(s): 
Keyword(s): 
Last Modified: 18-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for PC Networks, versions 3.0, 3.0a, 3.2, 3.2a, 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the External Mail program with 13 or more postoffices, and you
press the F2 key to view the status of postoffices and use the down arrow to
scroll to view other postoffices, you do not see the 13th postoffice in the
list.

RESOLUTION
==========

To see all available postoffices, use the down arrow key to scroll to the bottom
of the list; then use the up arrow key to go back to the top of the list. The
missing postoffice will appear as you scroll up.

STATUS
======

Microsoft has confirmed this to be a problem in versions 3.0, 3.0a, 3.2, 3.2a,
and 3.5 of Microsoft Mail for PC Networks. We are researching this problem and
will post new information here in the Microsoft Knowledge Base as it becomes
available.

Additional query words: 3.00 3.00a 3.20 3.20a 3.50 3.x
