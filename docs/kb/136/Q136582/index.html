Q136582: Help Converting from FoxPro 2.x to Visual FoxPro 3.0

Article: Q136582
Product(s): Microsoft FoxPro
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 11-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

For information on converting from FoxPro version 2.6 to Visual FoxPro, choose
Technical Reference from the Help Contents menu. Then click Upgrading, and click
Converting from FoxPro 2.6. In this topic, choose "Converting FoxPro 2.6 Files."

MORE INFORMATION
================

The "Converting FoxPro 2.6 Files" topic lists five types of file conversions:
projects, screens, reports, labels, and foundation READs. Click one of the
links, or read all the Conversion topics in order by clicking the right Browse
button (>>). Once you've converted the individual files, use the
"Redesigning Converted FoxPro 2.6 Files" topic to take advantage of the new
Visual FoxPro features.
