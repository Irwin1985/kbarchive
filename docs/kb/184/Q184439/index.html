Q184439: Clicking Web Link in Document Prompts You to Disconnect from MSN

Article: Q184439
Product(s): The Microsoft Network
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kbinterop kbmsn
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you open a Microsoft Office 97 document while you are connected to MSN, The
Microsoft Network, and you click a Web link embedded in the document, you
receive the following message:

  Thank you for using MSN. OK to disconnect? Click Yes to disconnect or No to
  remain connected.

If you click No, you remain connected to MSN and Microsoft Internet Explorer
opens the Web page the link points to. However, you continue to receive the
above message each time you click a Web link embedded in a document.

RESOLUTION
==========

To resolve this issue, upgrade to MSN version 2.5. To do so, use one of the
following methods:

Method 1
--------

To order your free MSN 2.5 CD-ROM either telephone (800) 373-3676 or visit the
following Microsoft Web site:

  http://free.msn.com

Method 2
--------

To install MSN 2.5 from the Internet, visit the following Microsoft Web site:

  http://install.msn.com

STATUS
======

Microsoft has confirmed this to be a problem in The Microsoft Network version
2.0.

Additional query words: msnet msnetwork net m.s.n. hyperlink off97
