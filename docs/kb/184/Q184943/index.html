Q184943: Monster Truck Madness 2: Screen Flashes Constantly

Article: Q184943
Product(s): Microsoft Home Games
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbenv kbhw kbimu msgame kbHardwarekbfaq
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Monster Truck Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Monster Truck Madness 2, the screen may flash constantly.

CAUSE
=====

This behavior can occur if all of the following conditions are true:

- An ATI 3D Rage Pro AGP video card is installed in your computer.
- Direct3D Hardware Acceleration is enabled on your computer.
- You are playing the game in a window.

RESOLUTION
==========

To resolve this issue, contact ATI Technologies to inquire about how to obtains
and install version 4.10.2278 or later of the video drivers for your ATI 3D Rage
Pro AGP video adapter. For information about how to contact ATI Technologies,
please see the following article in the Microsoft Knowledge Base:

  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K


MORE INFORMATION
================

For information about how to install an updated video driver in Microsoft
Windows, please see the following article in the Microsoft Knowledge Base:

  Q131806 Windows 95/98: How to Install or Change a Display Driver
