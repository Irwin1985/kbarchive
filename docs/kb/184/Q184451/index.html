Q184451: Web Advertising & Marketing by Design Comments and Corrections

Article: Q184451
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 06-APR-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Web Advertising and Marketing by Design ISBN 1-57231-836-8 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains information on known errors, corrections, and comments
relating to the Microsoft Press book "Web Advertising and Marketing by Design,"
ISBN 1-57231-836-8.

Contents:

- Page xvi: Web address incorrect

- Page 130: Web and E-mail Addresses Incorrect

MORE INFORMATION
================

Page xvi: Web address incorrect
-------------------------------

Page xvi: "Where to Find Art and Text Files"
The Microsoft Press Web address listed at the bottom of the page results in a
"404 Not Found" error.

Change:

  http://microsoft.com/mspress/products/1576

To:

  http://mspress.microsoft.com/mspress/products/1576


Page 130: Web and E-mail Addresses Incorrect
--------------------------------------------

The Web and e-mail addresses given on page 130 are not accurate. This is due to a
misspelling of DJ Delorie's name on page 130.

Change the Web address for DJ Delorie's Web Page Backwards Compatibility Viewer
to:

  http://www.delorie.com/web/wpbcv.html

Change DJ Delorie's e-mail address to:

  dj@delorie.com

Change each instance of "DJ Delore" to "DJ Delorie."
DJ Delorie's name is misspelled on page 130. Change each instance of "DJ Delore"
to "DJ Delorie".


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of the book, it may already
contain the above corrections.

Additional query words: press ms_presss 1-57231-836-8
