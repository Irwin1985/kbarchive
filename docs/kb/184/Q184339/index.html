Q184339: How to Configure Outlook Express to Automatically Check MSN Mail

Article: Q184339
Product(s): The Microsoft Network
Version(s): WINDOWS:2.5,4.0
Operating System(s): 
Keyword(s): kbui kbmsnkbfaq
Last Modified: 21-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.5 
- Microsoft Outlook Express version 4.0 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to configure Microsoft Outlook Express to
automatically check for new e-mail messages on MSN, The Microsoft Network.

MORE INFORMATION
================

To configure Outlook Express to automatically check for new e-mail messages on
MSN, follow these steps:

1. Start Outlook Express.

2. On the Tools menu, click Options.

3. On the General tab, click the Check for new messages every <n>
  minute(s) check box to select it, where <n> is a number between 1 and
  480.

4. In the Check for new mail every <n> minute(s) box, click the value you
  want for <n>. This value determines how often (in minutes) Outlook
  Express checks for new messages.

5. Click OK.

6. Click Start, and then double-click the MSN icon to start MSN.

7. On the MSN Sign-In screen, click the Dial Automatically check box to select
  it, and then click Connect.

NOTE: This procedure enables programs to connect to MSN when you are not present.
If you do not want programs to connect to MSN when you are not present, click
the Dial Automatically check box to clear it, and then click To work around this
issue, use either of the following methods:

- Wait several seconds until you attempt to reconnect to MSN.

- Attempt to sign in to MSN using an incorrect Member ID or password. When you
  receive the Cannot connect to the Microsoft Network... error message, click
  OK, type your correct MSN Member ID and password in the MSN Sign In screen,
  and then click Connect.

For additional information about how to configure Outlook Express for The
Microsoft Network, please see the following article in the Microsoft Knowledge
Base:

  Q173952<A0>How to Configure Outlook Express for The Microsoft Network
