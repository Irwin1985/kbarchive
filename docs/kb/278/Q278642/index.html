Q278642: CFS2: How to Import Missions from Combat Flight Simulator

Article: Q278642
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbenv kbimu kbGraphxLink kbgraphxlinkcritical
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to import Microsoft Combat Flight Simulator: WWII
Europe Series missions into Microsoft Combat Flight Simulator 2: WWII Pacific
Theater.

NOTE: Microsoft does not provide technical support for importing missions into
Combat Flight Simulator 2.

MORE INFORMATION
================

To import Combat Flight Simulator missions into Combat Flight Simulator 2:

1. Click Start, and then click Run.

2. In the Open box, type the following line, and then click OK:

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator\Aircraft

where <drive> is the letter of the hard disk on which Combat Flight
Simulator is installed.

3. On the Edit menu, click Select All.

4. On the Edit menu, click Copy.

5. Click Start, and then click Run.

6. In the Open box, type the following line, and then click OK:

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator
  2\Aircraft

where <drive> is the letter of the hard disk on which Combat Flight
Simulator 2 is installed.

7. On the Edit menu, click Paste.

NOTE: If you receive the following message, click No:

  This folder already contains a file named <filename>. Would you like to
  replace the existing file with this one?

8. Close all open windows on the desktop.

9. Repeat steps 1 through 8, except type the following line in step 2

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator\Gauges

and type the following line in step 6

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator 2\Gauges

10. Repeat steps 1 through 8, except type the following line in step 2

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator\Missions

and type the following line in step 6

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator
  2\Missions

11. Repeat steps 1 through 8, except type the following line in step 2

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator\Sound

and type the following line in step 6

  <drive>:\Program Files\Microsoft Games\Combat Flight Simulator 2\Sound

12. Click Start, point to Programs, point to Microsoft Games, point to Combat
  Flight Simulator 2, and then click Mission Builder.

13. Click the "Load mission" button ([GRAPHIC:the open folder icon]).

14. In the Filters section, click "Single mission".

15. In the "Player nationality" box, click Britain or Germany.

16. In the Missions box, click the mission that you want, and then click OK.

17. Click Yes each time you receive the following error message:

  Model <model number> isn't found in the object data base. Replace this
  model by the default model <model name>?

18. Click the Save mission button (the floppy disk icon) to save the mission in
  the Mission folder in Combat Flight Simulator 2.

19. Click the player nationality that you selected in step 15.

20. Click OK.

21. Click Yes to overwrite the file.

22. Repeat steps 12 through 21 to save other British and German missions that
  you want to play in Combat Flight Simulator 2.

To play Combat Flight Simulator missions in Combat Flight Simulator 2:

1. Start Combat Flight Simulator 2.

2. Click Single Missions.

3. In the "Select nationality" box, click British or German.
