Q278515: How to use the "page options" feature on MSN Companion

Article: Q278515
Product(s): The Microsoft Network
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The page options button on the toolbar reveals several options available to the
user.

MORE INFORMATION
================

There are several options available to the user regardless of what page you are
on. Click page options on the toolbar at the bottom of the screen to access
these choices.

- Frame a picture from this page opens the frame a picture page, at which you
  can select any picture from the page to become the picture viewed when your
  MSN Companion is at rest.

- Send this page by mail opens up an e-mail form that includes the Web address
  of the current page so that you can share it via e-mail.

- Add this page to my favorites opens up an add a favorite window in which you
  can accept the default name of the page or type a new name and then click add
  to favorites or click cancel.

- Print this page opens up a print options window in which you can elect to
  print in black and white and then click print page, don't print page, or
  help.

- Reload this page refreshes the current page.
