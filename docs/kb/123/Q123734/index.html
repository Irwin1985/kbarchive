Q123734: Writer/Artist: CD-ROM System Requirements

Article: Q123734
Product(s): Microsoft PowerPoint for Windows
Version(s): 1.1,1.1a
Operating System(s): 
Keyword(s): 
Last Modified: 06-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Creative Writer for Macintosh CD-ROM, versions 1.1, 1.1a 
- Microsoft Creative Writer for Windows CD-ROM, versions 1.1, 1.1a 
- Microsoft Fine Artist for Macintosh CD-ROM, versions 1.1, 1.1a 
- Microsoft Fine Artist for Windows CD-ROM, versions 1.1, 1.1a 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the minimum system requirements to successfully run Creative
Writer CD-ROM and Fine Artist CD-ROM:

WINDOWS
-------

Computer:     Intel 386SX or higher or other 100-percent compatible PC
             systems Minimum 16 MHz CPU.

Memory (RAM): Minimum of 4 megabytes (MB) RAM

Hard Disk Space: 9 MB free disk space

CD-ROM Drive: CD-ROM drive: The CD-ROM drive should transfer data from
             the CD-ROM at 150K per second while using less than 40
             percent of the CPU bandwidth and have an average seek time
             of less than 1 second.

             MS-DOS CD-ROM Extension (MSCDEX) version 2.2 or
             later. Check the documentation for your CD-ROM drive.

Video:        SVGA 640 x 480 X 16 colors (256 color SVGA recommended)

Operating System Requirements: MS-DOS 3.2 or later
                              Microsoft Windows 3.1 or later

Sound Requirements: Any MPC compatible sound card capable of
                   playing .WAV files, such as:

                    - Microsoft Windows Sound System
                    - Creative Labs Sound Blaster or compatible sound
                      card
                    - Media Vision Pro Audio or compatible sound card

Other Requirements: Stereo sound amplification (speakers) or headphones
                   Mouse

MACINTOSH
---------

Computer:        Macintosh LCII or higher (68030 processor 16MHz).

Memory (RAM):    Minimum of 4 megabytes (MB) RAM

Hard Disk Space: 8 MB free disk space

CD-ROM Drive:    Any Macintosh compatible CD-ROM drive.

Video:           Color capable monitor.
