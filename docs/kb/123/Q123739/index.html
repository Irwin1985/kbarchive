Q123739: Updated AMI BIOS Needed for Large IDE Drives and 64+ MB Memory

Article: Q123739
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): winnt:3.5
Operating System(s): 
Keyword(s): win31
Last Modified: 08-AUG-2001

3.10 3.50
WINDOWS
kb3rdparty

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 3.1 
- Microsoft Windows NT Workstation version 3.1 
- Microsoft Windows NT Advanced Server, version 3.1 
- Microsoft Windows NT Workstation version 3.5 
- Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

AMI BIOSs dated prior to 7/25/94 do not provide Logical Block Addressing
(LBA) and the necessary enhanced registers to work correctly with IDE hard
drives larger than 540 megabytes.

AMI BIOSs dated prior to 10/10/94 do not support more than 64 MB of
RAM on ISA systems. AMI BIOSs dated 10/10/94 and later support up to four
gigabytes on ISA systems.
