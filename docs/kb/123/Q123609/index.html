Q123609: Languages Supported by HP Plotters

Article: Q123609
Product(s): Microsoft Windows NT
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 3.5 
- Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

The Windows NT 3.5 hardware compatibility list (HCL) includes the HP-GL/2
plotter. The driver for these plotters that ships with Windows NT 3.5
should support all HP-GL/2 plotters, but you need to determine if your
plotter is an HP-GL/2 plotter. This article provides a list of HP Plotters
and their supported languages.

NOTE: HP-GL/2 is a superset of HP-GL. Plotters that support HP-GL, but not
HP-GL/2, do not work correctly with the HP-GL/2 plotter driver.

Not all of these devices are supported by Windows NT. This list will help
you determine if your plotter supports the HP-GL/2 language.
