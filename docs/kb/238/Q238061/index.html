Q238061: Encarta Dictionary 2000: No Integration with Word or PowerPoint

Article: Q238061
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:; :
Operating System(s): 
Keyword(s): kbinterop kbui kbdocerr kbimu
Last Modified: 27-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta World English Dictionary 
- Microsoft Encarta World English Dictionary 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Microsoft Word 2000 or Microsoft PowerPoint 2000, you may
experience one or more of the following symptoms:

- The Encarta World English Dictionary 2000 button may not appear on the Word
  2000 toolbar.

- If you click Look Up Reference on the Tools menu, Encarta World English
  Dictionary 2000 is not listed in the "Available reference titles" box.

CAUSE
=====

This behavior can occur if either of the following conditions is true:

- The Word 2000 standard toolbar is not selected.

- You install Encarta World English Dictionary 2000 before you install
  Microsoft Office 2000.

RESOLUTION
==========

To resolve this issue, use the following methods in the order in which they are
presented.

Select the Word 2000 Standard Toolbar
-------------------------------------

To do this:

1. On the Tools menu in Word 2000, click Customize.

2. On the Toolbars tab, click to select the Standard check box.

3. Click Close.

If the issue continues to occur, remove and reinstall Encarta World English
Dictionary 2000.

Remove and Reinstall Encarta World English Dictionary 2000
----------------------------------------------------------

To do this:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. Click Microsoft Encarta World English Dictionary 2000, and then click
  Add/Remove.

4. Follow the instructions on the screen to remove Encarta World English
  Dictionary 2000.

5. Close Control Panel.

6. Insert the Microsoft Encarta Reference Suite 2000 Installation and Resources
  CD-ROM or the Encarta World English Dictionary 2000 CD-ROM into your CD-ROM
  or DVD-ROM drive, and then follow the instructions on the screen to reinstall
  Encarta World English Dictionary 2000.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta World English
Dictionary 2000.

MORE INFORMATION
================

The "Defining words from within Microsoft Word or Microsoft PowerPoint" Help
topic in Encarta World English Dictionary 2000 incorrectly states that the
Define button is available on the standard toolbar in PowerPoint 2000.
