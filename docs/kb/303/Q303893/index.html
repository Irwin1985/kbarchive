Q303893: MechCommander 2.0: System Requirements

Article: Q303893
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MechCommander 2.0, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article summarizes the minimum system requirements to play Microsoft
MechCommander 2.0.

MORE INFORMATION
================

To run MechCommander 2.0, you need the following:

   - A 266-megahertz (MHz)-or-faster processor (400-MHz-or-faster processor with
  MMX or 3D Now if no hardware acceleration). Intel Pentium/Celeron family, AMD
  K6/Athlon/Duron family, or a compatible processor.

- Microsoft(r) Windows(r) 98, Microsoft Windows Millennium Edition (Me), or
  Microsoft Windows 2000 operating system with DirectX(r) 8.0 installed.
  (DirectX 8.0 is included on the MechCommander 2 compact disc.)

- A 16-bit sound card compatible with DirectX 8.0 or later. Speakers or
  headphones required for audio.

- Microsoft Mouse-or-compatible pointing device.

- Microsoft Keyboard-or-compatible keyboard device.

- A CD-ROM or DVD-ROM drive of 4x speed or faster.

- 64 megabytes (MB) of synchronous dynamic random access memory (SDRAM).

- 400 MB of available hard disk space for a typical install; an additional 200
  MB of hard disk space for the swap file.

- A Super VGA 16-bit color monitor. 8 MB video random access memory (VRAM) with
  6 MB for textures.

The following are additional minimum requirements for multiplayer game play:

   - A 56-kilobytes-per-second (Kbps) modem or local area network (LAN) with
  TCP/IP or Internetwork Packet Exchange (IPX) protocol.

- Internet access for Internet play.

  NOTE: May require payment of separate fee to an Internet service provider.
  Local and long-distance telephone toll charges may apply.
