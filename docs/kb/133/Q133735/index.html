Q133735: PPT: Rotated Text Prints Horizontally on Canon LBP 1260 Printer

Article: Q133735
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0,97
Operating System(s): 
Keyword(s): kb3rdparty kbprint kbdta kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint 97 for Windows 
- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you print a slide that contains rotated text to a Canon LBP 1260 printer, the
rotated text prints horizontally.

CAUSE
=====

This problem occurs if you use the Windows 3.x printer driver for this printer
under Windows 95.


WORKAROUND
==========

Use the Windows 95 "Add a Printer" Wizard to install the HP LaserJet IIISi
printer driver. Print to the LBP 1260 using this driver.

For more information about installing printers in Windows 95, please see the
following Windows Help topic:

  To set up a new printer

For more information about installing printers in Windows NT 3.51, please see
your Windows NT documentation.

MORE INFORMATION
================

This printer is PCL5 compliant so any PCL5 driver, such as the HP LaserJet
IIISi, should work correctly.

You cannot print in color using the HP LaserJet IIISi driver. The IIISi supports
only black and white printing.
