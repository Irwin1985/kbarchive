Q133290: Dogs: Control Menu Does Not Appear When Minimized

Article: Q133290
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0,3.x
Operating System(s): 
Keyword(s): win31kbbuglist kbfaq
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Dogs for Windows, version 1.0 
- the operating system: Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

The control menu, which contains Windows commands when an application is
minimized, does not appear when the Microsoft Dogs icon is clicked once. This
occurs on a personal computer configured for 640 x 480 video resolution.

MORE INFORMATION
================

When Windows applications are minimized in Microsoft Windows version 3.x and
Microsoft Windows for Workgroups version 3.x, clicking the minimized icon once
should produce a control menu with various commands. For example, you should see
the following commands:

  Restore
  Move
  Close
  Switch To

When Microsoft Dogs is minimized on a personal computer configured for 640 x 480
video resolution, the control menu is not accessible by clicking the icon once.
