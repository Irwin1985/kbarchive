SUMMARY
=======

The following table lists confirmed problems in Windows NT version 3.51. This is
not a comprehensive list and is current as of 21-Sep-1995. For more information
about each issue, query in the Microsoft Knowledge Base on the article
identifier (ID). For a current list of confirmed problems in Windows NT version
3.51, query on KBBUG3.51 in the Microsoft Knowledge Base.
