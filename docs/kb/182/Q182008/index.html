Q182008: WD97: Word Hangs Viewing Headers on Compaq Presario 1220 Laptop

Article: Q182008
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta word97
Last Modified: 14-NOV-2000

SYMPTOMS
========

Microsoft has received reports that on Compaq Presario 1220 laptop, Microsoft
Word may stop responding when you open files that contain headers and footers,
or when you click Headers And Footers on the View menu in a new document.


WORKAROUND
==========

Microsoft has received reports that this problem does not occur in Microsoft
Word 97 Service Release 1 (SR-1).

For additional information about SR-1, please see the following article in the
Microsoft Knowledge Base:

  Q172475 OFF97: How to Obtain and Install MS Office 97 SR-1
