Q259014: Golf 99 Err Msg: No Service Provider Detected

Article: Q259014
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf 1999 Edition, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to play a multiplayer game in Microsoft Golf 1999 Edition, you
may receive the following error message:

  No service provider detected.

CAUSE
=====

This behavior can occur if the French-language version of Microsoft Windows 98
is installed on your computer.


STATUS
======

Microsoft has confirmed that this is a problem in Microsoft Golf 1999 Edition.

MORE INFORMATION
================

This behavior occurs in both the English and French-language versions of Golf
1999.
