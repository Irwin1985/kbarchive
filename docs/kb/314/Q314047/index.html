MORE INFORMATION
================

Page 3: Incomplete Step
-----------------------

On page 3, change:
"2. Click Start, right-click My computer and select the files and folders you
want to write to the CD."

To:
"Click Start, right-click My Computer, and then click Explore and select the
files and folders you want to write to the CD."


Page 4: Missing Step Under Compressed Folder
--------------------------------------------

On page 4, under "Compressed Folder", add a step between steps 1 and 2, so the
process reads:

"1. Click Start, right-click My Computer, and then click Explore

2. Select a local drive letter, such as C:

3. On the File menu, click New, and then click Compressed Folder."


Page 8: Incorrect Steps For Accessing Instant Messaging
-------------------------------------------------------

On page 8,

Change:
"1. Click Start, point to all programs, and then click Internet Explorer.
2. Click Online Buddies, and double-click the contact name for the person you
want to talk to."

To:
"1. Click Start, point to all programs, and then click Windows Messenger.
2. Double-click the contact name for the person you want to talk to that shows up
under online."


Page 9: Incomplete Step
-----------------------

On page 9, in step 2,

Change:
"2. Point to Accessories, point to Communication, and then click Remote Desktop
Connection..."

To:
"2. Point to Accessories, point to Communication, and then click Remote Desktop
Connection, and then click Options..."


Page 16: Remove "Device Driver Referral Site" Section
-----------------------------------------------------

On page 16, please remove the entire "Device Driver Referral Site" section, as it
does not apply to Windows XP Final Release.


Page 67: "prepareinstall" Should Be "duprepare"
-----------------------------------------------

On page 67, under "/dushare: pathname", change:
"When run from your installation share and used with /prepareinstall, it prepares
the updated files for use in network-based client installations. When used
without /prepareinstall..."

To:
"When run from your installation share and used with /duprepare, it prepares the
updated files for use in network-based client installations. When used without
/duprepare..."


Page 68: "UDF File" Should Be "UDB File"
----------------------------------------

On page 68, in the last paragraph of Table 2.4, all references to "UDF file"
should be changed to "UDB file".

Change:
"/udf:id[,udb_file]

Indicates an identifier (id) that Setup uses to specify how a UDF modifies an
answer file. The UDF file overrides values in the answer file, and the
identifier determines which values in the UDF file are used. For example,
/udf:RAS_user, OUR_COMPANY.UDF overrides settings that are specified for the
RAS_user identifier in the OUR_COMPANY.UDF file. If you do not specify a UDF
file, Setup prompts you to insert a disk that contains the $UNIQUE$.UDF file."

To:
"/udf:id[,udb_file]

Indicates an identifier (id) that Setup uses to specify how a UDB modifies an
answer file. The UDB file overrides values in the answer file, and the
identifier determines which values in the UDB file are used. For example,
/udf:RAS_user, OUR_COMPANY.UDB overrides settings that are specified for the
RAS_user identifier in the OUR_COMPANY.UDB file. If you do not specify a UDB
file, Setup prompts you to insert a disk that contains the $UNIQUE$.UDB file."


Page 90: Incorrect Reference To Table 3.1
-----------------------------------------

The last bulleted item under "Password Requirements" on page 90 incorrectly
refers to Table 3.1:

"(See Table 3.1 for a list of characters that are not valid.)"

Table 3.1 lists the invalid nonalphanumeric characters for the user account name,
not that for the password.

change:

To:
"the Member Of tab in the user-name Properties dialog box"
