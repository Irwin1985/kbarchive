Q314701: ENC: How to Install Previous Version of Encarta

Article: Q314701
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 28-JUN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Reference Library 2003, version 1.0 
- Microsoft Encarta Reference Library 2003 - DVD Edition 
- Microsoft Encarta Reference Library 2002, version 1.0 
- Microsoft Encarta Reference Library 2002 - DVD Edition 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install Encarta Reference Library 2003, previous versions of Encarta
are uninstalled. The article describes how you can install and use earlier
versions of Microsoft Encarta with Encarta Reference Library 2003.

MORE INFORMATION
================

Encarta Reference Library 2003 includes the following titles:

- Encarta Encylopedia Deluxe

- Encarta Dictionary with pronunciation

- World Atlas

- Thesaurus

- Historical Timelines

- Encarta Researcher

- Encarta Africana

- Homework Starters

- Foreign language translations

- Quotations

- Literature Guides

- Chart Maker

- Subscription to Club Encarta

The previous versions of Encarta may have included material or features that are
no longer available. To install a previous version of Encarta, insert the
installation CD and follow the instructions on the screen. Select Custom
installation if this installation is available.

When you install Encarta Reference Library 2003, earlier versions of Encarta are
automatically uninstalled. However, you can install the earlier versions after
you have installed Encarta Reference Library 2003. All versions can co-exist.
Encarta Reference Library 2003 uses different folder names and registry entries,
so there are no known conflicts.
