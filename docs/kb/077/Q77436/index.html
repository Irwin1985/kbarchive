Q77436: DEC Printer Emulations for Windows 3.0

Article: Q77436
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 10-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SUMMARY
=======

The following are Digital Equipment Corporation (DEC) printer emulations for
Windows 3.0:

  DEC Printer             Windows Driver
  -----------             --------------

  LA70                    IBM ProPrinter
  LA75                    IBM ProPrinter
  LA210                   IBM Graphics
  LA324                   IBM ProPrinter 24
  LJ250                   HP PaintJet series
  LJ252                   HP PaintJet series
  LN03R                   Digital LN03R ScriptPrinter
  DECLaser 2100           Canon LBP8
   with CaPSL cartridge
  DECLaser 2150           PostScript printer
   with PostScript cartridge
  DECLaser 2200           Canon LBP8 III
   with CaPSL cartridge
  DECLaser 2250           PostScript printer
   with PostScript cartridge
  LPS20                   Digital LPS PrintServer 40
  LPS40                   Digital LPS PrintServer 40

MORE INFORMATION
================

For more information, contact DEC technical support.

The products included here are manufactured by vendors independent of Microsoft;
we make no warranty, implied or otherwise, regarding these products' performance
or reliability.

REFERENCES
==========

"Pathworks for DOS: Microsoft Windows Support Guide," pages 3-7.
