Q108363: Changes Made by Video for Windows 1.1 Upgrade

Article: Q108363
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:1.1
Operating System(s): 
Keyword(s): 
Last Modified: 22-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Video for Windows, version 1.1 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a complete listing of files that are copied and modified during the
installation of the Video for Windows 1.1 upgrade.

Files Copied to WINVIDEO Directory
----------------------------------

VIDCAP   HLP
VIDCAP   EXE
VFW      WRI
VIDEDIT  EXE
VIDEDIT  HLP
CAPSCRN  HLP
CAPSCRN  EXE
SAMPLE   AVI

Files Copied to WINDOWS Directory
---------------------------------

MPLAYER.REG
MCIOLE.DLL
MPLAYER.HLP
MPLAYER.EXE
INDEO.INI

Files Copied to WINDOWS\SYSTEM Directory
----------------------------------------

DVA.386

ACMCMPRS.DLL
CTL3D.DLL
AVICAP.DLL
MCIAVI.DRV
AVIFILE.DLL
COMPOBJ.DLL
MSVIDEO.DLL
IMAADPCM.ACM
MAP_WIN.HLP
MSADPCM.ACM
MSACM.DRV
MSACM.DLL
MSRLE.DRV
INDEOV.DRV
MSVIDC.DRV
IR21_R.DLL
ICCVID.DRV
IR30.DLL
CLEANUP.REG
OLE2.REG
OLE2PROX.DLL
OLE2CONV.DLL
OLE2NLS.DLL
OLE2DISP.DLL
TYPELIB.DLL
STORAGE.DLL
OLE2.DLL

-----------------------------------

CONTROL.INI
REG.DAT

WIN.INI

Changes to WIN.INI
------------------

[drawdib]

[embedding]
MPlayer=Media Clip,Media Clip,mplayer.exe,picture

[mci extensions]
avi=AVIVideo

---------------------

MSACM.imaadpcm=imaadpcm.acm
