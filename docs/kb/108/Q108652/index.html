Q108652: Flight Simulator 5.0: Undocumented Autopilot Quick Key, CTRL+Z

Article: Q108652
Product(s): Microsoft Home Games
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Flight Simulator 5.0 will engage the autopilot and activate Altitude Hold for
the current altitude if you press CTRL+Z.

This feature is not documented in the manual.

Additional query words: 5.00 flightsim fltsim dos fs5 control z control+z
