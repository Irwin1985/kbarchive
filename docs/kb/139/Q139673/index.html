Q139673: SNA Manuals for SNA Version 2.11 Reference SNA 2.10

Article: Q139673
Product(s): Microsoft SNA Server
Version(s): WINDOWS:2.11
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft SNA Server, version 2.11, on platform(s):
   - the operating system: Microsoft Windows NT 
-------------------------------------------------------------------------------

SUMMARY
=======

The set of manuals that ship with Microsoft SNA Server version 2.11 includes
five manuals written for version 2.1. The SNA server 2.11 Enhancements Guide
describes the new features of 2.11 as well as changes to the 2.1 manuals which
were not updated for the 2.1 release.

MORE INFORMATION
================

The following lists indicate which documents are 2.1 or 2.11.

2.11:
Enhancements Guide For SNA Server Version 2.11 SY62906-0595.
Comprehensive Index SY62909-0595.
TN3270 Service SY62908-5/95.
Planing Guide SY62907-4/95.

2.1:
Managing Microsoft SNA Server SY59564-09/94.
3270/5250 Applet Users Guide SY57267-09/94.
Installation Guide SY65689-05/95.
Reference SY57264-09/94
Administration Guide SY57265-09/94.

Also included in the package is the Companion Product Catalog For SNA Server
Update 18 SY63646-0595 NOTE: The update 20 version of the Companion Product
Catalog in on the SNA Server 2.11 CD at \collatrl\general\isvcatal.doc.

The SNA server version 2.11 CD-ROM includes the Software Developers Kit and the
APIs. The part number for the hard copy SDK documentation is 211-074-150. The
SNA Server 2.1 API Programmer's Guides has not changed for 2.11.
