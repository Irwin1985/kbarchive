Q139763: Julia Child: Cannot Minimize Program When Playing a Lesson

Article: Q139763
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play a video file from the Julia Child Lessons, if you click the
Minimize button, the video stops, but the program fails to minimize.


RESOLUTION
==========

Click the minimize button again (after the lesson stops playing) and the program
will minimize.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Julia Child: Home
Cooking with Master Chefs for Windows, version 1.0. We are researching this
problem and will post new information here in the Microsoft Knowledge Base as it
becomes available.

Additional query words: multi media multimedia multi-media mmtitles kbmm .avi audio video interleaved
