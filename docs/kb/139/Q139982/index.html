Q139982: Windows NT 3.5 & 3.51 Supported Network Card Drivers Location

Article: Q139982
Product(s): Microsoft Windows NT
Version(s): 3.50 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 3.5, 3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

Many of the network cards on the hardware compatibility list require drivers
from the Windows NT compact disc \DRVLIB\NETCARD\X86 directory to function under
Windows NT. You can use this Article as an index to help find the proper
subdirectory for a particular driver.
