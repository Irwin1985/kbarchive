Q139540: Windows for Workgroups Error: SMB Error (17) Working…

Article: Q139540
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.11
Operating System(s): 
Keyword(s): 
Last Modified: 22-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

When you run Windows for Workgroups 3.11 with the IPX/SPX Compatible
Transport, the following error message appears when you run applications:

  SMB Error (17) Working...

This error occurs when you use Windows for Workgroups Direct Hosting over
IPX in a Windows NT environment. This error is an indication the
application on the server side has received the request and is in the act
of processing it. This is a normal occurrence and is not an indication of a
problem.


Additional query words: directhosting 3.50 3.51 wfw wfwg
