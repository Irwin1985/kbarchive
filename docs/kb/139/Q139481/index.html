Q139481: ADT95: MS Access & ADT Features Checklist Available

Article: Q139481
Product(s): Microsoft Access Distribution Kit
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbfile kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Access Developer's Toolkit, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Newfeat.exe file contains a two-page checklist of features in Microsoft
Access for Windows 95 version 7.0 and the Microsoft Access Developer's Toolkit
for Windows 95 version 7.0.

The following file is available for download from the Microsoft Download Center:

  Download Newfeat.exe now
  (http://download.microsoft.com/download/access95/paper2/1/WIN98/EN-US/Newfeat.exe)

The Newfeat.exe file contains the following files:

  Newfeat.doc     Microsoft Access for Windows 95 and Microsoft Access
                  Developer's Toolkit for Windows 95 Feature
                  Checklist

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Additional query words: Newfeat exe doc
