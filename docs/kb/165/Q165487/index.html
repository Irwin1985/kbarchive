Q165487: Incorrect or Garbled Text on Tab Labels in Properties

Article: Q165487
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): kbfile win95 kbgraphxlinkcriticalkbfixlist
Last Modified: 03-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Windows 95 OEM Service Release, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may see incorrect or garbled text on tab labels when you are looking at an
item's properties.


STATUS
======

This issue is resolved by the following updated file for Windows 95:

  Comctl32.dll version 4.70 (dated 5/19/97) and later

For information about about obtaining the latest version of the Comctl32.dll
file, please click the article number below to view the article in the Microsoft
Knowledge Base:

  Q186176 INFO: Redistribution of Comctl32.dll

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  Download com32upd.exe now
  (http://download.microsoft.com/download/win95upg/update/1/w95/EN-US/com32upd.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Microsoft does not allow the separate redistribution of the Comctl32.dll file at
this time. Note that Microsoft does allow the redistribution of the complete
self-extracting archive file Com32upd.exe.

As an alternative, you should have your end users obtain the most current version
of this file from Microsoft Internet Explorer on the Microsoft Web site. Or, you
may have redistribution rights for this file under the terms of the Microsoft
Internet Explorer EULA or redistribution agreement. Please review the
information regarding the use and redistribution rights for Microsoft Internet
Explorer located at

  http://www.microsoft.com/ie/

and

  http://www.microsoft.com/ie/ieak/

Should you have questions regarding the licensing of Internet Explorer, please
contact ieak@microsoft.com <mailto:ieak@microsoft.com>.
