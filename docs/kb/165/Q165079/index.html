Q165079: WD97: "From Scanner" Missing from Insert Picture Menu

Article: Q165079
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbui kbusage winword word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

"From Scanner" is missing from the Insert Picture menu.

CAUSE
=====

Microsoft Photo Editor is not installed.

WORKAROUND
==========

To install Microsoft Photo Editor, use the following steps:

1. Quit Word and all Office applications, including the Microsoft Office
  Shortcut Bar.

2. On the Windows Start menu, point to Settings, and then click Control Panel.

3. Double-click Add/Remove Programs.

4. Click the Install/Uninstall tab.

5. Click "Microsoft Office 97" ("Microsoft Word 97"), and then click Add/Remove.
  The Microsoft Office 97 (Word 97) setup program will start.

6. In the Microsoft Office 97 Setup (Microsoft Word 97 Setup) dialog box, click
  Add/Remove.

7. In the Microsoft Office 97 - Maintenance (Microsoft Word 97 - Maintenance)
  dialog box, click Office Tools. Do not clear the check box.

8. Click Change Option. Click Microsoft Photo Editor to select the check box,
  and then click OK.

9. Click Continue to complete the setup.

MORE INFORMATION
================

Microsoft Photo Editor is not installed during a typical installation.

Other graphics programs with scanner capabilities may also place a menu command
in Word. If you have problems accessing your scanner from Word, please check the
documentation that comes with your software to correct the problem.

For additional information, please see the following articles in the Microsoft
Knowledge Base:

  Q93359 Word for Windows Does Not Support TWAIN Protocol

  Q163458 Err Msg: "Unable to Open Selected TWAIN Source" in Word 97
