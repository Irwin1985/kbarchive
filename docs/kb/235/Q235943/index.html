Q235943: Doc Err: Updated Version of Content Analyzer Is Available

Article: Q235943
Product(s): Internet Information Server
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): 
Last Modified: 05-AUG-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Internet Information Server 4.0 
- Microsoft Windows NT version 4.0 Option Pack 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Readme.sp4 file that is installed with Windows NT 4.0 Service Pack 4,
located in the %SystemRoot%\System32 directory, incorrectly states that the
updated Site Server 3.0 Express download will contain an updated version of the
Content Analyzer component.

The Site Server 3.0 Express download is now available, but it does NOT contain an
updated version of Content Analyzer.

MORE INFORMATION
================

The following text from the Readme.sp4 file, located under the 2.4.2 Site Server
Express 3.0 section, is incorrect:

  An updated version of the Content Analyzer will be available for download
  from the Windows NT Service Pack 4.0 Web site.

The updated version of Site Server 3.0 Express is located at the following
address:

  http://www.microsoft.com/NTServer/nts/downloads/management/siteserver/
