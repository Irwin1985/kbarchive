Q122565: FORTRAN PowerStation 32 Update for Windows NT 3.5

Article: Q122565
Product(s): Microsoft Fortran Compiler
Version(s): 1.0
Operating System(s): 
Keyword(s): kbfile
Last Modified: 18-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Fortran Powerstation 32 for Windows NT, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The linker (version 1.0, dated 23-Jun-1994) included with Microsoft FORTRAN
PowerStation 32 interacts incorrectly with Microsoft Windows NT version 3.5, but
works fine with Windows NT version 3.1.

Microsoft has confirmed this to be a problem with the version of LINK.EXE
(version 1.0, dated 23-Jun-1994) that shipped with Microsoft FORTRAN
PowerStation 32 version 1.0 for Windows NT. To correct this problem, please
obtain the new version of LINK.EXE (version 1.0, dated 9-Aug-1994) that corrects
the problem and replace the old LINK.EXE.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  fpsntu01.exe
  (http://download.microsoft.com/download/fortranps10bnt/Update1/1/N351/EN-US/fpsntu01.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
