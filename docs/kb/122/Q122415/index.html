Q122415: Updated EMM386.EXE and MSD.EXE Available for MS-DOS 6.0, 6.2

Article: Q122415
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:6.0,6.2,6.21
Operating System(s): 
Keyword(s): kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21 
-------------------------------------------------------------------------------

SUMMARY
=======

More information on this topic may be available from Microsoft Product Support
Services (PSS).

Microsoft Product Support Services are available on the World Wide Web by
connecting to http://www.microsoft.com/. The FTP site is located at
ftp.microsoft.com.

MORE INFORMATION
================

EMM386.EXE version 4.49 ships with MS-DOS 6.22. This version of EMM386.EXE
provides support for PCI cards. MSD.EXE version 2.11 should recognize the
Pentium processor. MSD.EXE 2.11 shipped with MS-DOS 6.21 and is currently
included in the MS-DOS 6.22 Upgrade and Step-Up packages.

EMM622.EXE has been made available for customers with MS-DOS 6.0 or later who
want to use EMM386.EXE 4.49 and MSD.EXE 2.11 but do not want to upgrade their
current MS-DOS setup to MS-DOS 6.22. Only the English language versions of these
files are available in EMM622.EXE.

The following file is available for download from the Microsoft Download Center:

  DownloadDownload Emm622.exe now

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

EMM622.EXE contains the following files:

  EMM386.EX#
  EMM386.EX^
  MSD.EXE
  MSUPDATE.EXE
  UPDATE.BAT

After you run EMM622.EXE and the above files are extracted, run UPDATE.BAT, which
upgrades EMM386.EXE to version 4.49 and MSD.EXE to version 2.11.

Additional query words: 6.00 4.49 2.11
