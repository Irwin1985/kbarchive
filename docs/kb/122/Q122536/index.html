Q122536: Arcade for Macintosh: How to Disable PowerBook Processor Cycling

Article: Q122536
Product(s): Microsoft Home Games
Version(s): MACINTOSH:1.0
Operating System(s): 
Keyword(s): kbenv kbtool kbimu msgamekbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Arcade for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Some models of the Apple Macintosh PowerBook computer include a Processor
Cycling feature that reduces the speed of the processor and eventually turns it
off when the processor is inactive for a certain period of time. You may want to
disable this feature when you play Arcade.

This article describes how to disable the Processor Cycling feature.

MORE INFORMATION
================

To disable the Processor Cycling feature on a PowerBook computer, use the
appropriate steps for your version of Mac/OS.

Mac/OS System 7.01
------------------

To disable the Processor Cycling feature on a System 7.01-based PowerBook
computer, follow these steps:

1. On the Apple menu, point to Control Panels, and then click Portable.

2. Press and hold down the OPTION key as you click Minutes Until Automatic
  Sleep.

3. Click Don't Rest.

Mac/OS System 7.1
-----------------

To disable the Processor Cycling feature on a System 7.1-based PowerBook
computer, follow these steps:

1. On the Apple menu, point to Control Panels, and then click PowerBook.

2. Press and hold down the OPTION key as you click Options.

3. Under Processor Cycling, click Don't Allow Cycling.

When you finish playing Arcade, remember to enable the Processor Cycling feature.
