Q122327: FoxPro Advisor Magazine Articles February 1994

Article: Q122327
Product(s): Microsoft FoxPro
Version(s): 2.5x 2.60 2.60a | 2.00 2.5x 2.

Keyword(s): kb3rdparty
Last Modified: 26-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a 
- Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, 2.5b, 2.6, 2.6a 
- Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c, 2.6a 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is the table of contents of the February 1994 issue of the "FoxPro
Advisor." This publication is published by Data Based Solutions, Inc. 4010
Morena Blvd., Suite 200, San Diego, CA 92117 (619) 483-6400. For phone orders
only, call: 800-336-6060

MORE INFORMATION
================

Developer View, page 8. Power Screen Generation with GENSCRNX. By Alan Schwartz.
GENSCREN.PRG, Screen Painter, enhanced, 3-D effects, chiseled and beveled,
_GENSCRN, Ken Levy.

Functional FoxPro, page 14. Index Functions. By Malcolm C. Rubel. KEY(), ORDER(),
SET INDEX TO, SET ORDER TO, TAG(), production.cdx. Discussion of index use,
procedure, and technique.

FoxPro Fundamentals, page 18. Controlling and Integrating BROWSE. By Miriam
Liskin. Cover title: It Can Be Done: Integrating BROWSE and READ. BROWSE command
clauses, ON KEY LABEL, :W WHEN, :V VALID,

Graphic FoxPro, page 28. Using Microsoft Graph with FoxPro for Windows. By. Ted
Roche. Cover title: Adding Graphs to FoxPro for Windows. Gengraph, GENGRAPH.APP,
Graph Wizard, General field type, OLE, MS-GRAPH, RQBE output, UPDATEGRPH,
REFRESHGRPH, @ SAY VERB, ISOMETRIC, Scale Picture.

Developer Toolbox, page 39. Adding an Application Activity Log. By Victor M.
Font, Jr. UDF, User Defined Function, TIME().

Advanced FoxPro, page 40. Inside the .CDX file. By Victor M. Font, Jr. Revealed:
Getting inside FoxPro Indexes. B-tree structure, index tags, DISPLAY STATUS,
index header, pointer to root node, exterior root node, interior root node.

FoxPro Tip, page 46. FoxFont and the FoxPro for Windows Distribution Kit. By Drew
Speedie. FOXFONT.FON, screen window display.

Q&A, page 50. FoxPro Solutions. By Steve Freides and Tamar E. Granor. ALIAS,
Group Breaks, macros, EVAL() performance speed, indirect referencing, REPLACE
scope, EOF(), REPORT FORM, UDF to advance to next page in middle of detail.
