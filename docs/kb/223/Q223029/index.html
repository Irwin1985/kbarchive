Q223029: Compaq Prosignia Server Hangs With Windows NT 4.0 Service Pack 4

Article: Q223029
Product(s): Microsoft Windows NT
Version(s): winnt:4.0,4.0 SP4
Operating System(s): 
Keyword(s): 
Last Modified: 09-APR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server versions 4.0, 4.0 SP4 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Windows NT 4.0 Service Pack 4 (SP4), the Compaq Prosignia 500
server may stop responding (hang) at the blue build screen.

After you perform the Emergency Repair Disk procedure and reapply Windows NT 4.0
Service Pack 3 (SP3), the server restarts normally. However, after you apply
Service Pack 4 again, the server hangs at the blue build screen.

CAUSE
=====

This issue occurs because the Compaq Prosignia 500 requires a firmware upgrade
from Compaq before you install Windows NT 4.0 SP4.

RESOLUTION
==========

To resolve this issue, contact Compaq Technical Support for the latest firmware
and Softpaks for the Compaq Prosignia 500.
