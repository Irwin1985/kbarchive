Q155959: WD97: Envelope Feed Method Displayed Incorrectly (HP DeskJet)

Article: Q155959
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kb3rdparty kbprint kbualink97 word97 kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use Envelopes and Labels on the Tools menu, the Envelopes tab may show
an incorrect envelope feed method.

CAUSE
=====

This problem has been reported to occur with the following Hewlett-Packard (HP)
DeskJet printer drivers: 500, 550c, 510, 560, 660c, 672c, 692c, 693c, 820cse,
855, 870csi, and 1200c.


WORKAROUND
==========

Manually select the proper orientation for the envelope feed method. To do this,
follow these steps:

1. On the Tools menu, click Envelopes and Labels.

2. Select the Envelopes tab, and click Options.

3. Select the Printing Options tab.

4. Select the proper feed method for the DeskJet printer you have, and click OK.

It may be necessary to change the clockwise rotation to on or off, depending on
the printer and printer driver version you are using.

Make sure you have the most current version of the printer driver installed.



Additional query words: position angle upsidedown wrong direction hp hpdj dj500 dj550c dj510 dj560 dj1200c
