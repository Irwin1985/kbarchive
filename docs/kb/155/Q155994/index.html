Q155994: NBA Full Court Press: Minimum System Requirements

Article: Q155994
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 08-NOV-2001

1.0
WINDOWS
kbsetup kbref kbusage kbfaq

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft NBA Full Court Press for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article lists the minimum system requirements to run Microsoft NBA
Full Court Press.

MORE INFORMATION
================

The minimum system requirements for NBA Full Court Press are:

- Multimedia PC with a 486/66 MHz or higher processor; Pentium 75 or higher
  recommended

- Microsoft Windows 95 operating system

- 8 MB of memory (RAM); 16 MB recommended

- 20 MB of available hard-disk space

- Double speed CD-ROM drive

- Super VGA monitor

- Audio board

- Microsoft Mouse or compatible pointing device; game pad or joystick
  recommended

Optional:

- Hayes-compatible modem required for two-player competition via modem

- Local area network required for network play
