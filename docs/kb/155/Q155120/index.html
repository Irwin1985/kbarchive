Q155120: Error Message: Error Copying File

Article: Q155120
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kbhw kbHardware
Last Modified: 13-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you are using Windows NT Explorer to copy a file from a CD Interactive
(CD-i) CD-ROM to the hard disk, you may receive the following error message:

  Error Copying File

  Cannot copy AVSEQ02: Invalid MS-DOS function.

CAUSE
=====

Windows NT 4.0 does not fully support the CD-i format.

STATUS
======

This behavior is by design.
