Q228522: Baseball 2000: How to Improve Graphics Performance

Article: Q228522
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbdisplay kbui kbimu msgame
Last Modified: 17-APR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Baseball 2000, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists some methods that you can use to improve graphics performance
in Microsoft Baseball 2000.

MORE INFORMATION
================

To improve graphics performance in Baseball 2000, use one or more of the
following methods. Microsoft recommends that you experiment with different
combinations of these methods until you achieve an acceptable level of graphics
performance.

Disable Gouraud Shading
-----------------------

To disable Gouraud Shading, follow these steps:

1. Start Baseball 2000.

2. On the Home Plate screen, click Options.

3. Click the Graphics tab.

4. Click to set the Gouraud Shading option to Off, and then click OK.

Disable Shadows
---------------

To disable Shadows, follow these steps:

1. Start Baseball 2000.

2. On the Home Plate screen, click Options.

3. Click the Graphics tab.

4. Click to set the Shadows option to Off, and then click OK.

Reduce the Level of Player Detail
---------------------------------

To reduce the level of player detail, follow these steps:

1. Start Baseball 2000.

2. On the Home Plate screen, click Options.

3. Click the Graphics tab.

4. Under Player Detail, click to select Medium or Low.

5. Click OK.

Reduce the Display Resolution
-----------------------------

To reduce the display resolution, follow these steps:

1. Start Baseball 2000.

2. On the Home Plate screen, click Options.

3. Click the Graphics tab.

4. Under Resolutions Available, click to select a lower display resolution than
  the display resolution that is currently selected.

5. Click OK.

Disable Sounds in the Game
--------------------------

To disable sounds in the game, follow these steps:

1. Start Baseball 2000.

2. On the Home Plate screen, click Options.

3. Click the Sounds tab.

4. Click to set each sound option to Off, and then click OK.
