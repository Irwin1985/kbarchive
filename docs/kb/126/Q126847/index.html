Q126847: How to Change the Title for an MS-DOS-Based Application

Article: Q126847
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95 appscomp kbAppCompatibility
Last Modified: 28-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

When you view an MS-DOS-based application's properties, the first line on the
Program tab specifies the following items:

- The title bar text of the application's window.

- The title of the button on the taskbar for the application.

The entry on this line does not specify the filename or a "friendly name" for the
application.

MORE INFORMATION
================

To view the property sheet for an MS-DOS-based application, follow these steps:

1. Double-click My Computer.

2. Double-click the folder that contains the MS-DOS-based application.

3. Use the right mouse button to click the application's icon and then click
  Properties on the menu that appears.
