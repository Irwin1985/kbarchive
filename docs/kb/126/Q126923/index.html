Q126923: PC DOS: How to Use CONVERT.EXE

Article: Q126923
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:3.0,3.2,3.2a
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for PC Networks, versions 3.0, 3.2, 3.2a 
-------------------------------------------------------------------------------

SUMMARY
=======

CONVERT.EXE, included with versions 3.0, 3.2, and 3.2a of Microsoft Mail for PC
Networks, is a utility that will convert the folders from Microsoft Mail for PC
Networks, MS-DOS workstation to Microsoft Mail for Windows folders.

By default, CONVERT.EXE is placed in the MAILEXE directory.

MORE INFORMATION
================

Steps to use CONVERT.EXE
------------------------

1. Make sure you are connected to the postoffice server.

2. Exit & Sign Out from Mail.

3. From File Manager, run CONVERT.EXE. (It is a Windows application.)

4. You will receive a status report from the utility. At the end of the
  conversion, you will be prompted as to whether or not you want to delete the
  old folders.

For additional information, please see the Microsoft Mail for PC Networks
"Administrator's Guide," page 334.
