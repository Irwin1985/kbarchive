Q67417: PIF Files Created by Windows 3.0 Setup from P to R

Article: Q67417
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 17-OCT-1999

3.00 3.00a
WINDOWS
kbsetup kbref

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the default parameters for program information files (PIFs)
created by the Microsoft Windows version 3.0 Setup program, in alphabetical
order from P to R. These defaults may be modified by loading the PIF file into
the Windows 3.0 PIF Editor, PIFEDIT.EXE.

MORE INFORMATION
================
