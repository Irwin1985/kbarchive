Q67132: Windows 3.0: Western Digital Controllers - Quick Ref. Card

Article: Q67132
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 31-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The following table is a list of Western Digital storage hard disk controllers.
This information comes from the "Western Digital Storage Controllers Quick
Reference Card."
