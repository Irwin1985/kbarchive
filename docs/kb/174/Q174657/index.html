Q174657: Expedia Streets 98: How to Give Map Feedback

Article: Q174657
Product(s): Microsoft Automap
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbfile kbimukbfaq
Last Modified: 21-DEC-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Expedia Streets 98 
- Microsoft Expedia Streets 98 Deluxe, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain and use the Feedback Wizard reporting tool
for Streets 98. This tool allows you to easily submit map feedback to the
Microsoft Expedia Streets team.

Microsoft developed the Feedback Wizard as an easy-to-use, structured way to send
feedback about Expedia Streets 98 maps. The wizard takes you through the steps
of identifying a feature on the map, specifying the type of feedback, and
sending in comments. You can also use the Feedback Wizard to point out features
that are missing on the map.

MORE INFORMATION
================

You can obtain the Expedia Streets 98 Feedback Wizard from the Microsoft
Download Center. For more information about obtaining the file, see the section
"Obtaining the Wizard" after the disclaimer section.

Obtaining the Wizard
--------------------

The following file is available for download from the Microsoft Download Center:

  St98fdwz.exe
  (http://download.microsoft.com/download/expediastreets98/wizard/1/WIN98/EN-US/St98fdwz.exe)

Release Date: Oct-02-1997

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Installing the File
-------------------

To install the Feedback Wizard, do the following:

1. Download the file, St98fdwz.exe.

2. Double-click the self-extracting file. This will start the installation of
  the files required to install the Feedback Wizard.

NOTE: The Feedback Wizard works only with the retail version of Microsoft Streets
98. It will not work with any OEM (Original Equipment Manufacturer) version of
Microsoft Expedia Streets 98 that may have come pre-installed when you purchased
your computer. You should exit all Windows programs before running this Setup
program.

To remove the Feedback Wizard, use the Add/Remove Programs icon found in the
Windows Control Panel. The Feedback Wizard is listed as "Microsoft Expedia
Streets 98 Feedback Wizard" in the list of installed components. Removing the
Feedback Wizard will disable the Feedback Wizard and will not remove any of the
files needed to run Expedia Streets 98.

Using the Wizard
----------------

To start using the Feedback Wizard, follow these instructions:

1. Open Streets 98.

2. Click Help, and then click Feedback On The Map.

This starts the Feedback Wizard. Follow the instructions to complete the report.

E-mail Issues
-------------

If you have Internet e-mail working with a 32-bit e-mail program that is
compatible with the Messaging Application Programming Interface (MAPI) (such as
Microsoft Exchange or Outlook), do the following:

- On the last panel of the wizard, click E-mail Your Feedback, and then click
  Finish.

If you are not sure your e-mail is MAPI compliant, follow these instructions:

1. On the last panel of the wizard, clear the check mark from the E-mail Your
  Feedback option, and then click Finish.

2. When you are prompted, save the file.

3. Mail the saved feedback as an attachment to:

  gmmer@microsoft.com

  If you do not have e-mail, print your feedback and send the report to:

  Map Editor, Trip Planner 98
  c/o Microsoft Corporation
  One Microsoft Way
  Redmond, WA 98052-6399
  FAX: (425) 936-7329 (No technical support is available at this number)
