Q174116: The Microsoft Network Does Not Run in Windows NT

Article: Q174116
Product(s): The Microsoft Network
Version(s): WINDOWS:1.0,1.05,1.2,1.3,2.0,2.5,2.6
Operating System(s): 
Keyword(s): 
Last Modified: 08-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0, 2.5, 2.6 
-------------------------------------------------------------------------------

SYMPTOMS
========

MSN, The Microsoft Network, cannot be installed or run on a computer running
Microsoft Windows NT.

CAUSE
=====

MSN is designed to run in Microsoft Windows 95/98.

MORE INFORMATION
================

Compatibility with Windows NT is under review and is being considered for a
future release of MSN.

For information on MSN's technical beta program for Windows NT users, connect to
the following Microsoft Web site:

  http://fdl.msn.com/public/setup/msn25nt4/default.htm

Peer-to-peer newsgroup support for the MSN beta program for Windows NT4 is
available at the following location:

  news://msnnews.msn.com/msn.beta.nt

Additional query words: msn
