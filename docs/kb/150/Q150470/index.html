Q150470: Mac Bookshelf 1996-97: Quickshelf Does not Look up Word

Article: Q150470
Product(s): Microsoft Home Multimedia Titles
Version(s): 1995 edition,1996 edition,1996-1997,1997 edition; MACINTOSH:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 18-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf version 1996-1997 for the Macintosh 
- Microsoft Cinemania for the Macintosh versions 1996 edition, 1997 edition 
- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
- Microsoft Encarta 96 Encyclopedia for Macintosh 
- Microsoft Encarta 97 Encyclopedia for Macintosh 
- Microsoft Music Central for the Macintosh, 1996, 1997 editions 
- Microsoft Works for the Macintosh, versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are in one of the following programs and try to look up a word using
the QuickShelf menu, Bookshelf 1996-97 loads, but the word is not found.

- Microsoft Cinemania, 1996, 1997 editions

- Microsoft Encarta, 1995 edition

- Microsoft Encarta Encyclopedia 1996, 1997 editions

- Microsoft Music Central, 1996, 1997 editions

WORKAROUND
==========

When Bookshelf 1996-97 is open on the screen, you can find the word manually. To
find the word manually, do the following:

1. On the Tools menu, click Find.

2. Type the word you want to find, and then click Find.

MORE INFORMATION
================

When you search for a word in Microsoft Works 3.0, the QuickShelf menu does not
load Bookshelf 1996-97. You need to start Bookshelf and then use the steps in
the "WORKAROUND" section to find your text.
