SUMMARY
=======

Curriculum activities and Teacher Activity Guides for several Microsoft Home
products are available on the Internet's World Wide Web. There are also Academic
versions of these programs that include printed curriculum materials.

To access these educational tools, do the following:

1. Visit the following Microsoft Web site:

  http://www.microsoft.com

2. Search for "K-12 Education".

3. Click Resources.

4. Scroll to the Curriculum Activities for K-12, select the product that you
  want from the list, and then click Go.

-or-

Visit the following Microsoft Web site to download current information:

  http://www.microsoft.com/education/schools/default.asp

MORE INFORMATION
================

Each program contains one or more of the following areas:
Product Ideas - Try a few of these product-specific integration ideas
               in your classroom.

Teacher's Activity Guide - Includes detailed classroom activities
designed to encourage product exploration and classroom integration.

The Academic Editions of the following applications include a printed
Teacher's Activity Guide or other curriculum materials:

Ancient Lands                  (Activity Poster)
