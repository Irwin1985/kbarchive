Q150966: Macintosh Bookshelf ‘96-‘97: Contents of Each Book

Last Modified: 05-MAY-1999

'96-'97
MACINTOSH

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf 1996-97 for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

Bookshelf contains large numbers of reference entries in a variety of media
formats. This article lists the total number of articles in each book and the
types and quantity of media elements used.

To find additional information on each book, do the following:

1. Start Bookshelf

2. Select a book from the drop down list above the Contents panel.

3. On the Help menu (the question mark in the upper-right corner of the menu
  bar) click About <name of selected book>. Where <name of selected
  book>, is the book you selected in Step 2.
