Q150550: HOWTO: Obtain Windows 3.1/Windows 95 Btrieve ODBC Interface

Article: Q150550
Product(s): Microsoft FoxPro
Version(s): WINDOWS:2.5,3.0,3.0b,5.0,6.0
Operating System(s): 
Keyword(s): kb3rdparty kbinterop kbvfp300 kbvfp500 kbvfp600 kbMDAC250
Last Modified: 03-AUG-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 6.0 
- Microsoft Data Access Components version 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to get the Btrieve ODBC Interface for Windows 3.1,
and the Btrieve ODBC Interface for Windows 95.

MORE INFORMATION
================

NOTE: Btrieve ODBC Interface for Windows 3.1, Btrieve ODBC Interface for Windows
95, and Btrieve 6.15 are manufactured by Btrieve Technologies, Inc., a vendor
independent of Microsoft; we make no warranty, implied or otherwise, regarding
these products' performance or reliability.

The Btrieve ODBC Interface for Windows 3.1 is currently available from Btrieve
Technologies. It is packaged with the following products:

- Btrieve Engine for Windows 3.1 - Single Workstation License
- Scalable SQL for Windows 3.1 - Single Workstation License
- ODBC Interface Commercial Deployment Kit for Windows 3.1/Windows 95

The Btrieve ODBC Interface for Windows 95 is available from Btrieve Technologies.
It is packaged with the following products:

- Btrieve Engine for Windows NT/Windows 95 - Single Workstation License
- ODBC Interface Commercial Deployment Kit for Windows 3.1/Windows 95

To run client/server applications taking advantage of Btrieve database engines
and the Btrieve ODBC Interface, use the Btrieve server engine for Windows NT or
NetWare, version 6.15 or higher.

To receive further information, or to order the Btrieve ODBC Interface for
Windows 3.1 or the Btrieve ODBC Interface for Windows 95, please contact Btrieve
Technologies at 800-BTRIEVE (800-287-4383), or 512-794-1719.
