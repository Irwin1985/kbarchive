Q150411: Macintosh Bookshelf 1996-97: Manual Installation Instructions

Article: Q150411
Product(s): Microsoft Home Multimedia Titles
Version(s): MACINTOSH:
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf 1996-97 for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article describes how to manually install Microsoft Bookshelf
1996-97 for the Macintosh to your hard drive.

MORE INFORMATION
================

Manually Installing Bookshelf
-----------------------------

To manually install Bookshelf, do the following:

1. Create a folder on your hard drive and name that folder Bookshelf 1996-97.

2. Insert the Bookshelf 1996-97 compact disc into the CD-ROM drive.

3. Copy the following items from the Bookshelf 1996-97 disc to the folder you
  created in step 1:

  Bookshelf 1996-97 Read Me
  Bookshelf Guide
  Microsoft Multimedia Catalog
  Microsoft Bookshelf (PPC)

  -or-

  Microsoft Bookshelf (68k)

  NOTE: PPC refers to the Power Macintosh version of Bookshelf. Copy this if you
  are using a Power Macintosh. 68k refers to non-Power Macintosh (680x0)
  computers. Copy this if you are using a non-Power Macintosh computer.

4. Locate and open the Installer Files folder on the compact disc.

5. Copy the following item to the Bookshelf 1996-97 folder that you created in
  step 1:

  SimpleText

6. Copy the following item to your System Folder on the hard drive (when you are
  prompted to place the item into a specific folder, click OK):

  QuickShelf

7. Open the System Files folder (located in the Installer Files folder on the
  compact disc).

8. Copy the following items to your System Folder on the hard drive. (when you
  are prompted to place the item into a specific folder, click OK):

  Apple Guide
  AppleScript
  AppleScriptLib
  QuickTime
  QuickTime Musical Instruments
  QuickTime PowerPlug
  Sound

  NOTE: If your System Software version is earlier than 7.5.3, copy Sound
  Manager, too. If your System Software is earlier than 7.5, then copy 7.0/7.1
  Apple Guide Enabler.

9. Close the System Files folder on the compact disc.

10. Open the Fonts folder (located in the Installer Files folder on the compact
  disc).

11. Copy all the items in the Fonts folder to your System Folder on the hard
  drive. When you are prompted to place items into specific folders, click OK.

12. Close the Fonts Folder on the compact disc.

13. Close the Installer Files folder on the compact disc.

14. Open your System Folder on the hard drive.

15. Click the Preferences folder once.

16. On the File Menu, click Make Alias.

17. Drag this alias icon to the Bookshelf 1996-97 folder that you created in
  step 1.

18. Rename the alias to the following:

  Drop Updates Here.

NOTE: To install Internet Explorer, continue with the next section. Otherwise,
the installation of Bookshelf 1996-97 is finished and you can skip to the
"Completing the Installation" section at the end of this article.

Installing Internet Explorer
----------------------------

1. Locate and open the Bookshelf 1996-97 folder that you created in step 1 of
  the "Manually Installing Bookshelf" section. Then, create a folder inside
  that folder named:

  Internet Explorer.

2. Locate and open the Internet Explorer 2.0 folder on the compact disc.

3. Copy the following items to the Internet Explorer folder that you created in
  step 1 of this section:

  Internet Explorer (PPC)

  -or-

  Internet Explorer (68k)
  Internet Explorer License
  Internet Explorer Read Me
  Help
  Plug-ins

  NOTE: If your System Software version is earlier than 7.5, then also drag the
  Thread Manager to your System Folder on the hard disk. When you are prompted
  to place the item into a specific folder, click OK.

4. Close the Internet Explorer 2.0 folder on the compact disc.

5. Locate and open the Internet Config 1.2 folder on the compact disc.

6. Copy the following item into the Drop Updates Here folder you created in step
  18 of the "Manually Installing Bookshelf" section:

  Internet Preferences

7. Copy the following item to the Bookshelf 1996-97 folder that you created in
  step 1 of this section:

  Internet Config

8. Copy the following item to your System Folder on the hard drive (when you are
  prompted to place the item into a specific folder, click OK):

  Internet Config Extension

9. Follow the instructions in the "Completing the Installation" section at the
  end of this article:

Completing the Installation
---------------------------

1. Close all open windows and restart the computer.

2. Start Bookshelf 1996-97 from the Bookshelf 1996-97 folder that you created in
  step 1 of the Manually Installing Bookshelf section of this article.
