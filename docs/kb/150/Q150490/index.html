Q150490: 3D Movie Maker: Features and Limitations

Article: Q150490
Product(s): Microsoft Home Kids Products
Version(s): 1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 19-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 3D Movie Maker for Windows, version 1.0 
- Microsoft Nickelodeon 3D Movie Maker for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes the features and limitations of 3D Movie Maker. Although
the following list is not exhaustive, it gives you a good idea of the things
that you can and cannot do in 3D Movie Maker.

(Another source for information about features is the 3D Movie Maker Help
Machine.)

This article is broken down into two sections "What you CAN do in 3D Movie
Maker," and "What you CANNOT do in 3D Movie Maker."

WHAT YOU CAN DO IN 3D MOVIE MAKER
---------------------------------

You can get more information on how to perform most of the following activities
by consulting the online Help included with 3D Movie Maker.

Make Movies:

For more information on making movies, see 3D Movie Maker online Help.

Import Wave files:

For information about importing sounds, see the following article in the
Microsoft Knowledge Base:

  Q150487 3D Movie Maker: How to Import Sounds

Record Wave Sounds:

For information about recording sounds, see the following article in the
Microsoft Knowledge Base:

  Q150488 3D Movie Maker: How to Record Sounds and/or Dialog

Import MIDI Files:

For information about importing MIDI files, see the following article in the
Microsoft Knowledge Base:

  Q150487 3D Movie Maker: How to Import Sounds

Remove Sounds from Movies and/or Actors:

  For information about removing sounds, see the following article in the
  Microsoft Knowledge Base:

  Q145825 3D Movie Maker: How to Remove Recorded/Imported...

Move Sounds from One Movie to Another Movie:

For information about moving sounds, see the following article in the Microsoft
Knowledge Base:

  Q143365 3D Movie Maker: How to Move Sounds Between Movies

Change Performance Options:

For information about changing performance options (by changing the quality of
animation and running the program in either full-screen, or window mode), see
the following article in the Microsoft Knowledge Base:

  Q141821 3D Movie Maker: How to Improve Performance

Add Frames to Middle of Movie:

For information about adding frames to the middle of a movie, see the following
article in the Microsoft Knowledge Base:

  Q143369 3D Movie Maker: How to Add Frames While Freezing...

Start Your Search for Talent Book Pages Again:

For information on resetting the talent book, see the following article in the
Microsoft Knowledge Base:

  Q139495 3D Movie Maker: How to Reset the Talent Book

WHAT YOU CANNOT DO IN 3D MOVIE MAKER
------------------------------------

Record MIDI:

3D Movie Maker does not provide a way to record MIDI. To record MIDI, you need
additional hardware and software.

Export Movies to AVI (Videos):

For more information on this, see the following article in the Microsoft
Knowledge Base:

  Q143230 3D Movie Maker: Movies Cannot be Saved in AVI...

Play Movies on Another Computer Without 3D Movie Maker Installed:

3D Movie Maker must be installed to play 3D Movie Maker files. There is no
run-time version of 3D Movie Maker.

Import or Create Actors, Props, or 3D Objects:

3D Movie Maker comes complete with all of the actors, props, 3D objects, and
backgrounds it can use. You cannot import other images (for example, .bmp, .wmf
files) into your movies.

Remove Frames from Middle of Movie:

You cannot remove frames from within a movie with 3D Movie Maker. In other words,
you cannot shorten a movie from 100 frames down to 90 frames.

Keep Action Going While Scroll Box Is Scrolling:

For more information on this, see the following article in the Microsoft
Knowledge Base:

  Q142108 3D Movie Maker: Animation Stops While Credits...
