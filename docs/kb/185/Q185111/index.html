Q185111: FP: Error Publishing FrontPage Web Using Web Publishing Wizard

Article: Q185111
Product(s): Word Front Page
Version(s): WINDOWS:1.1,97
Operating System(s): 
Keyword(s): kbdta
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FrontPage 97 for Windows with Bonus Pack 
- Microsoft FrontPage for Windows 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Web Publishing Wizard version 1.0 or 1.1 to publish your web,
you receive the following error:

  Web Publishing was unable to post your file(s) to your web site.

When you use the Web Publishing Wizard version 1.5 to publish your web, you
receive the following error:

  The Web Publishing Wizard was unable to publish your file(s) to the Web
  server for the following reasons:

  The file upload has been cancelled.

  Click OK to exit.

CAUSE
=====

The Web Publishing Wizard cannot publish to a web server on a port other than
port 80.
