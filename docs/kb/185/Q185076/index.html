Q185076: XGEN: How to Obtain Exchange Server 4.0 Service Pack 5

Article: Q185076
Product(s): Microsoft Exchange
Version(s): 4.0
Operating System(s): 
Keyword(s): kbgraphxlinkcritical
Last Modified: 27-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

VERY IMPORTANT: Please note that while Service Pack 5 (SP5) contains the
fixes included in SP3 and SP4, it does not contain the fixes included in
SP2. You MUST install SP2 before installing SP5.

SUMMARY
=======

** THIS U.S. SERVICE PACK IS FOR MICROSOFT EXCHANGE SERVER 4.0 ONLY!! **

To find out which bugs are fixed by this update and how to install this Service
Pack, see the Readme.txt file that accompanies the Service Pack or query on the
following word in the Microsoft Knowledge Base (without the spaces):

  S E R V P A C K

Available Formats for Microsoft Exchange 4.0 Service Pack 5
-----------------------------------------------------------

Service Pack 5 is available as single self-extracting archive files for Intel
(x86) and ALPHA based computers.

Ordering Microsoft Exchange 4.0 Service Pack 5
----------------------------------------------

If you do not have access to the electronic services listed below, you can order
Service Pack 5 (CD-ROM or Disk Set) from the Microsoft Sales Information Center
(MSIC) at (800) 426-9400.

Instructions for Downloading
----------------------------

The following files are available for download from the Microsoft Download
Center:

Microsoft Exchange Service Pack 5 Server Files:

  DownloadDownload Sp5_400a.exe now
  (http://download.microsoft.com/download/exch40/exsp_dec/3/ALPHA/EN-US/SP5_400A.EXE)

  DownloadDownload Sp5_400i.exe now
  (http://download.microsoft.com/download/exch40/exsp_int/4/NT4/EN-US/SP5_400i.EXE)

Microsoft Exchange Service Pack 5 Server Support Files:

  DownloadDownload Sp5s400a.exe now
  (http://download.microsoft.com/download/exch40/exsp_dec/2/ALPHA/EN-US/SP5S400a.EXE)

  DownloadDownload Sp5s400i.exe now
  (http://download.microsoft.com/download/exch40/exsp_int/2/NT4/EN-US/SP5S400I.EXE)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
