Q185028: XGEN: WWW Location of Microsoft Exchange Server Resource Kit

Article: Q185028
Product(s): Microsoft Exchange
Version(s): winnt:5.5
Operating System(s): 
Keyword(s): exc55
Last Modified: 12-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Exchange Server Resource Kit is part of the Microsoft BackOffice
Resource Kit, Part Two, for Microsoft Exchange Server 5.5 and is available to
order on the World Wide Web at the following URL:

  http://mspress.microsoft.com/books/1264.htm

MORE INFORMATION
================

The Microsoft Exchange Server Resource Kit contains the following tools, which
are available for download from Microsoft's World Wide Web server:

- Administrative Mailbox Agent

- ASpiriN Tool

- AT&T Mail Migration

- AutoFill Tool

- AutoSig Tool

- Change Route Tool

- CleanSweep

- Crystal Reports V5.0

- Distribution List Import Tool

- On-line Documentation

- Buddy Dropper Tool

- Duplicate Proxy Search Tool

- EFD Handcode Agent

- Event Log Tools

- Microsoft Exchange Messages Reference

- Microsoft Mail Export/Import Tool

- Global Send Options Tool

- Import Header Tool

- Internet Mail Connector Extension

- Internet Idioms Tool

- Mailstorm Tool

- MAPI Notes Document Link Tool

- MAPIsend Tool

- Mailbox Cleanup Agent V1.6

- Mailbox Migration Tool

- Management Chain List Tool

- SMTP Gateway Proxy Migration Tool

- Microsoft Exchange Server Modeling Tool

- Microsoft Mail for Windows Tools

- Natural Language Support Configuration Tool

- On Distribution List Tool

- PFAdmin Tool

- Public Folders Tree Info Tool

- Phone Book Search Tool

- Proxy Kicker Tool

- Profile Generator

- Proxy Conversion Tool

- Replver Tool

- RPC Counter

- Microsoft Schedule+ Address Book Assistant

- Bulk Advanced Security Tool

- Self-Certification Form

- Sentinel Tool

- Simple Mail Handler

- Server Space Client Extension

- Mailbox Statistics Tool

- SwitchForms Tool

- Preview Pane Client Extension

- UNIX Mail Source Extractor sample

These utilities are available for different platforms. See
http://mspress.microsoft.com/books/1264.htm for a list of which platforms are
available to run these tools.

The Microsoft BackOffice Resource Kit, Part Two is also available at computer
stores and bookstores. The ISBN number is 1-57231-534-2.

The complete Resource Kit contains the following documentation:

- Microsoft SNA Server Resource Guide

- Microsoft SQL Server Resource Guide

- Microsoft Exchange Server Resource Guide, Supplement One

- Microsoft Systems Management Server Resource Guide, Supplement One

The CD-ROM contains more than 100 tools.
