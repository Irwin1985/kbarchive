Q148409: PRB: HPPCL5 Driver Causes Resource Depletion in Windows 3.11

Article: Q148409
Product(s): Microsoft FoxPro
Version(s): WINDOWS:3.0,3.0b
Operating System(s): 
Keyword(s): kb3rdparty
Last Modified: 10-FEB-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b 
-------------------------------------------------------------------------------

SYMPTOMS
========

Windows version 3.11 system resources diminish or become depleted when you use
the HPPCL5 driver version 1.2.

CAUSE
=====

This is a problem with the HPPCL5 driver.

RESOLUTION
==========

The problem is corrected by a new driver version 1.5 or later.
