SUMMARY
=======

The following article lists the directory of the 3.5-inch Microsoft Scenes
Sierra Club Collection Disk 1 (Disk Assy 213-051-045). The collection comes on
two 3.5-inch disks. You can order the program on 5.25-inch disks using the 1.2
MB Disk Offer coupon that comes in the package.
