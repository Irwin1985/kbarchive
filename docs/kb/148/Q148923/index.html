Q148923: Problems Printing Envelopes with Panasonic KXP-4400 Printer

Article: Q148923
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print an envelope with the Panasonic KXP-4400 printer, the address is
printed at the top of the envelope.

CAUSE
=====

This problem occurs when you use the default Panasonic KXP-4400 printer drivers
included with Windows 95. The following table lists these drivers:

  Driver         Version
  ---------------------------------
  Hppcl.drv      4.00.950, GPC 3.01
  Finstall.dll   4.0.2.01
  Unidrv.dll     4.00.950
  Iconlib.dll    4.00.950

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
