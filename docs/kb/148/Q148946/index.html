Q148946: AT Gen: Who Provides Mail for AppleTalk Networks Support?

Article: Q148946
Product(s): Microsoft Mail For Appletalk Networks
Version(s): 3.0x,3.1d,3.1f,3.1x,3.2
Operating System(s): 
Keyword(s): 
Last Modified: 13-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 3.0x, 3.1x 
- Microsoft Mail Connection for PC and AppleTalk Networks, version 3.2 
- Microsoft Schedule+ for AppleTalk Networks version 1.0 on Microsoft AppleTalk Mail, versions 3.1d, 3.1f 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft U.S. no longer supports Mail for AppleTalk Networks or the Mail
Connection Gateway. Anyone calling in for support on these products should be
asked to call StarNine/Quarterdeck at (510) 704-1272.

MORE INFORMATION
================

The following products are now supported by StarNine/Quarterdeck:

- Microsoft Mail for AppleTalk Networks server and client version 3.1x

- All components in the Mail for AppleTalk Networks server box

- OOF Server

- Remote Server Connection

- Mail Switch

- MS Mail Dialin

- Hypercard SDK

- MS Mailer

- GWS Monitor

- Forms Designer

- Microsoft Mail Connection for PC and AppleTalk Networks version 3.2

- Microsoft Mail for AppleTalk Networks SDK

- Microsoft Mail for AppleTalk Networks Gateway SDK version 3.0

- Microsoft Mail for AppleTalk Networks AppleLink GW version 3.0

The following Macintosh products are still supported by Microsoft:

- Microsoft Exchange Connector to Microsoft Mail for AppleTalk Networks


- Microsoft Mail for PC Networks Macintosh workstation version 3.x

- Microsoft Schedule+ for Windows version 7.0

The following products are not supported by either Microsoft or
StarNine/Quarterdeck:

- Gateway Name Utility (GNU)

- BBS 1.0 Front-End

- Microsoft Mail for AppleTalk Networks server and client version 3.0b

- Microsoft Mail for AppleTalk Networks Connection Gateway version 1.0b

- Microsoft Mail for AppleTalk Networks Gateway to Fax version 3.00a

- Microsoft Mail for AppleTalk Networks PhoneNet Liaison Gateway

- Microsoft Schedule+ for AppleTalk Networks version 1.0

- ListMail

- MailValet

- Message Mover

- MoveMail

- Print Personal Address List (PrintPAL)

- ConvertRecips

- Localized versions of the Mail for AppleTalk Networks and Mail Connection
  Gateway products

- StatWizard

- StatAlert

- Mailbox Connection


Additional query words: 3.1a 3.1c 3.1d 3.1f starnine quarterdeck
