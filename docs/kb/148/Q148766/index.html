Q148766: Return of Arcade Disk Contents (3.5 inch/1.44 MB DMF)

Article: Q148766
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Return of Arcade for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article contains the directory listings and cabinet file contents
for the Return of Arcade version 1.0, disks 1-3, 3.5-inch, 1.44 MB DMF. The
disks in Return of Arcade use the DMF format.
