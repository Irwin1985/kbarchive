Q93400: Microsoft Language Utility Version History

Article: Q93400
Product(s): Microsoft Programming Utilities
Version(s): 1.0,1.11,1.13,1.2,1.3,1.4,1.5,1.6,2.5,3.0,3.1,3.14,3.17,3.18,3.2,3.20.010,3.31,3.4,4.0
Operating System(s): 
Keyword(s): kbCodeView kbDebug kbVC
Last Modified: 26-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft LIB for MS-DOS, versions 3.1, 3.17, 3.18, 3.2, 3.20.010, 3.31, 3.4 
- Microsoft LIB Utility for Windows NT, versions 1.0, 2.5, 3.0, 3.1, 4.2 
- Microsoft LINK for MS-DOS, versions 5.01.20, 5.1, 5.13, 5.15, 5.1x, 5.3, 5.31.009, 5.5, 5.6 
- Microsoft LINK for Windows NT, versions 1.0, 2.5 
- Microsoft NMAKE Utility for MS-DOS, versions 1.11, 1.13, 1.2, 1.3, 1.4, 1.5 
- Microsoft NMAKE Utility for Windows NT, versions 1.4, 1.5, 1.6 
- Microsoft CodeView for MS-DOS, versions 3.14, 4.0, 4.01, 4.1 
- Microsoft CodeView for Windows, versions 4.0, 4.01, 4.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The text below lists the versions of the Microsoft utilities distributed with
various versions of Microsoft developer products.
