Q249697: Encarta 2000: How to Disable or Enable MindMaze

Article: Q249697
Product(s): Microsoft Home Multimedia Titles
Version(s): ; WINDOWS:
Operating System(s): 
Keyword(s): kbenable kbtool kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
- Microsoft Encarta Reference Suite 2000 
- Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to disable or enable MindMaze, a game within
Microsoft Encarta Encyclopedia 2000 that tests your knowledge by asking
questions about a variety of subjects.

NOTE: If you run Encarta Encyclopedia 2000 on a Microsoft Windows 2000-based
computer or a Microsoft Windows NT 4.0-based computer, you need Administrator
privileges to disable or enable MindMaze.

Disable MindMaze
----------------

To disable MindMaze:

1. Insert the Installation & Resources CD-ROM into the CD-ROM drive. Press
  and hold down the SHIFT key as you insert the CD-ROM to prevent the program
  from starting automatically.

2. Click Start, and then click Run.

3. In the Open box, type the following line that is appropriate for your version
  of Encarta.

  For Encarta 2000:

  <drive>:\Support\Regfiles\Ee\Mm_off.reg

  where <drive> is the drive letter of the CD-ROM drive, the DVD-ROM
  drive, or the drive where the shared folder is located on the network.

  For Encarta 2001:

  <drive>:\Support\Encyc\Mm_off.reg

4. Click OK.

Enable MindMaze
---------------

To enable MindMaze:

1. Insert the Installation & Resources CD-ROM into the CD-ROM drive. Press
  and hold down the SHIFT key as you insert the CD-ROM to prevent the program
  from starting automatically.

2. Click Start, and then click Run.

3. In the Open box, type the following line that is appropriate for your version
  of Encarta.

  For Encarta 2000:

  <drive>:\Support\Regfiles\Ee\Mm_on.reg

  For Encarta 2001:

  <drive>:\Support\Encyc\Mm_on.reg

4. Click OK.
