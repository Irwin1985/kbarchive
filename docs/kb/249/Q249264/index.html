Q249264: AC Err Msg: Failed to Establish Connection to the Server

Article: Q249264
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbnetwork igz kbimu msgame
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call, version 1.0 
- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Play in a Microsoft Asheron's Call game room on Zone.com, a
screen with two progress vials may be displayed but neither vial may be filled.

In addition, you may receive the following error message after several minutes:

  Failed to establish connection to the server. Please try again in a few
  minutes.

CAUSE
=====

This behavior can occur for any of the following reasons:

- You are connecting to the Asheron's Call server through a proxy server or
  firewall.

- You are connecting to the Asheron's Call server using Network Address
  Translation.

- Your are experiencing high Internet latency.

RESOLUTION
==========

To resolve this issue, use the following methods in the order in which they are
presented. If the issue continues to occur, proceed to the next method.

Configure the Proxy Server or Firewall
--------------------------------------

For additional information about how to configure a proxy server or firewall for
use with Asheron's Call, click the article number below to view the article in
the Microsoft Knowledge Base:

  Q236430 Asheron's Call: Ports Required to Play Game Through a Firewall

Configure Asheron's Call to Work with Network Address Translation
-----------------------------------------------------------------

For additional information about how to configure Asheron's Call to work with
Network Address Translation, click the article number below to view the article
in the Microsoft Knowledge Base:

  Q238325 Asheron's Call: Only One Player on Your LAN Is Able to Play Game

Reduce Your Internet Latency
----------------------------

For additional information about how to reduce your Internet latency, click the
article number below to view the article in the Microsoft Knowledge Base:

  Q232392 Zone.com: High Latency in Game Lobbies or When Playing Games
