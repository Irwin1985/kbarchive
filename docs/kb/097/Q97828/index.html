Q97828: WEP 2: Level Passwords for Pipe Dream

Article: Q97828
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbui kbimu msgame
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Best of Windows Entertainment Pack, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a list of the passwords for the Pipe Dream game in
Microsoft Entertainment Pack Volume 2.

MORE INFORMATION
================

The following table shows the password you need to type to start a game in Pipe
Dream at the corresponding level:

  Level           Password
  ------------------------

  Level 5         PLAM
  Level 9         ZAMI
  Level 13        POOM
  Level 17        DINK
  Level 21        SPOO
  Level 25        BORK
  Level 29        TARR
  Level 33        FOOZ

Additional query words: 2.00 Pipedream windows fun games wep w_wep msgame
