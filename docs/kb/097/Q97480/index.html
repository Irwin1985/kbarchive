Q97480: Setting the Default Word Processing Application in Encarta

Article: Q97480
Product(s): Microsoft Home Multimedia Titles

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia 
- Microsoft Encarta 96 Encyclopedia for Windows 
- Microsoft Multimedia Viewer 
- Microsoft Word for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Encarta includes a button to quickly access a word processor, so you can take
notes or copy articles. Encarta allows you to specify which of the word
processing applications installed on your system you want to access when you
click this button.

Encarta 1995 and 1996 Editions
------------------------------

To set a default word processor in Encarta 1995 and 1996 editions, do the
following:

1. Click the I button.

2. Click Settings(Encarta 95) or Options(Encarta 96).

3. In the next dialog box, click Choose Word Processor.

4. In the Select Word Processor window, select the word processing program you
  want to use. (You may have to change folders to locate the program you want
  to use.)

5. Click OK.

6. Click OK.

Encarta 1993 and 1994 Editions
------------------------------

To set a default word processor in Encarta 1993 and 1994 editions, do the
following:

1. On the main button bar, click the Menu button.

2. Choose the Options button.

3. In the Options dialog box, click Choose Word Processor.

4. In the Select Word Processor window, select the word processing program you
  want to use. (You may have to change directories to locate the program you
  want to use.)

5. Click OK.

6. Click Set Options.

MORE INFORMATION
================

Many programs can be used with Encarta, including the following:

- Lotus Works
- Easy Working "Eight-in-One"
- PFS Windows Works (by Spinnaker)
- Claris Filemaker Pro
- Microsoft Excel
- Microsoft Word for Windows
- Microsoft Windows Notepad
- Microsoft Windows Write

NOTE: WordPerfect does not work with Encarta 1993 or 1994.
