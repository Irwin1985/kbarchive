Q78458: NEC 80286 APC4 Is Not Supported by MS-DOS 5.x & 6.x

Article: Q78458
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SUMMARY
=======

According to NEC, MS-DOS 5.0 and later do not work on the NEC PowerMate 80286
APC4. The APC4 is an old machine that was designed to run MS-DOS versions up to
and including version 3.2.

NOTE: Some customers have been able to get MS-DOS 5.0 and later to work on the
APC4 by upgrading their BIOS to version 3.07. However, NEC does not support this
setup because it is may not be stable.
