Q78659: Sytos and Sytos Plus Tape Backups with MS-DOS 5.0

Article: Q78659
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 03-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Sytron has confirmed that the Sytos tape backup system has not been tested with
Microsoft MS-DOS 5.0 by either Microsoft or Sytron, and may or may not work with
MS-DOS 5.0.

Sytos Plus has been tested by Sytron. It is compatible with MS-DOS 5.0.

Sytos and Sytos Plus are tape backup systems manufactured by Sytron Corp.

This information applies to Microsoft MS-DOS version 5.0.

MORE INFORMATION
================

Sytos tape operating system version 3.11 was discontinued in 1990, and has not
been thoroughly tested with MS-DOS 5.0. If MS-DOS is not loaded in the high
memory area (HMA), Sytos may work. In the event Sytos is not compatible with
MS-DOS 5.0, Sytron recommends upgrading to Sytos Plus version 1.21 or later.

Sytos Plus File Backup Manager versions 1.21 and later have been tested with
MS-DOS 5.0 by Sytron, and Sytron has confirmed these versions are compatible
with MS-DOS 5.0. The current version (as of November 7, 1991) of Sytos Plus File
Backup Manager is version 1.31.

The products included here are manufactured by vendors independent of Microsoft;
we make no warranty, implied or otherwise, regarding these products' performance
or reliability.
