Q284195: Encarta 99: Cannot Use Research Organizer on Windows 2000

Article: Q284195
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,2000
Operating System(s): 
Keyword(s): kbenv kberrmsg kbimu
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 99 Research Organizer, used with:
   - Microsoft Windows 2000 Professional 
- Microsoft Encarta 98 Research Organizer, version 1.0, used with:
   - Microsoft Windows 2000 Professional 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to run Microsoft Encarta 99 Research Organizer on a Microsoft
Windows 2000-based computer, you receive the following error message in a Dr.
Watson Log file:

  The application, , generated an application error The error occurred on
  01/03/2001 @ 14:51:35.042 The exception generated was c0000005 at address
  00407681

CAUSE
=====

This behavior occurs because Encarta 99 Research Organizer is not compatible
with Windows 2000.

MORE INFORMATION
================

Although Encarta 99 Research Organizer does not work in Windows 2000, all the
other Encarta 99 Reference products work.

Microsoft Encarta Research Organizer 2000 and Microsoft Encarta Researcher 2001
are compatible with Windows 2000. For information about these programs, please
contact Microsoft Customer Service at 1-800-426-9400.
