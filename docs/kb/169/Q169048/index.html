Q169048: Full Court Press: New Season Game Patches Are Available

Article: Q169048
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbfile kbusagekbfaq
Last Modified: 20-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft NBA Full Court Press for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you own the full version of NBA Full Court Press, you can download the
1995-96 or 1996-97 patch to make game play more fun and realistic.

NOTE: These patches will not work for the international version of NBA Full Court
Press.

The file size of each patch is approximately 0.8 MB.

MORE INFORMATION
================

In the both patches, the nets are animated on made shots, the 24-second clock
resets correctly after all defensive fouls and illegal defenses, and Lob passes
occur less frequently.

The 1995-96 patch also includes the 1995-96 NBA season player and team ratings
attribute changes.

The 1996-97 patch also includes the updated 1996-97 NBA rosters.

NOTE: This patch does not include bitmap mug shots or Kevin Calabro announcements
of player names for any of the 1997 rookies or free agents.

TO OBTAIN THESE FILES
---------------------

The following files are available for download from the Microsoft Download
Center:

  Exeratin.exe
  (http://download.microsoft.com/download/nbafullcourtpresswin95/Update/1/WIN98/EN-US/Exeratin.exe)

  Exerost.exe
  (http://download.microsoft.com/download/nbafullcourtpresswin95/Update/2/WIN98/EN-US/Exerost.exe)

Release Date: Sep-04-1997

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
