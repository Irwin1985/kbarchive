Q132387: Sound Settings Are Not Saved

Article: Q132387
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Dogs for Windows, version 1.0 
- Microsoft Oceans for Windows, version 1.0 
- Microsoft World of Flight for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start one of the applications listed above, sound effects and
narrations function, even though you turned them off the last time you used the
program.

STATUS
======

This feature is part of the program design.

MORE INFORMATION
================

Microsoft Dogs, Microsoft Oceans and Microsoft World of Flight do not restore
sound settings from previous sessions when you start the program. If narrations
and/or sound effects are turned off during a session, the next time you start
the application, the sounds play.
