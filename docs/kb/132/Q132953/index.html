Q132953: PC Gen: Using the Clipboard Option in the MAILINT.EXE Utility

Article: Q132953
Product(s): Microsoft Mail For PC Networks
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 11-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for PC Networks Resource Kit 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Mail Integrity Check utility, included with the Microsoft Mail for
PC Networks Resource Kit, allows you to copy the error list to the Clipboard.
Then it can be pasted into another application to be saved or printed.

When you copy the error list to the Clipboard, the first error is truncated. As a
result, it is not viewable in the Clipboard or when pasted into the destination
application.

MORE INFORMATION
================

The Microsoft Mail Integrity Check utility is a read-only utility used to check
for missing or corrupt Mail database files and directories. The errors reported
by this program will be displayed in the Mail Check Error List portion of the
dialog box and may subsequently be copied to the Clipboard by utilizing the Clip
option or by selecting Copy Error List to Clipboard from the File menu.

This utility is licensed as is without warranty of any kind and is not supported
by Microsoft.
