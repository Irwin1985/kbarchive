Q132888: Encarta 95: GP Fault When Using Dictionary or Thesaurus

Article: Q132888
Product(s): Microsoft Home Kids Products
Version(s): 1995 edition
Operating System(s): 
Keyword(s): kberrmsg kbtool kbimu
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you look up a word in the Encarta 1995 Dictionary or Thesaurus, you may
receive the following error message:

  Dict caused a general protection fault in module Redllwin.dll.

If you click Close, you quit Encarta and cannot restart the program.

If you use the Encarta 1995 Thesaurus in Microsoft Windows 95/98, you may receive
the following error message:

  This program has performed an illegal operation and will be shut down. If the
  problem persists, contact the program vendor.

If you click Close, you quit the Thesaurus but Encarta remains running.

If you click Details, you receive the following error message:

  Dict caused a general protection fault in module Redllwin.dll.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta for Windows,
1995 edition.

WORKAROUND
==========

To work around this issue, exit Windows (if you are using Microsoft Windows
3.x), and then restart the computer. You can now restart Encarta.
