Q125598: Ensoniq Soundscape Support in Windows NT

Article: Q125598
Product(s): Microsoft Windows NT
Version(s): 3.10 3.50 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 3.1 
- Microsoft Windows NT Workstation version 3.1 
- Microsoft Windows NT Advanced Server, version 3.1 
- Microsoft Windows NT Workstation versions 3.5, 3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

The Ensoniq line of sound cards are not currently on the Hardware
Compatibility List (HCL). However, Ensoniq does provide Windows NT drivers
for the following cards on their Web page (WWW.ENSONIQ.COM) and Bulletin
Board System (BBS) at (610) 647-3195:

  Soundscape
  Soundscape Elite
  Soundscape (OPUS based)
  Soundscape VIVO

The download web page for the drivers is:

  http://www.ensoniq.com/html/downloads.htm

For more information, contact Ensoniq Technical Support at (800) 942-0096.

Soundscape is a 16-bit, 32-voice wavetable synthesizer designed for
maximum compatibility with all PC audio standards, with an emphasis on use
with games and multimedia applications. The sound card is also featured in
most Gateway 2000 multimedia systems.

The third-party products discussed here are manufactured by vendors
independent of Microsoft; we make no warranty, implied or otherwise,
regarding these products' performance or reliability.
