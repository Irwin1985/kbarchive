Q125734: Word Processor in Encarta Asks for Bookshelf CD

Article: Q125734
Product(s): Microsoft Home Kids Products
Version(s): 1993 edition,1994 edition,1995 edition; WINDOWS:2.0a,2.0b,2.0c,3.1,3.11,6.0
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
- Microsoft Bookshelf for Windows versions 1993 edition, 1994 edition, 1995 edition 
- Microsoft Bookshelf 1996-97 for Windows 
- Microsoft Word for Windows, versions 2.0a, 2.0b, 2.0c, 6.0 
- the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

If Microsoft Word for Windows is selected as the default word processor in
Microsoft Encarta, and if Microsoft Bookshelf is installed with Word for Windows
integration, you will be prompted to enter the Bookshelf CD when running Word
for Windows from Encarta.

CAUSE
=====

The behavior occurs if Bookshelf is integrated with Word for Windows and the
Quote Of The Day option is selected in Word. To retrieve the quotation, Word for
Windows looks for the Bookshelf CD.

NOTE: This problem does not occur in Windows 95.

WORKAROUND
==========

To run Word for Windows from Encarta without having to switch CDs, perform one
of the following workarounds:

Workaround 1
------------

The following message is displayed when you run Word for Windows from Encarta

  Please Insert the Bookshelf <year> disk in the CD-ROM drive. If the
  drive letter shown below is not the correct drive, please change it.

where <year> is either 1993 or 1994. OK and Cancel are the only options.
Click the Cancel button when prompted for the Bookshelf CD. After you click
Cancel, the Quote Of The Day dialog box appears with the options:

  [X] Display Quotation Daily Box
  Another Quote
  Paste Quote
  Close
  Help

Selecting Close runs Word for Windows; however, the next calendar day produces
the same symptom after you run Word for Windows from within Encarta.

Workaround 2
------------

Clear the Display Quotation Daily option.
