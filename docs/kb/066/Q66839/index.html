Q66839: Flight Simulator: Plane Does Not Crash into Scenery Object

Article: Q66839
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

Microsoft Flight Simulator will not detect a building or mountain crash unless
one of the 3-D views is set to cockpit.

MORE INFORMATION
================

There are four different views that you can set for the active 3-D window:
cockpit, tower, track, or spot. Cycle through the choices by pressing S.
