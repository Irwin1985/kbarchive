Q66350: IdleWild Screen Saver Does Not Hide Mouse Pointer

Article: Q66350
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

When the IdleWild screen saver blanks the screen under Windows, the mouse
pointer is not concealed, and will be the same shape that it was when the screen
blanked.

If the mouse is moved after IdleWild blanks the screen, the movement will be
detected and the normal screen will be restored. IdleWild has no effect on the
mouse cursor.

Additional reference word(s): wep entertainment pack I
