Q324477: Dungeon Siege 1.0: Dungeon Siege Update

Article: Q324477
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Dungeon Siege, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses the Microsoft Dungeon Siege 1.1 update. The first part of
this article discusses how to obtain the updated version of Dungeon Siege, and
the second part of this article lists the fixes that this update provides.
Microsoft recommends that you install this update.

MORE INFORMATION
================

How to Install the Dungeon Siege Update
---------------------------------------

To obtain the most recent Dungeon Siege update, follow these steps:

1. Start Dungeon Siege.

2. Click Multiplayer.

3. Click ZoneMatch.

4. Click Yes when either of the following messages appears:

  A new version of the game update software is available. Do you want to
  install this now?

  -or-

  There is an update for Dungeon Siege available. Do you want to install this
  now?

Issues Fixed by the Dungeon Siege 1.1 Update
--------------------------------------------

The Dungeon Siege 1.1 update contains the following updates and changes.

General:

- Fixed all known Journal update issues related to quests.

- Fixed all known content placement issues such as inaccessible containers and
  "floating" objects.

- Increased game performance in several low-performance areas.

- Resolved all known issues with end bosses.

- Fixed all known fading issues.

- Updated ZoneMatch user interface and improved ZoneMatch functionality and
  performance:

   - Icon indicating password protection.

   - More user-friendly chat interface.

   - New searching support based on user-defined filters.

   - Ability to message friends from ZM interface, the staging area, and
     end-game screen.

   - Increased stability.

- Resolved issues with sounds cutting out.

- Added safety to prevent the game from starting if critical resource files are
  missing.

- Added data corruption retry code.

- Game tries to recover from unstable hardware, which causes corruption in data
  that is loaded from hard disk.

- Moved screen shots into their own Screen Shots subdirectory.

- Added new command line options:

   - noalttab=true: Prevents task switching during gameplay (supported on
     Windows XP only).

   - nowinkeys=true: Turns off (disables) the Windows keys (supported on
     Windows 2000 and Windows XP only).

   - user_path=<path>: Specifies where the user files go (instead of My
     Documents\Dungeon Siege).

NOTE:Other path-related command line options include the following: keys_path,
map_paths, mod_paths, res_paths, save_path, and shots_path.

- Fixed the issue of a missing or moved "My Documents" folder that caused
  issues.

- Improved many error messages to include more information to help better
  diagnose hardware configuration problems.

- Added an hourglass pointer to show when the game is busy.

- Switched to 100-percent client/server model to be more NAT and
  firewall-friendly.

- "Chunky" and "superchunky" cheat codes removed.

- Displacers now save their state in the character save file in multiplayer by
  using new quest saving functions.

- "Transmute" scroll can no longer transmute characters.

- Fixed the issue of characters getting "stuck" in the Swamps.

Mod Related:

- Switched map node mesh index format to be more mod-friendly when gmax comes
  out.

- Added a new naming key format to be more mod-friendly.

- Exported many new functions.

- Added .dsdll extension method.

- Exported user interface functions and added "command" and "call" actions for
  user interface modding.

Additional query words: ui patch
