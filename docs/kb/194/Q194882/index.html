Q194882: Encarta Virtual Globe 99: Cannot Use IntelliMouse Wheel to Zoom

Article: Q194882
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbhw kbinterop kbimu kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

While you use Encarta Virtual Globe 99, you may be unable to use the Microsoft
IntelliMouse wheel to zoom in or out on the map.

CAUSE
=====

This behavior can occur for any of the following reasons:

- You use the IntelliMouse wheel to pan the map across the screen.

- You draw a box around a specific map area you want to zoom in on.

- You use the Dynamic Sensor tool or the Measure tool.

RESOLUTION
==========

To restore functionality to the IntelliMouse wheel, drag the zoom slider located
at the bottom of the map to the left or the right.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
99, version 1.0.
