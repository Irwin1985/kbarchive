Q194366: Encarta: How to Contact Encarta Annual Update Program

Article: Q194366
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbpolicy kbimu
Last Modified: 01-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 98 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 99 
- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes the methods by which an Encarta Annual Update Program
(AUP) subscriber can contact the AUP.


MORE INFORMATION
================

Contact the Microsoft Encarta Annual Subscription team using one of the
following methods:

Telephone
---------

Telephone (888) 803-4570 to contact the AUP.

Fax
---

In the United States, fax your request to the following number:

  716-447-7330

In Canada, fax your request to the following number:

  905-374-3855

Mail
----

In the United States, mail your request to the following address:

  Microsoft Encarta
  Annual Update Program
  PO Box 1094
  Buffalo NY 14240-8720

In Canada, mail your request to the following address:

  Microsoft Encarta Canada
  Annual Update Program
  PO Box 643
  Fort Erie ON L2A 9Z9
  Canada

The Encarta Annual Update Program was a special program that sent subscribers
each year's new edition of select Microsoft Encarta reference products.
Subsription was free, and customers were notified of special-price offers to for
new products.

The AUP was discontinued in July 2001, but existing subscribers may still contact
the AUP for questions about existing orders.

AUP has been replaced "Club Encarta" which is only available to users of 2002
Encarta products. To join Club Encarta, open your 2002 Encarta product and on
the Help menu, click Join Club Encarta.

For information on the latest Encarta reference products, point your Web browser
to:

  http://encarta.msn.com/products
