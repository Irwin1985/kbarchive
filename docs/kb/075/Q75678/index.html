Q75678: Alphabetic Listing of MS-DOS 5 Upgrade Files with Disks

Article: Q75678
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 21-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a cross-reference of Microsoft MS-DOS 5 Upgrade files. The
files are listed in alphabetic order followed by their location on the 5.25- and
3.5-inch MS-DOS 5 Upgrade disks.
