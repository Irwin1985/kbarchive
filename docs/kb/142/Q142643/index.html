Q142643: Windows NT 4.0 Driver Library

Article: Q142643
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbprint kbPrinting kbHardware kbgraphxlinkcritical
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

NOTE: The drivers listed in this article have been tested and certified only
with Windows NT version 4.0.

Microsoft provides the files in the Microsoft Windows NT Driver Library (WNTDL).
Files in the WNTDL support many printers, displays, sound cards, and network
card adapters that the Windows NT version 4.0 release does not support directly.
If you have a modem, the drivers are available at no charge.

The Microsoft Windows Driver Library (WDL) is available through online services.
For additional information about the WDL, click the article number below to view
the article in the Microsoft Knowledge Base:

  Q135314 Windows 95 Driver Library

Please note that standard connect-time fees and long-distance telephone charges,
if any, apply when you download files.

MORE INFORMATION
================

Windows NT Driver Library
-------------------------

The Microsoft Windows NT Driver Library (WNTDL) contains device drivers for
printers, displays, audio devices, storage devices, and network card adapters.
These drivers enable these devices to run with the Microsoft Windows NT
operating system.

As new and updated drivers become available, they are added to the WNTDL.

NOTE: All of the drivers in this list are intended for use with Windows NT
version 4.0.

Instructions for Downloading
----------------------------


To download the files:

1. If you are downloading to a floppy disk, you need to have a formatted, blank
  disk (copy the files to the root directory of the floppy disk). If you are
  downloading to your hard disk, create a new subdirectory to place the files
  in.

  CAUTION: Do not download files directly to your main Windows NT directory.
  Doing so could overwrite files essential to the proper operation of your
  system.

2. Follow the download procedure. The file you download is the executable or
  program (.exe) file that you identified in the "Files Available from Online
  Services" section of this article. This file contains all of the files that
  you need to support your device.

  Download the .exe file to your floppy disk or to the new subdirectory that you
  created on your hard disk.

3. Change to the floppy disk drive (or the subdirectory on your hard disk) that
  contains the .exe file. At the MS-DOS prompt, type "<filename> -d"
  (without the quotation marks), and then press the ENTER key.

  When the .exe file finishes running, all of the files that you need to support
  your device, such as a .drv file and an Oemsetup.inf file, are set up. You
  are also provided with a .txt file that contains instructions on how to
  install the device drivers (or other software) and a licensing agreement.

  The -d switch causes the file to recreate any subdirectories in the original
  files, and is very important when you extract files from executables that
  contain multiple platform versions.

  If you have problems when you extract the files, try downloading the files
  again.
