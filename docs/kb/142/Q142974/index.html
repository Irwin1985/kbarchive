Q142974: Views.exe - One-To-Many Form Using a Parameterized View

Article: Q142974
Product(s): Microsoft FoxPro
Version(s): WINDOWS:3.0,3.0b,5.0,6.0
Operating System(s): 
Keyword(s): kbfile kbsample kbDatabase kbSQL kbvfp300 kbvfp500 kbvfp600 kbSQLProg
Last Modified: 27-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Views.exe is a sample that provides an introduction to the concepts of
parameterized views and how they can be applied in Visual FoxPro.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

  Views.exe
  (http://download.microsoft.com/download/vfox60/sample/2/WIN98/EN-US/Views.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

This document provides a basic understanding and overview of views, parameterized
views, the flexibility and power of using views, and a practical application of
these concepts.

The material explains how to control the display of related data using
parameterized views and gives related step-by-step procedures that guide you
through the process of creating and using parameterized views.
