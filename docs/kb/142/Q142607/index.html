Q142607: Extra Sets for Pan Chinese Windows NT Workstation 4.0

Article: Q142607
Product(s): Microsoft Windows NT
Version(s): 
Operating System(s): 
Keyword(s): kbenv
Last Modified: 19-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation, Pan Chinese Edition, version 4 
-------------------------------------------------------------------------------

SUMMARY
=======

There are two Extra sets for Pan Chinese Windows NT Workstation 4.0:

- Extra set for Chinese Taiwan locale (BIG5 Code)

- Extra set for Chinese PRC locale (GB or GBK Code)

MORE INFORMATION
================

Extra Set for Chinese Taiwan locale (BIG5 Code)
-----------------------------------------------

This set includes the following features:

- Hong Kong common characters End-User Defined Characters (EUDCs) for Taiwan
  locale.

- Two additional fonts are included (Hk-xmt.ttf and Hk-zht.ttf). These fonts
  match the MingliU typeface in style and contains 3,000+ Hong Kong specific
  characters.

This set contains the following files:

  File Name      Modified Date       Size
  ---------------------------------------------
  B5font.inf     5/30/1997          5,668 bytes
  Hk-xmt.ttf     1/16/1997      1,233,920 bytes
  Hk-zht.ttf     1/14/1997      1,396,516 bytes
  Unb5font.inf   5/30/1997          3,132 bytes


Perform the following steps to install the Extra set for Chinese Taiwan locale:

1. Download Disk1.exe from the following location:

  http://www.microsoft.com/hk/pcntw/html/extras.htm

2. In My Computer or Windows NT Explorer, double-click the Disk1.exe file you
  downloaded in step 1.

  NOTE: The file Mingliu.ttf should already be installed.

3. Right-click the B5font.inf file and then click Install.

4. Shut down and restart your computer.

Perform the following steps to uninstall the Extra set for Chinese Taiwan
locale:

1. Right-click the Unb5font.inf file and then click Install.

2. Shut down and restart your computer.

For information how to install MingLiU font in Pan Chinese Windows NT Workstation
4.0, please see the following articles in the Microsoft Knowledge Base:

  ARTICLE-ID: Q136477
  TITLE : How to Install MingLiU Fonts for BIG5 Applications

  ARTICLE-ID: Q166426
  TITLE : Internet Explorer in PCNTW Cannot view Chinese BIG5 Characters

Extra set for Chinese PRC locale (GB or GBK Code)
-------------------------------------------------

This set includes the following features:

- Hong Kong common characters End-User Defined Characters (EUDCs) for (PRC)
  locale.

- Two additional fonts are included (Gbk-hkht.ttf and Gbk-hkst.ttf). These
  fonts contain 1,500+ Hong Kong specific characters that are not already in
  GBK.

- Help files for the modified ZhengMa input method (Hkzm.cnt and Hkzm.hlp).

- Utility to convert Word 95 documents between BIG5, GBK, and Unicode
  Convert.exe

- Turning on/off the system default Font Association. For certain English
  applications, turning off the Font Association is necessary to prevent High
  ANSI characters (for example, smart quotes) from displaying as Chinese
  characters.

- The source file of a Traditional Chinese Quick Cangjie input method
  (Quick_u.txt).

This set contains the following files:

  File Name      Modified Date       Size
  ---------------------------------------------
  Convert.exe    10/19/1997       387,072 bytes
  Gbk-hkht.ttf   12/10/1997       550,012 bytes
  Gbk-hkst.ttf   01/14/1997     1,396,516 bytes
  Hkfont.inf     05/30/1997         3,646 bytes
  Hkzm.cnf       12/17/1996           764 bytes
  Hkzm.hlp       12/13/1996       310,716 bytes
  Quick_u.txt    02/20/1997       173,290 bytes
  Unhkfont.inf   05/30/1997         3,166 bytes
  Fa_off.inf     12/15/1996           405 bytes
  Fa_on.inf      12/15/1996           407 bytes
  Readme.txt     12/19/1996           563 bytes


Perform the following steps to install the Extra set for Chinese PRC locale:

1. Download Disk2.exe from the following location:
  http://www.microsoft.com/hk/pcntw/html/extras.htm

2. In My Computer or Windows NT Explorer, double-click the Disk2.exe file you
  downloaded in step 1.

3. Right-click the Hkfont.inf file and then click Install.

4. Shut down and restart your computer.

Perform the following steps to uninstall the Extra set for Chinese PRC locale:

1. Right-click the Unhkfont.inf file and then click Install.

2. Shut down and restart your computer.
