Q142380: Downloading Encarta Updates From MSN Produces Error Message

Article: Q142380
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Windows 
- the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

Attempts you make to download Encarta 96 monthly updates from MSN, The Microsoft
Network, fail.

When you connect to MSN from within Encarta 96 Encyclopedia or directly to MSN
and attempt to download the monthly updates, you receive the following error
message:

  This file cannot be accessed.
  Make sure that you have security privileges on the network drive.

CAUSE
=====

The drive that you are have selected a destination for the download is the
CD-ROM. The CD-ROM drive is read only and cannot be used as a destination for
downloaded files.

RESOLUTION
==========

Use one of the following two methods to download a monthly update to the
appropriate drive and folder:

Method 1
--------

1. Double-click the Shortcut To Yearbook Builder BBS On MSN icon in Encarta 96.

2. Double-click the monthly update you want to download.

3. Scroll to the bottom of the message and double-click the monthly update
  attachment. For example, double-click "jan96a.eyb" (without the quotation
  marks)

4. Click Download File.

Method 2
--------

1. Double-click the Shortcut To Yearbook Builder BBS On MSN icon in Encarta 96.

2. Double-click the monthly update you want to download.

3. On the File menu, click Save As.

4. In the Save box, click Attachments.

5. In Drives box, select the hard drive where Encarta is installed.

6. In the Folders box, select the Yearbook folder (located under the Encarta
  Folder).

7. Click OK.
