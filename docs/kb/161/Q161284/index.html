Q161284: WD97: Word Hangs When Ungrouping 32nd Linked Text Box To Object

Article: Q161284
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): 
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

Undoing a grouping of the 32nd linked text box to an object causes Word to hang
(stop responding).

WORKAROUND
==========

Method 1: Do not Undo
---------------------

Use the ungroup feature, instead. Right-click the grouped object, point to
Grouping on the shortcut menu, and then click Ungroup.

Method 2: Do not group to the 32nd text box
-------------------------------------------

Do not group an object with the 32nd text box in a sequence. This also includes
every 32nd instance of a text box in linked sequences. For example, do not group
to the 32nd text box, the 64th text box, the 96th text box, and so forth in
linked sequences.

STATUS
======

Microsoft has confirmed this to be a problem in the products listed at the
beginning of this article.

For additional information, please see the following article here in the
Microsoft Knowledge Base:

  Q157544 Cannot Link More than 32 Text Boxes
