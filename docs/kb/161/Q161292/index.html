Q161292: WD97: Undo Insert Page Numbers Causes Word To Hang

Article: Q161292
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbualink97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

Word hangs (Not Responding) when an Undo is performed to remove Insert Page
Numbers in a section that has "Same as Previous" turned off.

WORKAROUND
==========

Do not perform an Undo. Delete the page number instead.

NOTE: Use this procedure if the page numbers were added by clicking Page Numbers
on the Insert menu.

To Delete the Page Number
-------------------------

1. On the View menu, click Header and Footer.

2. Change to the header or footer that contains the page number.

3. Move the mouse cursor over the page number until you get the four arrow
  cursor, and then click once. This will select the frame that contains the
  page number.

4. Delete the selected frame.

STATUS
======

Microsoft has confirmed this to be a problem in the products listed at the
beginning of this article.


MORE INFORMATION
================

An Undo can be done by either of the following:

- Clicking the Undo button on the Standard toolbar.

  -or-

- Pressing the keyboard combination CTRL+Z.

For more information on Undo:

Click the Office Assistant, type "What is the Undo button?", and then click
Search. Click the topic "Undo mistakes".

For additional information, please see the following article(s) in the Microsoft
Knowledge Base:

  Q157935 Keyboard Shortcuts for Word 97
