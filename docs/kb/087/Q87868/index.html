Q87868: Copy Picture and Copy Audio Unavailable After Search

Article: Q87868
Product(s): Microsoft Home Multimedia Titles
Version(s): 1992 edition
Operating System(s): 
Keyword(s): 
Last Modified: 05-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf for Windows 1992 edition 
-------------------------------------------------------------------------------

SUMMARY
=======

When performing a Search in Microsoft Bookshelf for Windows 1992 Edition, all
references found are displayed in a xxx Topics Found Window (where xxx is the
number of topics). If any of the topics include the American Heritage Dictionary
or the Concise Columbia Encyclopedia, the Copy Picture and Copy Audio command
under the Edit menu are disabled.

Workaround
----------

Use the Browse button to browse forward/backward and then backward/forward to the
original article. At this point, the Copy Picture and Copy Audio options are
enabled.

MORE INFORMATION
================

Steps to Reproduce Problem
--------------------------

1. Load the Books Screen.

2. Search for Cabot.

3. From the Topics Found Window double-click the item labeled

  Dict92:Cabot -Biography

  to move to this topic.

4. Open the Edit menu. The Copy Picture and Copy Audio options are disabled.

Additional query words: 1992 1.00 usage viewer 1.0 win
