Q72926: Upper Memory Managers for 8086, 8088, or 80286 Computers

Article: Q72926
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
Operating System(s): 
Keyword(s): msdos
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of software applications that are capable of using the
upper memory area (UMA) and/or high memory area (HMA) to load device drivers
and/or terminate-and-stay-resident programs (TSRs) on pre-80386 computer systems
to free up conventional memory. The list below specifies the product,
manufacturer, system requirements, and other information. For more complete
information on any of these products and the latest software version, please
contact the manufacturer.

Headroom, Helix Software Co., Inc.
----------------------------------

47-09 30th St., Long Island City, NY 11101
(718) 392-3100, Technical Support (718) 392-3735

System requirements include IBM PC, XT, or AT compatibles with expanded or
extended memory, or later computer processors.

Move'em, Qualitas Inc.
----------------------

7101 Wisconsin Ave., Suite 1386, Bethesda, MD 20814
Technical Support (301) 907-7400

System requirements include any IBM PC compatible with EMS 4.0 board with at
least 256KB or 80286/80386 with a NEAT or AT/386 CHIPSet from Chips and
Technologies with at least 1 MB of memory.

Netroom, Helix Software Co., Inc.
---------------------------------

47-09 30th St., Long Island City, NY 11101
(718) 392-3100, Technical Support (718) 392-3735

System requirements include 8086 or 8088 with expanded memory and expanded memory
manager specific to LIM/EMS 4.0. 80286, 80386, or 80486 with extended or
expanded memory.

Qram, Quarterdeck Office Systems
--------------------------------

Santa Monica, CA
(310) 309-3700 for Sales
(310) 309-4250 for Technical Support

System requirements include 8086, 8088, or 80286. To run device drivers and/or
TSRs between 640K and 1024K, LIM/EMS 4.0, EEMS expanded memory or Chips &
Technologies shadow RAM must be available.
