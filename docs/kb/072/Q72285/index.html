Q72285: Non-MS-DOS Format Conversion Utility: Xeno Copy PC

Article: Q72285
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:3.x,4.x,5.0
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Xeno Copy makes a product called Xeno Copy PC that translates between different
media formats. Xeno Copy PC mainly converts other soft- sectored, double-sided
floppy formats to a format MS-DOS can recognize (definitions of soft-sectored
and double-sided are listed below).

MORE INFORMATION
================

For example, Xeno Copy PC can convert a CP/M format to a Kaypro II format.

For more information on Xeno Copy PC, contact Xeno Copy at (415) 644-9366.

The following definitions are taken from the "Dictionary of Computer Terms," 2nd
Edition, by Barron's:

  Soft-sectored -- When a soft-sectored disk is formatted, the computer writes
  a magnetic pattern onto the disk to mark the boundaries of each sector.

  Double-sided -- When a double-sided disk has a magnetic coating on both sides
  and therefore can have data stored on both sides.
