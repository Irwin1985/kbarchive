Q266521: Encarta 2001: Shockwave Is Automatically Reinstalled

Article: Q266521
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbsetup kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Africana, Third Edition 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
- Microsoft Encarta Interactive World Atlas 2001 
- Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SUMMARY
=======

If you uninstall the Macromedia Shockwave software, and then you start any of
the programs listed at the beginning of this article, Shockwave is reinstalled
automatically.

MORE INFORMATION
================

If you manually remove Shockwave or if a Shockwave component is damaged, you may
need to uninstall and reinstall the Shockwave software. To do this:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. Click Shockwave, and then click Add/Remove.

4. Follow the instructions on the screen to uninstall the Shockwave software.

After you uninstall the Shockwave software, start one of the Encarta programs
listed at the beginning of this article to automatically reinstall the Shockwave
software.


The Encarta programs listed at the beginning of this article include Shockwave
version 8.0.0.196. To download and install the latest version of Shockwave,
visit the following Macromedia Web site:

  http://www.macromedia.com/shockwave/download
