Q285201: Entertainment PocketPak Setup Does Not Recognize Device Type

Article: Q285201
Product(s): Microsoft Home Games
Version(s): 2.0,2.11,2.12,3.0
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Pocket PC Entertainment PocketPak 
- the operating system: Microsoft Windows CE, versions 2.0, 2.11, 2.12, 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install the Microsoft Pocket PC Entertainment PocketPak, you
may receive the following error message:

  Microsoft <game> (pocket pc) does not support the connected device
  type...

Or the Setup program may not finish.

CAUSE
=====

This behavior can occur if you are using Microsoft Windows CE Operating System
version 2.12 or earlier.

RESOLUTION
==========

Microsoft Pocket PC Entertainment PocketPak requires Windows CE version 3.0 to
work correctly.


MORE INFORMATION
================

If your Windows CE device runs Windows CE version 2.12, 2.11, or 2.0, you can
purchase Microsoft Entertainment Pack 2.0 for Windows CE at a retail software
store.

Additional query words: msgame wince pocketpack
